## 1. Electron App 기본 실행 테스트

### 목표

pnpm 환경에서 Electron 앱이 정상적으로 실행되는지 확인.

### 구현 방식

1. 프로젝트 초기화 및 Electron 설치.
2. 간단한 Electron 창 띄우기.
3. 앱 실행 환경이 정상적으로 동작하는지 확인.

### 코드

```bash
# 프로젝트 초기화
mkdir electron-test
cd electron-test
pnpm init -y

# Electron 설치
pnpm add electron
```

```jsx
// main.js
const { app, BrowserWindow } = require('electron');

app.on('ready', () => {
    const mainWindow = new BrowserWindow({
        width: 800,
        height: 600
    });
    mainWindow.loadURL('<https://example.com>'); // 기본 URL
});
```

**2. 투명 배경 앱 설정**

**목표**

Electron 창의 배경을 투명하게 설정하고, Always-on-top 모드 활성화.

**구현 방식**

1. Electron 창 설정에서 transparent와 frame 옵션 활성화.
2. 투명 창을 띄운 후 동작 확인.

**코드**

```jsx
// main.js
const { app, BrowserWindow } = require('electron');

app.on('ready', () => {
    const mainWindow = new BrowserWindow({
        width: 800,
        height: 600,
        transparent: true, // 투명 배경
        frame: false,      // 창 테두리 제거
        alwaysOnTop: true, // 항상 위에 표시
    });
    mainWindow.loadFile('index.html'); // 기본 HTML 파일
});
```

```jsx
<!-- index.html -->
<!DOCTYPE html>
<html>
<head>
    <style>
        body {
            margin: 0;
            background-color: transparent;
        }
    </style>
</head>
<body>
    <h1 style="color: white;">Transparent Window Test</h1>
</body>
</html>
```

**3. 키보드 훅 테스트 (iohook)**

**목표**

전역 키보드 입력을 감지하고, 콘솔에 입력 이벤트 출력.

**구현 방식**

1. iohook 라이브러리를 설치.
2. 키보드 이벤트를 감지하고 로그 출력.

**코드**

```jsx
# iohook 설치
pnpm add iohook
```

```jsx
// main.js
const iohook = require('iohook');
const { app, BrowserWindow } = require('electron');

app.on('ready', () => {
    const mainWindow = new BrowserWindow({
        width: 800,
        height: 600,
        transparent: true,
        frame: false,
    });
    mainWindow.loadFile('index.html');

    // 키보드 입력 감지
    iohook.on('keydown', event => {
        console.log(`Key pressed: ${event.keycode}`);
    });
    iohook.start();
});
```

**4. 오른쪽 클릭 컨텍스트 메뉴**

**목표**

오른쪽 클릭 시 사용자 정의 컨텍스트 메뉴를 표시.

**구현 방식**

1. Electron의 Menu API 사용.
2. 브라우저 창에서 오른쪽 클릭 이벤트 감지 및 메뉴 표시.

**코드**

```jsx
// main.js
const { app, BrowserWindow, Menu } = require('electron');

app.on('ready', () => {
    const mainWindow = new BrowserWindow({
        width: 800,
        height: 600
    });
    mainWindow.loadFile('index.html');

    const contextMenu = Menu.buildFromTemplate([
        { label: 'Reload', click: () => mainWindow.reload() },
        { label: 'Inspect', click: () => mainWindow.webContents.openDevTools() },
    ]);

    mainWindow.webContents.on('context-menu', () => {
        contextMenu.popup();
    });
});
```

```jsx
<!-- index.html -->
<!DOCTYPE html>
<html>
<body>
    <h1>Right-Click Test</h1>
    <p>Try right-clicking on this page to see the context menu!</p>
</body>
</html>
```
