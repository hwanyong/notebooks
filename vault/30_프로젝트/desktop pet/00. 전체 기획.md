# Electron 기반 앱 기획서

## 1. 프로젝트 개요

**앱 이름**: KeyTracer

**목적**:

- MacOS에서 모든 키보드 입력을 실시간으로 감지.
- Live2D 캐릭터와 상호작용하는 인터랙티브 앱 개발.
- 키 입력 통계 시각화 및 캐릭터 애니메이션 지원.

---

## 2. 주요 기능

### 2.1 Live2D 캐릭터와 상호작용

- 키보드 입력에 따라 캐릭터가 반응.
- 애니메이션 변화, 이동, 표정 변경.

### 2.2 전역 키 입력 감지

- MacOS에서 모든 프로세스의 키 입력 이벤트 감지.
- 특정 키 입력 기록 및 로그 저장.

### 2.3 커스터마이징 가능한 반응

- 키 입력에 따른 캐릭터 동작 설정.
- 사용자 설정 UI 제공.

### 2.4 투명 창 모드

- 창의 배경 투명화 및 Always-on-top 지원.
- 배경 없는 캐릭터만 표시 가능.

### 2.5 키 입력 통계 기록

- 키 입력 빈도 및 시간별 분석.
- 대시보드에 실시간 시각화.

---

## 3. 기술 스택

### 3.1 프론트엔드

- HTML5 + CSS3 + JavaScript
- [Live2D Cubism SDK](https://www.live2d.com/en/)

### 3.2 백엔드

- Electron + Node.js 환경
- iohook 라이브러리로 전역 키보드 입력 감지

### 3.3 추가 라이브러리

- PIXI.js: Live2D 캐릭터 렌더링
- Chart.js: 입력 통계 시각화

---

## 4. UI 설계

### 4.1 Live2D 화면

- 투명 배경 위에 Live2D 캐릭터 표시.
- Always-on-top 모드로 다른 작업 중에도 표시.

### 4.2 설정 메뉴

- 캐릭터 모델 선택 및 반응 설정.
- 투명도, 창 크기, 위치 조정.

### 4.3 대시보드

- 실시간 키 입력 통계 표시 (그래프, 히트맵 등).
