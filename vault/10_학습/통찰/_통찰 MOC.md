# 💡 통찰·사고 MOC

> 여러 학문·영역이 **한 원리로 수렴**하는 cross-domain 메타 통찰을 모으는 곳. 특정 학문 노트에 안 들어가는 융합 사고.
> 여기선 *분류(입구)보다 연결(융합)이 핵심* — 각 통찰은 관통하는 여러 영역으로 다관점 연결된다.

🌐 영역 간 연결: [[_지식 연결 허브]] ｜ 🔙 [[_학습 MOC]]

## 통찰 목록
- [[엔트로피와 질서 — 최소 에너지 법칙과 엔지니어링|엔트로피 vs 질서]] — 무질서는 자연 법칙(최소 에너지·엔트로피), 질서 유지는 고에너지 역행. **뇌·자연·코드·LLM 관통** (열역학·인지과학·진화생물학·SW공학·AI·철학)

- [[벡터 — 기하·물리·선형대수를 관통하는 한 뿌리 (피타고라스·삼각함수)|벡터의 한 뿌리]] — 노름=거대해진 피타고라스, 코사인 유사도=확장된 삼각함수. **기하·물리·선대(AI) 관통** (수학·물리·AI)
- [[왜 선형대수인가 — 데이터·알고리즘의 표준 언어|왜 선형대수인가]] — 선형대수=추상적 데이터 형상·흐름을 전달하는 **표준 소통 언어** + 알고리즘 최적화(피보나치 O(logN)·인접행렬·PageRank), 짝꿍은 이산수학. **CS·아키텍처·수학·AI 관통 + 메타인지**
- [[두 전파 — 역전파 vs 라우팅 (그래프는 같고 메시지는 다르다)|두 전파(역전파 vs 라우팅)]] — 둘은 *그래프 위상*만 같고 별개 알고리즘. 진짜 공통은 **그래프 위 메시지 전달**(역전파=역방향 자동미분, 거리벡터=Bellman-Ford). **AI·CS/네트워크·그래프 관통** (T01 과잉단정 교정)
- [[신경망은 왜 행렬인가 — 형식·가중치·학습가능성|신경망은 왜 행렬인가]] — 신경망 그림=행렬(엣지=가중치=원소), 모델 자체가 행렬 더미; 표현·합성·**미분**이 동시에 닫혀 *학습가능성이 행렬을 강제*; 선형 붕괴가 비선형을 부름. **수학(선대)·AI·CS(Software 2.0) 관통**
- [[LLM 전체 그림 — 벡터·행렬·통계가 어디서 제어하나 (학습 로드맵)|LLM 전체 그림 (학습 로드맵)]] 🧭 — LLM 파이프라인 한 장 지도(임베딩→층→어텐션→softmax→손실→역전파); **벡터·행렬·통계·미적분이 각 구간을 제어** + 배움/예정(2·3단계) 위치 표시. *진도 북극성*

## 운영 원칙
- **통찰 = 여러 영역을 관통하는 메타 원리.** 한 학문 노트로 안 들어갈 때 여기에.
- 각 통찰은 관통 영역들에 **다관점 연결 + 전역 허브 다리**로 묶는다. (예: 국소 최적점 ↔ AI 경사하강법 ↔ 진화 ↔ 조직)
- 책 진도와 무관한 **가로축(융합)** 지식 (→ 폴더 구조 가이드 3-8).
