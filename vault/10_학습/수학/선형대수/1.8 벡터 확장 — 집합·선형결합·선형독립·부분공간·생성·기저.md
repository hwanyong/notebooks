# 벡터 확장 — 집합·선형 가중 결합·선형 독립·부분공간·생성·기저

> 📐 **수학 개념원자(SSoT)** — 선형대수. 정의·성질·증명·기하 직관. AI 적용·연결은 코스 렌즈가 끌어다 쓴다.
> 🔗 [[_수학 MOC]] ｜ 🌐 [[_지식 연결 허브]] ｜ 🎓 AI 렌즈: [[06 벡터 확장 — 집합·선형결합·선형독립·부분공간·생성·기저|06(AI)]]

> 벡터 한 개를 넘어 **여러 벡터가 만드는 공간의 구조**로. 핵심 사슬: *벡터 집합 → 선형 가중 결합(조합) → 그 조합들이 채우는 공간(생성·부분공간) → 그 공간을 최소·유일하게 설명하는 자(기저)*. 그리고 이 모든 걸 가르는 한 가지 성질이 **선형 독립성**(정보가 겹치나 안 겹치나).

> 🎯 **핵심 (TL;DR)**
> - **선형 가중 결합(linear weighted combination([ˈlɪniər ˈweɪtɪd ˌkɒmbɪˈneɪʃən], 리니어 웨이티드 콤비네이션, 리니어 웨이티드 컴비네이션)):** $w=\lambda_1 v_1+\cdots+\lambda_n v_n$ — 각 벡터에 스칼라(계수 λ) 곱해 더함. *선형대수에서 가장 중요한 연산*. 뺄셈은 λ 음수로 흡수.
> - **선형 독립(linear independence([ˈlɪniər ˌɪndɪˈpɛndəns], 리니어 인디펜던스, 리니어 인디펜던스)):** 한 벡터도 *나머지의 선형결합으로 표현 불가* = 정보가 안 겹침. 표현 가능하면 **선형 종속**(중복). 정식 정의(식 2-3): $0=\lambda_1 v_1+\cdots+\lambda_n v_n$ 에 **비자명해(λ≠0)** 가 있으면 종속.
> - **생성(span)·부분공간(subspace([ˈsʌbspeɪs], 서브스페이스, 서브스페이스)):** 생성=집합의 *모든* 선형결합(무한). 부분공간=덧셈·스칼라곱으로 닫힌(원점 포함) 부분집합. 독립 벡터 수 = 그 공간의 **차원**.
> - **기저(basis):** 공간을 *생성* + *선형 독립* = 공간을 재는 **자(ruler)**. 표준기저 외에도 무한히 많음. **독립이라야 좌표가 유일**(데이터·ML의 핵심: 최적 기저 찾기 = 차원축소·PCA·ICA·특징추출).
> - **영벡터 포함 집합은 항상 종속.** "독립성"은 *개별 벡터*가 아니라 *집합*의 속성.

---

> [!question] 🎯 필요성 사슬 (결핍 → 필요 → 발명)
> 추상 정의를 외우지 말고, **각 개념이 *앞 단계가 만든 결핍*의 답**임을 보라. *출발 문제:* 데이터(벡터)를 수천 개 다루고, 비슷한 걸 찾고 핵심만 압축하고 싶다 → 거기서 §2 전체가 줄줄이 태어난다.
> 1. 데이터가 한 개 아닌 **수천 개** → 떼로 지칭할 말이 없다 → **벡터 집합**
> 2. 모은 벡터로 *새 값*(예측·평균·특징)을 만들고 싶다 → 섞는 **만능 연산**이 없다 → **선형 가중 결합**
> 3. 모으니 어떤 건 **재탕(중복)** → "진짜 정보가 몇 개?"를 잴 잣대가 없다 → **선형 독립/종속**
> 4. "이 벡터들로 **닿는 범위**는 어디까지? 답이 그 안에 있나?" 모른다 → **생성·부분공간** (표현 가능 영역의 경계)
> 5. 같은 공간을 설명하는 자(좌표계)가 **무한히 많다** → *최소·유일*한 설명자가 없다 → **기저** (= 데이터에 맞는 *최적 기저 찾기*)
>
> **한 줄:** 벡터를 → (떼로)**집합** → (섞으려)**선형결합** → (군더더기 빼려)**독립성** → (닿는 범위 알려)**생성·부분공간** → (최소·유일하게 설명하려)**기저**. *각 개념 = 바로 앞 결핍의 해결책.*

> [!abstract] 📘 책 핵심 — [실전선대] §2.1~2.6
> **① §2.1 벡터 집합(set([sɛt], 셋, 셋))** — 벡터들의 모음 $V=\{v_1,\dots,v_n\}$. 유한·무한 가능(무한 집합은 통계모델 적합에 영향). 빈 집합 $V=\{\}$.
>
> **② §2.2 선형 가중 결합 (식 2-1)**
> $$ w=\lambda_1 v_1+\lambda_2 v_2+\cdots+\lambda_n v_n $$
> - 모든 $v_i$ 는 **같은 차원**(아니면 덧셈 불가). $\lambda$ 는 0 포함 임의 실수. 별칭: 선형 혼합·가중 결합, 계수(coefficient([ˌkoʊɪˈfɪʃənt], 코우이피션트, 코이피션트)).
> - 예(식 2-2): $\lambda{=}[1,2,{-}3]$, $v_1{=}[4,5,1]^{\mathsf{T}},v_2{=}[{-}4,0,{-}4]^{\mathsf{T}},v_3{=}[1,3,2]^{\mathsf{T}}$ → $w{=}[{-}7,{-}4,{-}13]^{\mathsf{T}}$. `l1*v1+l2*v2+l3*v3`.
> - 응용 *(🔭 예고 — 지금은 통과, 가리키는 곳만 앵커):* ① 회귀=독립변수(regressor)와 계수의 선형결합(10·11장) ② PCA 성분=분산 최대화 가중치와 데이터 채널의 선형결합(14장). *(정복은 해당 장에서 — 규칙 17)*
>
> **③ §2.3 선형 독립성**
> - **선형 종속:** 집합에서 *적어도 한 벡터*를 나머지의 선형결합으로 표현 가능. **선형 독립:** 어떤 벡터도 불가.
> - 예: $V=\{[1,3]^{\mathsf{T}},[2,7]^{\mathsf{T}}\}$ 독립 / $S=\{[1,3]^{\mathsf{T}},[2,6]^{\mathsf{T}}\}$ 종속($s_1{=}.5s_2,\ s_2{=}2s_1$). $T$(4×4) 종속(처음 세 벡터 합 = 네 번째의 2배 — 눈으로 못 봄).
> - 데이터 판별: 벡터로 행렬 만들어 **계수(rank)** 를 행·열 수 중 작은 값과 비교(5장).
> - **정식 정의(식 2-3):** $0=\lambda_1 v_1+\cdots+\lambda_n v_n,\ \lambda\in\mathbb{R}$. 모든 λ=0(자명해)이면 늘 성립하니, **비자명해(적어도 한 λ≠0)** 가 존재하면 **종속**. 좌변을 0으로 두는 건 *전체 집합*의 성질로 보기 위함(특정 벡터에 "종속" 특권 안 줌).
>
> **④ §2.3.2 독립성과 영벡터** — 영벡터 포함 집합은 **항상 종속**($\lambda_0 0=0v_1+\cdots$ 비자명해 존재). 비선형 독립성은 선형대수 범위 밖(선형 연산=행렬곱으로 변환하려는 제약).
>
> **⑤ §2.4 부분공간·생성**
> - **생성(span):** (유한) 집합의 *동일 벡터들*을 무한히 선형결합해 만드는 **모든 벡터**. = 부분공간을 만드는 *메커니즘*.
> - **부분공간(subspace):** 덧셈·스칼라곱으로 **닫힌**(결과가 다시 그 안) 부분집합 — 반드시 **원점(영벡터) 포함**.
> - 그림 2-1(1벡터→1D 선)·2-2(독립 2벡터→2D 평면)·2-3(종속 2벡터→1D 선). $V=\{[1,1,1]^{\mathsf{T}},[2,2,2]^{\mathsf{T}}\}$ 는 R³에 있어도 생성은 **1D**(한 벡터가 다른 것의 생성 안 = 중복).
> - **차원 ↔ 벡터 수:** 독립이면 생성 부분공간 차원 = 벡터 수 / 종속이면 차원 < 벡터 수(정확한 관계는 5장 rank).
>
> **⑥ §2.5 기저(basis)** — 공간을 재는 **자(단위)**. 거리 "2,000"이 마일/㎞ 단위(기저)에 따라 의미가 정해지듯.
> - **표준 기저(standard basis):** $S_2{=}\{[1,0]^{\mathsf{T}},[0,1]^{\mathsf{T}}\}$, $S_3$ — 직교·단위. 어디서나 존재.
> - 유일하지 않음: $T{=}\{[3,1]^{\mathsf{T}},[-3,1]^{\mathsf{T}}\}$ 도 R² 기저. 같은 점 $p,q$ 를 $S$ 좌표 $(3,1),(-6,2)$ vs $T$ 좌표 $(1,0),(0,2)$ 로(그림 2-4) — 데이터는 같고 *자(기저)* 만 다름.
> - **2.5.1 정의:** 기저 = (1)부분공간을 **생성** + (2)**선형 독립**. 측정하려면 생성해야 하고(그림 2-6: 생성 밖 점 r은 못 잼), 좌표가 **유일**하려면 독립이라야(그림 2-5 PCA/ICA). $U{=}\{[1,0]^{\mathsf{T}},[2,0]^{\mathsf{T}},[0,1]^{\mathsf{T}}\}$ 는 유효 벡터집합이나 **기저 아님**(종속 → 좌표 비유일).

## 🧠 근본 이해 · 융합 · 통찰

### ⚖️ 왜 가중치를 다르게 주나 — '변조'가 아니라 비중 조절(합성·추출)
> 💭 막힘: *"변수마다 가중치를 다르게 주면 데이터가 변조(왜곡)되는 것 아닌가? 왜 다르게 줘야 하나?"*

- **변조가 아니라 합성·추출.** 선형 가중 결합은 원본 $v_1,v_2,\dots$ 를 *덮어쓰지 않는다*. 원본은 그대로 두고, 그것들을 *이런 비율로 섞은* **새 결과 벡터 $w$** 를 만들 뿐. 원본 보존 → 변조 아님.
- **왜 똑같이 안 주나? 정보마다 중요도·스케일이 다르니까.** 핵심 반전: **"가중치를 다 똑같이(1) 주는 것"도 *"모두 똑같이 중요하다"는 하나의 가정(편향)***일 뿐 더 중립이 아니다. 현실 데이터는 차원마다 중요도가 달라, *다르게 주는 게 오히려 정확*.
  - 성적: `학점 = 0.3·중간 + 0.3·기말 + 0.4·과제` — 똑같이 1/3씩 주면 "과제가 더 중요"란 현실을 무시.
  - 회귀(예측): `집값 = w₁·면적 + w₂·역거리 + …` — 면적이 더 좌우하면 $w_1$ 커야 정확. **이 가중치를 데이터에서 자동으로 찾는 게 최소제곱·머신러닝** (🔭 예고 — 10·11장).
  - PCA: *분산 큰 방향*에 큰 가중치 → 노이즈 버리고 핵심 신호 추출(왜곡이 아니라 압축).
- **왜 '왜곡'과 갈리나 = 선형(linear)이라서.** 가중치를 달리 줘도 **스케일링+덧셈**만 하지 휘거나 끊지 않는다 → 직선은 직선으로 남고(선형성) 가역(가중치를 알면 되돌림). *형상을 비트는 게 아니라 축의 비중만 바꾸는 것* (= 선형 변환은 격자를 균일하게 늘이/돌릴 뿐 구불구불하게 안 만듦, [[12 행렬 곱셈·행렬식 — 변환 합성·면적 배율·선형종속|12]]·[[T06 벡터=상태 vs 행렬=연산자 (기저 재정의·SIMD·GEMM)|T06]]).

> 한 줄: **가중치 = "이 데이터가 결과에 얼마나 기여할지"의 다이얼.** 똑같이 두는 것도 *하나의 선택*이고, 중요도가 다른 현실을 반영하려면 다르게 줘야 정확. 원본은 안 바뀌고 *새 조합 w*를 만들 뿐 → 변조 아님. 이 "비중 조절"이 곧 신경망 가중치(Wx)·회귀·PCA의 본질이며, **신경망이 '학습'한다 = 각 입력에 줄 가중치를 데이터에서 알아낸다**는 뜻.

### 🔑 선형 독립 = '정보 비중복'
- **독립 = 각 벡터가 새 정보를 더함.** 종속 = 한 벡터가 *이미 있는 정보의 재탕* → 공간을 못 넓힘(그림 2-3: 두 벡터인데 1D).
- **독립이라야** 각 축이 *서로 다른 의미*를 담는다. 종속 축 = 낭비·중복 → 실질 차원(계수)이 낮아짐 = **차원 붕괴**(위 §4.1.3 입문 그림 4-11·4-12와 같은 현상).

### 🔑 "기울기가 다르다=독립"의 함정 — 2D 지름길일 뿐, 3D+에서는 무너짐 (2026-07-13)

> [!question] 🎯 가설 → 검증 → 정정
> 가설: "선형 독립 집합 = 각 벡터가 다른 벡터(들)의 생성(span)에 안 담기는 것 — 2D에서는 곧 '기울기가 다른 벡터끼리'."
> **2D에서는 정확히 맞다.** 위 §2.3 예시 $V=\{[1,3]^{\mathsf{T}},[2,7]^{\mathsf{T}}\}$(기울기 3 vs 3.5, 다름) → 독립. $S=\{[1,3]^{\mathsf{T}},[2,6]^{\mathsf{T}}\}$(기울기 둘 다 3, [2,6]=2·[1,3]) → 종속. 2D에선 방향(직선)이 하나뿐이라 "기울기 다르다 ⇔ 독립"이 정확히 동치다.
> **3D+에서는 이 지름길이 무너진다.** 위 [그림 2-2/4-12] 인터랙티브 예시가 정확히 이 함정을 이미 보여주고 있었다: $a{=}(2,0,1),\ b{=}(0,2,1),\ c{=}a{+}b{=}(2,2,2)$ — 셋 다 서로 배수가 아니라 "쌍마다 방향이 다름"에도 **$c$는 $a,b$가 이미 채운 평면(span$\{a,b\}$) 안에 있어서 여전히 종속**이다(범례에 처음부터 "c=a+b (종속)"으로 명시돼 있었다).
> **진짜 일반 기준(모든 차원 공통):** 새 벡터가 *이미 있는 벡터들의 span을 벗어나는가* — 벗어나면 독립, 그 안에 갇히면 종속. "기울기가 다르다"는 이 일반 기준의 **2D 한정 축약형**일 뿐, 3D 이상에선 필요조건이지 충분조건이 아니다.
>
> 한 줄: 벡터쌍끼리 방향이 다 달라도 3D+에서는 독립을 보장 못 한다 — 진짜 기준은 언제나 "새 벡터가 기존 span을 뚫고 새 차원을 여는가"이고, "기울기가 다르다"는 2D에서만 그 기준과 우연히 같아지는 특수 사례다.

### 🧩 생성(동사) vs 부분공간(명사) — 같은 것의 두 얼굴
- **생성(span) = 동사(메커니즘):** "모든 선형결합을 만들어내는 과정".
- **부분공간(subspace) = 명사(결과):** "그렇게 만들어진, 더 큰 공간 속 닫힌 부분".
- 둘은 보통 같은 무한 벡터 집합을 가리켜 혼용해도 무방. *닫힘*(덧셈·스칼라곱 결과가 다시 안에)·*원점 포함*이 부분공간의 형식 조건.

### 🔑 기저 = '최소 + 유일'한 설명자 (좌표 유일성)
- 기저의 두 조건이 곧 두 효용: **생성**(빠짐없이 다 잰다) + **독립**(좌표가 *유일*). $U$ 처럼 종속이면 한 점의 좌표가 여러 개((3,0,1)·(0,1.5,1)…) → 측정값이 안 정해짐.
> [!quote] 🌟 왜 기저가 데이터·ML에서 *매우* 중요한가 (책 강조점 정리)
> 책이 "기저는 데이터 과학·ML에서 매우 중요"라고 못 박은 이유를 한 묶음으로:
> 1. **데이터는 안 변한다 — 보는 *자(기저)*가 분석을 정한다.** 같은 점 $p,q$ 도 어떤 기저로 재느냐로 좌표(기술)가 달라진다(그림 2-4). 즉 "기저를 고른다 = 데이터를 보는 관점을 고른다".
> 2. **좋은 기저 = 적은 축으로 핵심을 담는다.** 분산 큰 방향에 정렬된 기저로 바꾸면 *몇 개 축*만으로 데이터를 설명 → **차원 축소·압축·노이즈 제거·해석 가능성**.
> 3. **그래서 차원·특징 기법이 한 문제로 환원된다 = "이 부분공간의 *최적 기저* 찾기".** 차원축소·특징추출·**PCA·ICA·요인분석·SVD·LDA**·이미지 근사·데이터 압축·임베딩 — 겉보기엔 다른 기법이지만 본질은 전부 *데이터를 가장 잘 설명하는 축(기저)을 고르는 것*. (이 본질 자체가 수학; AI 응용 다리는 렌즈에서.)
> **한 줄:** 기저가 중요한 건 *데이터를 안 바꾸고 관점만 바꿔* 핵심을 적은 축에 담기 때문 — 그래서 "최적 기저 찾기"가 차원축소·특징추출 등 분석의 **공통 목적지**다.

## 🧩 코드 실습 (§2.2~2.3)
> 🎯 목표: 선형결합 계산(식 2-2 재현) + 선형 독립/종속을 rank로 판별.
> 코드: `~/LOCAL/03-00_STUDIES/AI/01_linear_algebra/la2_combo_independence.py` (검증: w=[-7,-4,-13]; 독립 rank=2, 종속 rank=1)

```python
import numpy as np
# 선형 가중 결합 (식 2-2)
l1,l2,l3 = 1,2,-3
v1,v2,v3 = np.array([4,5,1]), np.array([-4,0,-4]), np.array([1,3,2])
print(l1*v1 + l2*v2 + l3*v3)          # [ -7  -4 -13]
# 선형 독립 판별 = rank(열로 쌓은 행렬)
indep = np.array([[1,3],[2,7]]).T     # 열=벡터
dep   = np.array([[1,3],[2,6]]).T     # s2 = 2·s1
print(np.linalg.matrix_rank(indep))   # 2  (독립: rank=벡터수)
print(np.linalg.matrix_rank(dep))     # 1  (종속: rank<벡터수)
```
관찰: **rank = 독립 벡터의 수 = 생성 부분공간의 차원.** rank < 벡터 수 → 종속(중복 차원).
> ⚠️ 책 각주: 리스트·정수 곱(`[1,2]*3`)은 스칼라곱이 아니라 *리스트 반복* → 반드시 `np.array`.

## 🆕 새 용어 정리
| 용어 | 한 줄 정의 |
|---|---|
| **벡터 집합** (set) | 벡터들의 모음 V={v₁,…,vₙ}; 유한·무한 |
| **선형 가중 결합** (linear weighted combination) | w=Σλᵢvᵢ; 각 벡터에 계수 곱해 더함(가장 중요한 연산) |
| **계수** (coefficient) | 선형결합의 스칼라 λ |
| **선형 독립 / 종속** | 한 벡터도 나머지 결합으로 표현 불가(독립) / 가능(종속·중복) |
| **비자명해** (nontrivial) | 식2-3 0=Σλᵢvᵢ에서 λ≠0 해; 존재하면 종속 |
| **생성** (span) | 집합의 모든 선형결합(무한) — 부분공간 만드는 메커니즘. **=스팬**([필수수학] 표기, 같은 개념) |
| **부분공간** (subspace) | 덧셈·스칼라곱으로 닫힌 부분집합(원점 포함) |
| **차원** (dimension) | 부분공간을 생성하는 독립 벡터의 최소 수(=rank) |
| **기저** (basis) | 부분공간을 생성 + 선형 독립; 공간을 재는 자(좌표 유일) |
| **표준 기저** (standard basis) | 직교·단위 기저 S₂={[1,0]ᵀ,[0,1]ᵀ} 등 |
