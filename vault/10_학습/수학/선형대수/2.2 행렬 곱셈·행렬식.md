> 🎯 **핵심 (TL;DR)**
> - **행렬 곱 = 변환 합성(composition([ˌkɒmpəˈzɪʃən], 컴퍼지션, 컴포지션)).** $AB$ = "$B$ 변환 먼저, 그다음 $A$" (안→밖).
> - **비교환(non-commutative([nɒn kəˈmjuːtətɪv], 논 커뮤터티브, 논 커뮤터티브)):** $AB \ne BA$ — 순서가 결과를 바꾼다.
> - **행렬식(determinant([dɪˈtɜːrmɪnənt], 디터미넌트, 디터미넌트)) = 면적/부피 배율.** 전단·회전 → det=1(불변).
> - **det 부호 = 방향.** det<0 = 뒤집힘. **det=0 = 차원 붕괴 = 선형 종속** → 역행렬 없음·연립방정식 못 풂.
> - **유효성 📌:** $(M{\times}N)(N{\times}K)$ — **'내부'($N$) 일치할 때만 유효, 결과 크기 = '외부' $M{\times}K$.** 행렬-벡터 곱: $Av$(열벡터 뒤)·$v^{\mathsf{T}}A$(행벡터 앞)만 유효, 결과 방향 = 곱한 벡터 방향.
> - **"표준"이라 부르는 이유:** 아다마르곱·스칼라 곱셈([[2.1 행렬 연산 — 덧셈·뺄셈·이동·스칼라곱·아다마르곱|2.1]] ⭐혼동 표)과 구별. 표기: 아다마르 $A\odot B$ / 행렬곱 $AB$(기호 없음).
> - 코드: `A @ B`(합성, `matmul` — `A*B`는 아다마르!), `np.linalg.det(M)`(행렬식).

> [!tip] 🗺️ "○○곱" 헷갈릴 때 → [[_용어집#🔢 곱(product) 연산 한눈 지도|곱 연산 한눈 지도(용어집)]]
> 행렬곱(AB)=변환 합성이지 *벡터곱*이 아님. 내적·외적·교차곱·행렬×벡터·행렬×행렬 구분은 용어집 한 표로. (행렬곱 각 칸 = 행·열 내적)

> [!question] 🎯 필요성 사슬 (결핍 → 필요 → 발명)
> *출발 문제:* 행렬이 *하나의 변환*이라면, 변환을 여러 번 잇고, 그 변환이 정보를 잃는지 진단해야 한다.
> 1. 변환 하나로는 부족, 여러 변환을 잇따라 적용하고 싶음 → 변환을 합치는 연산 필요 → **행렬 곱(변환 합성 AB)**
> 2. 순서를 바꿔도 같을 줄 알았는데 결과가 달라짐 → 순서가 의미를 바꾼다는 사실 필요 → **비교환성(AB≠BA)**
> 3. 이 변환이 공간을 얼마나 늘이나·뒤집나를 한 숫자로 알고 싶음 → 면적/부피 배율 측정 필요 → **행렬식(determinant)**
> 4. 변환이 차원을 깎아 되돌릴 수 없는지(선형종속·비가역) 진단해야 함 → 0 판별 필요 → **det=0 ⇔ 차원 붕괴·역행렬 없음**
> **한 줄:** 변환을 잇는 행렬곱(합성·비교환) → 공간을 얼마나 늘이나(행렬식) → 정보를 잃었나 진단(det=0=선형종속·비가역).

> [!question] 🌱 더 깊은 뿌리 — "왜 이렇게 *이상하게* 정의했나?" (연립방정식)
> 행렬곱은 처음 보면 *당황스러울 만큼* 특이하게 정의된다(왜 행·열 내적?). 역사적 뿌리는 **변환 합성이 아니라 연립일차방정식 $Ax=b$**.
> - 연립방정식 $\begin{cases}ax+by+cz=p\\ \cdots\end{cases}$ 를 **계수행렬 $A$ · 미지수 벡터 $x$ · 상수 벡터 $b$** 로 분리 → $Ax=b$. 행렬곱은 *이 분리를 성립시키려고* 그렇게 정의된 것.
> - **곱 정의 조건(앞 열수 = 뒤 행수)** 도 여기서 나옴: 유일해를 가지려면 *미지수 수 = 방정식 수* → 계수행렬 열 수 = 미지수 벡터 행 수.
> ⇒ 행렬곱엔 **두 기원**: ① *왜/역사* = 연립방정식 표현, ② *무엇/기능* = 변환 합성. 같은 연산의 두 얼굴. (원문 정리 → [[행렬 곱셈의 두 관점 (Gosamy 원문 정리)|Gosamy]])

> [!abstract] 📘 책 핵심 — [필수수학] §4.3 행렬 곱셈 · §4.4 행렬식
> **① §4.3 행렬 곱 = 변환 합성**
> - 두 행렬을 곱함 = 두 변환을 **순차 적용**. *가장 안쪽 먼저 → 바깥쪽으로.*
> $$ \begin{bmatrix}a&b\\c&d\end{bmatrix}\begin{bmatrix}e&f\\g&h\end{bmatrix}=\begin{bmatrix}ae+bg&af+bh\\ce+dg&cf+dh\end{bmatrix} $$
> - 예제 4-11(회전∘전단): `combined = transform2 @ transform1`; `combined.dot([1,2])` → `[-1, 1]`.
> - 예제 4-12(순서 반대): `transform1 @ transform2` → `[-2, 3]`. **순서 바뀌면 결과 다름 = 교환법칙 성립 안 함.**
> - ⚠️ **`@`(matmul) vs `dot`:** 2D는 같지만 3D+ 텐서에선 다름 → 행렬곱엔 `@`/`matmul` 권장. 비정방 곱은 *앞 행렬 열 수 = 뒤 행렬 행 수*.
>
> **② §4.4 행렬식 = 면적(부피) 배율**
> - 변환이 두 기저로 만든 단위 영역을 **몇 배로** 늘이나 = 행렬식. 예: $\hat\imath{=}[3,0],\hat\jmath{=}[0,2]$ → 넓이 1→6, **det=6**.
> - **단순 전단·회전 → det=1**(넓이 안 변함). 예제 4-14 전단 det=1.
> - **부호:** $\hat\imath,\hat\jmath$ 순서가 시계방향으로 뒤집히면 **det<0**(공간 반전). 예제 4-15 det=−5.
> - **det=0 → 모든 공간이 더 낮은 차원으로 압축**(2D→1D 선, 3D→2D 면) = **선형 종속**. 예제 4-16 det=0. → 역행렬 없음 신호.

> [!abstract] 📘 책 핵심 — [실전선대] §4.3 표준 행렬 곱셈 · §4.3.1 유효성 · §4.3.2 메커니즘 · §4.3.3 행렬-벡터 곱셈
> **① §4.3 — 비직관적이지만 어렵지 않다**
> - 표준 행렬 곱셈은 원소별이 아니라 **행과 열 단위**로 동작 — *한 행렬의 행과 다른 행렬의 열 사이 스칼라 곱셈의 조직적 집합.* 공식 명칭은 그냥 **행렬 곱셈**; **"표준"을 붙인 이유 = 아다마르곱·스칼라 곱셈과 명확히 구별하기 위해.**
>
> **② §4.3.1 — 유효성 규칙** 📌
> - 크기 $M{\times}N$ 과 $N{\times}K$: **'내부' 차원($N$)의 수가 일치할 때만 유효**, 곱의 크기는 **'외부' 차원 $M{\times}K$** (그림 4-3). 공식적으론 "왼쪽 열 수 = 오른쪽 행 수"지만 **내부/외부로 기억하는 게 더 쉽다.**
> - 여기서도 비교환이 보임: $AB$ 유효해도 $BA$ 무효 가능; 둘 다 유효(정방)해도 $C{=}AB\ne D{=}BA$ 일반적.
> - **표기:** 아다마르곱은 $A\odot B$, 행렬 곱셈은 **아무 기호 없이** $AB$.
>
> **③ §4.3.2 — 메커니즘: 곱의 $(i,j)$ = (왼쪽 $i$행)·(오른쪽 $j$열) 내적** (유효성 규칙의 이유)
> $$ \begin{bmatrix}2&3\\4&5\end{bmatrix}\begin{bmatrix}a&b\\c&d\end{bmatrix}=\begin{bmatrix}(2a{+}3c)&(2b{+}3d)\\(4a{+}5c)&(4b{+}5d)\end{bmatrix}\quad\text{(식 4-1)} $$
> - 손가락 요령(그림 4-4): 왼손은 행을 따라, 오른손은 열을 따라 훑기.
> - **해석:** 내적 = 두 벡터 관계를 인코딩한 숫자 → 행렬곱 = **왼쪽 행·오른쪽 열 *모든 쌍*의 선형 관계를 저장하는 행렬**("정말 아름답다"). 공분산·상관 행렬·일반 선형 모델·특잇값 분해의 기초.
>
> **④ §4.3.3 — 행렬-벡터 곱셈** (행렬 하나가 벡터인 행렬 곱셈일 뿐 — 응용이 많아 투자 가치)
> - **유효성:** 열벡터만 오른쪽($Av$), 행벡터만 왼쪽($v^{\mathsf{T}}A$). $Av^{\mathsf{T}}$·$vA$ 무효. ($M{\times}N$ 왼쪽에 $1{\times}M$, 오른쪽에 $N{\times}1$)
> - **결과는 항상 벡터**, 방향은 곱하는 벡터를 따름(행벡터 앞곱→행벡터, 열벡터 뒤곱→열벡터). 📌
> - **두 가지 해석:** ⑴ **선형 가중 결합** — 벡터들을 행렬 열에, 가중치를 벡터 원소에: $4[3,0,6]^{\mathsf{T}}{+}3[1,2,5]^{\mathsf{T}} = \begin{bmatrix}3&1\\0&2\\6&5\end{bmatrix}\begin{bmatrix}4\\3\end{bmatrix}$ (행벡터 결합은 계수를 행벡터로 *앞*곱 — 각주). ⑵ **기하학적 변환** — 벡터를 회전+크기 조정(스칼라곱은 크기만).
> - **고유벡터 맛보기(그림 4-5):** $M{=}\begin{bmatrix}2&3\\2&1\end{bmatrix}$, $x{=}[1,1.5]^{\mathsf{T}}$ → $Mx$ 회전+늘림. 그런데 $v{=}[1.5,1]^{\mathsf{T}}$ → $Mv$ 는 **회전 없이 크기만**(스칼라곱처럼) — 우연 아님: **$v$는 $M$의 고유벡터, 배율이 고윳값** `(🔭 예고 — 12장)`.
> - 응용: 통계 예측 $X\beta$ · PCA $(Y^{\mathsf{T}}Y)v$(v=고유벡터) · 공간 필터 $w^{\mathsf{T}}S$ · 그래픽스 변환 $Tp$. 행렬-벡터 곱 = **행렬 공간의 기저** `(🔭 예고 — 다음 장 후반)`.

## 🧠 근본 이해 · 융합 · 통찰

### 🔑 행렬 곱 = 합성, 그래서 순서가 중요
- $AB$ 는 "$B$를 먼저 적용한 뒤 $A$" — **함수 합성 $A(B(v))$** 와 같다. 그래서 *가장 안쪽(입력 쪽)부터*.
- **비교환**은 버그가 아니라 본질: 회전 후 전단과 전단 후 회전은 다른 변환.

### 🔑 행렬곱을 보는 두 시점 — 칸=내적 / 열=선형결합
같은 행렬곱 $AB=C$ 도 **줌 레벨**에 따라 두 얼굴이다. 둘 다 결국 **선형 가중 결합** 한 원자로 환원된다.

- **① 칸(원소) 시점:** $C$의 한 칸 = (A의 그 *행*) · (B의 그 *열*) **내적** = $\sum_k A_{ik}B_{kj}$. *각 칸 하나 = 가중합 하나.*
- **② 열 시점:** $C$의 한 *열* 전체 = **A의 열들을 B의 그 열로 선형 가중 결합** — [[06 벡터 확장 — 집합·선형결합·선형독립·부분공간·생성·기저|06]]의 $w=\lambda_1 v_1+\lambda_2 v_2+\lambda_3 v_3$ **그 식 그대로**(λ = 계수 = B의 열).
- 그래서 **행렬×벡터 $Av$ = A의 열들을 v 성분으로 선형결합**, **행렬×행렬 $AB$ = 그걸 B의 여러 열에 한꺼번에**.

![[MD4-matmul 두 시점 칸내적·열선형결합 (재구성).png|860]]

🔄 **재구성** [본문 그림 + 성적표 예시] — (왼쪽) **칸 = 내적**: C의 빨간 칸 = A 1행 · B 1열, 파란 칸 = A 3행 · B 2열. (오른쪽) **열 = 선형결합**: C의 한 열 = A의 열들을 계수 (3,3,4)로 스케일해 더한 것.
> [!quote]- 📖 책 원본 [§4.3 행렬곱 모식도] *(캡처 시 attachments 추가)*

#### 📐 같은 두 시점을 $Ax=b$ 위에 — Strang의 row picture / column picture
연립방정식 $Ax=b$(예: $x-2y=1,\ 3x+2y=11$, 해 $(3,1)$)를 위 두 시점으로 보면 **두 개의 기하 그림**이 된다(Gilbert Strang). *원문 → [[행렬 곱셈의 두 관점 (Gosamy 원문 정리)|Gosamy 원문 정리]].*

- **① 행 그림(row picture):** $Ax = \big((\text{row}_i\,A)\cdot x\big)$. 각 식이 **한 직선**, 해 = **직선들의 교점** $(3,1)$.
- **② 열 그림(column picture):** $Ax = x(\text{col}_1 A)+y(\text{col}_2 A)=b$. 해 $(x,y)$ = **$b$ 를 $A$ 의 열들로 만드는 선형결합 계수**. 예: $3\!\begin{bmatrix}1\\3\end{bmatrix}+1\!\begin{bmatrix}-2\\2\end{bmatrix}=\begin{bmatrix}1\\11\end{bmatrix}=b$.
- **연결:** "방정식을 푼다 = ① 직선들이 어디서 만나나 ② $b$ 를 만드는 열들의 가중치는?" — *같은 해의 두 그림*. (정식 풀이·역행렬 → [[14 역행렬·연립방정식 — 특수 행렬·AX=B 풀이|14]])

![[MD4-rowcol Ax=b 행그림·열그림 (재구성).png|860]]

🔄 **재구성** [Gosamy/Strang 2절] — (왼쪽) 행 그림: 두 직선의 교점 = 해 $(3,1)$. (오른쪽) 열 그림: $3\cdot$col1 $+\,1\cdot$col2 $= b$.

- **내부/외부 프레임이 상위 호환이다.** "앞 열수=뒤 행수"(연립방정식 기원)와 같은 말이지만, $(M{\times}N)(N{\times}K){\to}M{\times}K$ 로 쓰면 **유효성 판정과 결과 크기가 한 줄**에 나온다. 행렬-벡터 곱의 방향 규칙($Av$=열벡터, $v^{\mathsf{T}}A$=행벡터)도 이 프레임의 특수 사례일 뿐.

- **"모든 쌍의 관계 저장"이 두 시점을 묶는다.** 위 '칸=내적' 시점의 재해석: 행렬곱 $C$ 는 왼쪽 *모든 행* × 오른쪽 *모든 열* 쌍의 내적(관계 점수)을 한 번에 저장한 표다. 공분산($X^{\mathsf{T}}X$)·상관 행렬·어텐션 점수표가 전부 이 구조 — "많은 내적의 컨테이너".
- **행렬-벡터 곱의 두 해석 = 이 원자의 두 시점 그대로.** ⑴ 선형 가중 결합 = *열 시점*(A 열들을 v 계수로 결합) ⑵ 기하 변환 = *연산자 시점*([[T06 벡터=상태 vs 행렬=연산자 (기저 재정의·SIMD·GEMM)|T06]]). 책이 §4.3.3에서 준 두 해석이 이미 우리 구조와 1:1.

> 항등·이동·스케일·회전·전단·반사 9종을 F 한 글자에 적용한 총람이 [[3D 변환 행렬 — 동차좌표·회전·투영 (선형대수 응용)#🧭 2D 아핀 변환 총람 — det 분류·고유벡터 사례 (2026-07 보강)|3D 변환 노트의 2D 총람 절]]에 있다 — det 3분류(+1/WH/−1), **원점 반사=180° 회전(det=+1)**, 전단·반사·스케일의 고유벡터, 회전의 실고유벡터 부재까지. 이 원자의 유효성·행렬식·고유벡터 맛보기가 전부 그 한 장에서 실물로 확인된다.

### 🔑 행렬식 = 면적 배율 + 방향 + 차원 붕괴 진단
- **크기:** |det| = 영역이 몇 배 — 변환이 공간을 얼마나 늘이/줄이나.
- **부호:** det<0 = 거울처럼 **뒤집힘**.
- **0의 의미가 핵심:** det=0 ⇔ **선형 종속**([[06 벡터 확장 — 집합·선형결합·선형독립·부분공간·생성·기저|06]]) ⇔ 계수(rank) 부족 ⇔ **역행렬 없음** ⇔ 연립방정식 해 유일하지 않음.

> [!warning] ⚠️ 영인자(Zero divisor) — 행렬곱은 실수 곱과 다르다
> 실수에선 $ab=0 \Rightarrow a=0$ 또는 $b=0$. **행렬은 아니다:** $A,B$ 둘 다 영행렬이 아닌데 $AB=O$ 일 수 있고, 이런 $A,B$를 **영인자**라 한다. 예: $\begin{bmatrix}1&0\\3&0\end{bmatrix}\begin{bmatrix}0&0\\0&-1\end{bmatrix}=\begin{bmatrix}0&0\\0&0\end{bmatrix}$.
> *왜?* 영인자는 모두 **det=0(특이·비가역)** — 한쪽이 차원을 붕괴시켜 다른 쪽의 상(image)을 0으로 보내기 때문. ⇒ 소거법칙 깨짐.

![[MD4-20·22·23 행렬식 면적배율 (재구성).png|820]]

🔄 **재구성** [그림 4-20·4-22·4-23] — (좌) 3·2배 확대 → det=6. (중) 방향 뒤집힘 → det<0. (우) 한 직선으로 붕괴 → det=0(선형 종속).
> [!quote]- 📖 책 원본 [그림 4-20·22·23] *(캡처 시 attachments 추가)*

## 🧩 코드 (§4.3~4.4)
> 🎯 목표: 행렬곱=합성(비교환) + 행렬식(면적·부호·0=종속).
> 코드: `~/LOCAL/03-00_STUDIES/AI/01_linear_algebra/md4_matmul_det.py` (검증: 합성 [-1,1]≠[-2,3]; det 6·1·−5·0)

```python
import numpy as np
from numpy.linalg import det
T1 = np.array([[0,-1],[1,0]]); T2 = np.array([[1,1],[0,1]]); v = np.array([1,2])
print((T2 @ T1) @ v)                 # [-1  1]  (회전→전단)
print((T1 @ T2) @ v)                 # [-2  3]  (전단→회전) ≠
b = lambda i,j: np.array([i,j]).T
print(round(det(b([3,0],[0,2]))))    # 6   면적 6배
print(round(det(b([1,0],[1,1]))))    # 1   전단=넓이 불변
print(round(det(b([-2,1],[1,2]))))   # -5  방향 뒤집힘
print(round(det(b([3,-1.5],[-2,1]))))# 0   선형종속=차원 붕괴
```

## 🆕 용어
| 용어 | 한 줄 정의 |
|---|---|
| **행렬 곱 = 변환 합성** | AB = "B 먼저, 그다음 A"; 함수 합성 A(B(v)); `@`/`matmul` |
| **행렬곱의 기원** | 연립방정식 Ax=b 표현(역사) + 변환 합성(기능); 계수행렬 A |
| **행렬곱 두 시점** | 칸=내적(Σaₖbₖ) / 열=선형결합(A 열들의 가중결합) |
| **row / column picture** | Ax=b를 직선들 교점 / b를 만드는 A열 결합 계수로 (Strang) |
| **영인자** (Zero divisor) | A,B≠O인데 AB=O; 모두 det=0(특이); 소거법칙 깨짐 |
| **비교환성** | AB ≠ BA — 변환 순서가 결과를 바꿈 |
| **행렬식** (determinant) | 면적(2D)·부피(3D) 배율; `np.linalg.det` |
| **det 부호 / det=0** | +유지·−반전 / 0=차원붕괴=선형종속=rank 부족=비가역 |
| **내부/외부 규칙** | (M×N)(N×K): 내부 N 일치=유효, 외부 M×K=결과 크기 |
| **"표준" 행렬 곱셈** | 아다마르·스칼라곱과 구별하기 위한 접두어; 표기는 기호 없이 AB(⊙는 아다마르) |
| **행렬-벡터 곱 방향 규칙** | Av(열벡터 뒤)·vᵀA(행벡터 앞)만 유효; 결과 방향=곱한 벡터 방향 |
| **행렬-벡터 곱 두 해석** | ① 열들의 선형 가중 결합(계수=벡터 원소) ② 기하 변환(회전+확대) |
| **고유벡터 맛보기** (예고) | Mv가 방향 유지·배율만 = v는 고유벡터, 배율=고윳값(12장) |
