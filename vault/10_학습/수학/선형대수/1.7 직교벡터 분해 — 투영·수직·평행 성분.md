# 직교벡터 분해 — 투영·수직·평행 성분

> 📐 **수학 개념원자(SSoT)** — 선형대수. 정의·성질·증명·기하 직관. AI 적용·연결은 코스 렌즈가 끌어다 쓴다.
> 🔗 [[_수학 MOC]] ｜ 🌐 [[_지식 연결 허브]] ｜ 🎓 AI 렌즈: [[05 직교벡터 분해 — 투영·수직·평행 성분|05(AI)]]

> **분해(decomposition([ˌdiːkɒmpəˈzɪʃən], 디컴포지션, 디컴포지션))** = 하나의 수학적 대상을 더 단순한 조각들의 **합**(또는 곱)으로 표현하는 것. 이 절은 벡터 하나 $t$ 를 **기준벡터 $r$ 과 직교하는 성분**과 **$r$ 과 평행한 성분**의 합으로 쪼갠다 — 전통·응용 선형대수 대부분이 이런 행렬·벡터 "분해"와 연관된다.

> 🎯 **핵심 (TL;DR)**
> - **직교 투영(orthogonal projection):** 벡터 $b$ 에서 벡터 $a$ 위로 내린 수선의 발 = $\beta a$, 여기서 $\beta=\dfrac{a^\mathsf{T}b}{a^\mathsf{T}a}$.
> - **유도 원리:** $b-\beta a$ 가 $a$(= $\beta a$)와 **직교할 때** 거리가 최소 → 내적 0 조건에서 $\beta$ 를 대수적으로 풀어냄.
> - **일반화(분해):** 목표벡터 $t$ = **평행 성분** $t_{\parallel r}=r\dfrac{t^\mathsf{T}r}{r^\mathsf{T}r}$ + **수직 성분** $t_{\perp r}=t-t_{\parallel r}$.
> - **검증:** $(t_{\perp r})^\mathsf{T}r=0$ — 두 성분은 항상 직교.
> - **왜 중요한가 🎯:** 이 절의 결과물(직교 투영법)이 **최소제곱식·통계학·머신러닝 등 많은 응용 분야의 기초** — 책이 §1.6을 마무리하며 직접 못박은 문장(⑦ 참조).
> - **쓰임 예고 🔭:** 그람-슈미트 과정·QR 분해 · 최소제곱법 · 통계 역문제 — **8·9·10장**에서 정식으로 확장.

> [!question] 🎯 필요성 사슬 (결핍 → 필요 → 발명)
> *출발 문제:* 벡터 $b$ 를 다른 벡터 $a$ 로 "설명"하고 싶다 — 그런데 대부분의 경우 $b$ 는 $a$ 의 스칼라배가 **아니다**(완전히 같은 방향이 아님).
> 1. $a$ 로 $b$ 를 최대한 근사하고 싶다 → $a$ 위의 점 중 $b$ 에 **가장 가까운 점**을 찾을 방법이 필요 → **직교 투영**($\beta a$, 최단거리 조건 = 직교).
> 2. 근사가 끝나면 "$a$ 가 못 설명한 나머지"가 남는다 → 그 나머지를 명시적으로 분리할 방법이 필요 → **잔차(수직 성분) $t_{\perp r}=t-t_{\parallel r}$**.
> 3. "설명된 부분(평행)"과 "설명 못한 부분(수직)"을 나누는 이 조작을 **여러 기준벡터로 반복**하면? → 서로 직교하는 축들을 만드는 절차가 필요 → **그람-슈미트 과정**(🔭 8장) — 이게 바로 회귀분석의 잔차, PCA의 사영, RAG 검색의 관련도 랭킹 밑바닥에 있는 하나의 연산이다.
> **한 줄:** 못 겹치는 두 벡터를 근사(평행)와 잔차(수직)로 쪼갠다(직교 투영) → 반복하면 직교 좌표계를 만든다(그람-슈미트) → 이게 최소제곱법의 심장이다.

> [!abstract] 📘 책 핵심 — [실전선대] §1.6 직교벡터 분해
> **① 분해란 무엇인가 — 두 스칼라 예시**
> - **덧셈 분해:** $42.01 = 42 + 0.01$ — 오차 제거 또는 데이터 압축이 목적일 수 있음(정수 42가 부동소수점 42.01보다 메모리를 덜 씀).
> - **곱셈 분해(소인수분해, prime factorization([praɪm ˌfæktəraɪˈzeɪʃən], 프라임 팩터라이제이션, 프라임 팩터라이제이션)):** $42 = 2\times3\times7$ — 수치 처리·암호학에 응용.
> - 공통점: **하나의 수학적 대상을 더 작고 간단한 조각으로 표현**한다는 것.
>
> **② 이 절의 분해: 투영 문제로 시작 (그림 1-6)**
> - 두 벡터 $a,b$ 가 있을 때, **$b$ 의 머리와 가장 가까운 $a$ 위의 점**을 찾는 문제 = 최적화 문제: 투영 거리 $\lVert b-\beta a\rVert$ 가 최소가 되는 스칼라 $\beta$ 를 구하라.
> - **핵심 통찰:** $b$ 의 머리와 $\beta a$ 가 **직각으로 만나도록** 그리면 그 점이 곧 최단 거리 점이다 — 원점·$b$ 의 머리·$\beta a$ 로 이루는 삼각형에서, $\angle\beta a$ 가 90°보다 작거나 클수록 $b-\beta a$ 의 길이는 **길어진다**(직각일 때 최단).
> - 종합: **$b-\beta a$ 가 $\beta a$ 와 직교 ⇔ 내적 0.**
>
> **③ 식 1-12 — β 유도**
> $$a^\mathsf{T}(b-\beta a)=0 \;\Rightarrow\; a^\mathsf{T}b-\beta a^\mathsf{T}a=0 \;\Rightarrow\; \beta=\frac{a^\mathsf{T}b}{a^\mathsf{T}a}$$
> (내적 분배 법칙 적용 — [[1.6 벡터 내적 — 점곱·기하·직교·아다마르곱·외적|1.6 내적 원자]]의 도구 재사용.) 이것이 **직교 투영법(orthogonal projection([ɔːrˈθɒɡənəl prəˈdʒɛkʃən], 오쏘거널 프러젝션, 오쏘고널 프로젝션))** — 최소제곱식·통계학·머신러닝의 기초(🔭 8·9·10장에서 정식으로 다룸).
>
> **④ 본격 분해 — 목표벡터·기준벡터 (그림 1-7)**
> - 용어: **목표벡터(target vector) $t$**, **기준벡터(reference vector) $r$**. 목표는 $t$ 를 두 벡터의 합으로 쪼개는 것: (1) 합이 $t$, (2) 하나는 $r$ 과 직교($t_{\perp r}$), 다른 하나는 $r$ 과 평행($t_{\parallel r}$).
> - **식 1-13 — 평행 성분:** $r$ 과 평행한 벡터는 $r$ 의 스케일 조정본이므로 식 1-12의 투영 공식을 그대로 벡터로 확장:
> $$t_{\parallel r}=r\,\frac{t^\mathsf{T}r}{r^\mathsf{T}r}$$
> (식 1-12와 차이: 거기선 스칼라 $\beta$ 만 구했지만, 여기선 **크기를 조절한 벡터** $\beta r$ 자체를 구한다.)
> - **수직 성분:** 두 성분의 합이 $t$ 라는 걸 이미 알고 있으므로 더 간단:
> $$t=t_{\perp r}+t_{\parallel r} \;\Rightarrow\; t_{\perp r}=t-t_{\parallel r}$$
>
> **⑤ 검증 — 수직 성분이 정말 기준벡터와 직교하는가**
> $$(t_{\perp r})^\mathsf{T}r=\Bigl(t-r\,\frac{t^\mathsf{T}r}{r^\mathsf{T}r}\Bigr)^{\!\mathsf{T}}\!r=0$$
> (책은 "대수는 간단하지만 지루해서" 증명을 생략 — 아래 🧠에서 직접 전개.)
>
> **⑥ 마무리 원리:** 분해의 세부 내용은 **제약 조건**(여기선 "직교 + 평행")에 따라 달라진다. 다른 제약이면 같은 벡터도 다른 방식으로 분해될 수 있다.
>
> **⑦ 책의 마무리 — 왜 이 절이 중요한가 (§1.6 종결 문단, 원문 취지 그대로):** 간단한 기하학적 그림(그림 1-6)에서 시작해 그 기하학의 함의를 탐구하고, 그 함의를 공식(식 1-12)으로 표현하고, 약간의 대수학까지 적용한 결과 **점을 최소 거리로 선에 투영하는 공식**을 발견했다 — 이를 **직교 투영법(orthogonal projection)** 이라 하며, 이는 선형 모델을 푸는 데 잘 알려진 **최소제곱식·통계학·머신러닝** 등 많은 응용 분야에서 **기초**가 된다(**8장·9장·10장**에서 직교 투영법을 정식으로 다룬다). → 이 문장이 이 노트를 "다시 찾아와야 할 이유"다: 8·9·10장을 공부할 때 이 원자(β 유도·직교 조건·분해 검증)로 되돌아와 복습할 것.
>
> **⑧ [실전선대] §1.7 1장 정리 — 이 절이 남긴 핵심 문구:** 책의 1장 요점정리(5개 항목 중 마지막)는 직교벡터 분해를 이렇게 못박는다 — "분해 공식은 기하학적으로 도출될 수 있지만, 공식이 내포한 개념인 **'크기에 대한 매핑'**이라는 문구를 기억해야 한다." 즉 책은 이 공식의 요점을 **유도 과정(기하)이 아니라 결과가 담은 의미(크기 매핑)**로 압축해서 기억하라고 지시한다.
> > [!info]- 🔍 해석 시도 (원문 뉘앙스 100% 확정은 어려움 — 대수식 구조 근거)
> > $\beta=a^\mathsf{T}b/a^\mathsf{T}a$ 와 $t_{\parallel r}=r\cdot t^\mathsf{T}r/r^\mathsf{T}r$ 는 둘 다 "**(내적) ÷ (기준벡터의 크기제곱 $\lVert r\rVert^2$)**" 형태 — 기준벡터 **자신의 크기로 정규화한 비율**을 구한 뒤, 그 비율만큼 기준벡터를 다시 스케일(매핑)하는 조작이다. "직선 위에서 가장 가까운 점을 찾는다"는 기하 묘사보다 "**한 벡터의 크기를 기준으로, 다른 벡터를 그 크기에 비례해 다시 그린다**"는 크기(magnitude) 중심 관점이 이 공식의 요지라는 뜻으로 읽힌다. → 🌉 다리(아래) 문단과 동일한 통찰의 책 자신의 표현.

### 📐 책이 생략한 두 대수 — 미적분 버전 증명 + 수직성 검증 전개
**① β 유도를 미적분으로도 할 수 있다(같은 답, 다른 길):** 책은 기하(직교조건)로 $\beta$ 를 구했지만, "거리 제곱을 최소화"하는 표준적인 방법으로도 같은 답이 나온다.
$$f(\beta)=\lVert b-\beta a\rVert^2=(b-\beta a)^\mathsf{T}(b-\beta a)=b^\mathsf{T}b-2\beta\,a^\mathsf{T}b+\beta^2 a^\mathsf{T}a$$
$a^\mathsf{T}a>0$(a≠0)이므로 이건 β에 대한 **아래로 볼록한 포물선** — 최솟값은 $\dfrac{df}{d\beta}=-2a^\mathsf{T}b+2\beta a^\mathsf{T}a=0$ 에서: $\beta=\dfrac{a^\mathsf{T}b}{a^\mathsf{T}a}$. **기하(직교) 경로와 미적분(도함수) 경로가 같은 곳에서 만난다** — 이게 곧 8~10장에서 미분 없이도 최소제곱법을 풀 수 있는 이유(직교 조건 = 미분 조건의 기하적 동치물).

**② 수직성 검증 — 책이 생략한 전개:**
$$\Bigl(t-r\frac{t^\mathsf{T}r}{r^\mathsf{T}r}\Bigr)^{\!\mathsf{T}}\!r = t^\mathsf{T}r-\frac{t^\mathsf{T}r}{r^\mathsf{T}r}\,r^\mathsf{T}r = t^\mathsf{T}r-t^\mathsf{T}r=0\qquad\blacksquare$$
($r^\mathsf{T}r$ 이 분모·분자에서 정확히 소거된다 — [[1.6 벡터 내적 — 점곱·기하·직교·아다마르곱·외적|내적 분배 법칙]]만으로 끝나는 짧은 증명.)

> [!warning] ⚠️ 정확성 — $\beta,\, t_{\parallel r}$ 는 $a,r \neq \vec 0$ 일 때만 정의된다   `#정확성`
> $\beta=a^\mathsf{T}b/a^\mathsf{T}a$ 와 $t_{\parallel r}=r\,t^\mathsf{T}r/r^\mathsf{T}r$ 모두 분모가 $a^\mathsf{T}a=\lVert a\rVert^2$·$r^\mathsf{T}r=\lVert r\rVert^2$ — **영벡터로는 나눌 수 없다.** 기준벡터가 영벡터면 "방향"이 없으므로 애초에 투영·분해 자체가 정의 불가.

### 🔑 왜 "직교+평행"인가 — 유일성의 근거
임의의 제약 조건으로 쪼개면 분해 방식이 무한히 많다(§⑥). "직교+평행"을 고르는 이유는 **이 조합만 유일한 해를 준다**는 데 있다 — $r$ 이 정해지면 $t_{\parallel r}$(식 1-13)과 $t_{\perp r}=t-t_{\parallel r}$ 이 **각각 정확히 하나로 결정**된다. 다른 방향으로 쪼개려 하면(예: $r$ 과 평행하지 않은 임의의 두 방향) 무수히 많은 분해가 가능해져 "그 벡터의" 성분이라 부를 수 없다. **직교성이 유일성을 보장**하는 구조적 이유 — 이게 그람-슈미트가 "직교" 기저를 고집하는 이유이기도 하다(🔭 8장).

> [!tip] 🪂 심화 — 소인수분해와 벡터 분해, 닮은 만큼 다른 것   `#사이드/심화`
> 책의 도입부 비유($42=2\times3\times7$)는 "분해=단순화"라는 큰 그림만 빌려온 것이지 수학적으로 동일한 연산은 아니다. 소인수분해는 **정수의 곱셈 구조**(唯一 인수분해 정리)를, 직교벡터 분해는 **벡터공간의 덧셈+직교 구조**를 각각 쓴다. 그래도 "복잡한 대상을 더 다루기 쉬운 조각으로" 라는 철학은 그대로 이어져 **행렬 분해(SVD·고윳값분해)** 로 계속된다 → [[왜 선형대수인가 — 데이터·알고리즘의 표준 언어|행렬 분해]].

> [!quote] 🌉 다리 — 직교 분해 = 그림자(내적)의 벡터 버전   `#연결/수학`
> [[1.6 벡터 내적 — 점곱·기하·직교·아다마르곱·외적|1.6 원자]]의 "내적 = 그림자 × 길이" 직관을 기억하면, $\beta=\dfrac{a^\mathsf{T}b}{a^\mathsf{T}a}=\dfrac{a^\mathsf{T}b}{\lVert a\rVert^2}$ 은 **"b가 a 위에 드리우는 그림자 길이를, a 자신의 길이로 정규화한 배율"**이다. 즉 직교 투영은 내적의 기하적 결과물을 좌표(스칼라 β)로 다시 읽은 것 — 내적이 "닮음을 재는 한 숫자"였다면, 투영은 "그 숫자만큼 a를 늘여 만든 새 벡터".

## 🧩 코드 실습 (§1.6)
> 🎯 목표: β·투영 벡터·직교/평행 성분을 코드로 재현하고, 미적분 경로와 기하 경로가 같은 β를 주는지, 수직성이 실제로 0인지 검증.
> 코드: `~/LOCAL/03-00_STUDIES/AI/01_linear_algebra/la1_6_orthdecomp.py`

```python
import numpy as np

a = np.array([4.0, 3.0])
b = np.array([2.6, 4.4])

# 식 1-12: 직교 투영 계수 β
beta = (a @ b) / (a @ a)
proj = beta * a
print("beta =", beta)                          # 0.944
print("직교성 확인 (b-βa)·a ≈ 0:", (b - proj) @ a)   # ≈ 0

# 미적분 경로 검산: f(β)=‖b-βa‖² 를 촘촘히 스캔해도 같은 β에서 최소
betas = np.linspace(0, 2, 20001)
f = [(b - x*a) @ (b - x*a) for x in betas]
print("수치 최소화로 찾은 β:", betas[np.argmin(f)])  # ≈ 0.944 (기하 경로와 일치)

# 식 1-13: 목표벡터 t를 기준벡터 r로 분해
r = np.array([5.0, 1.2])
t = np.array([2.6, 4.2])
t_par = r * (t @ r) / (r @ r)      # 평행 성분
t_perp = t - t_par                 # 수직 성분

print("t_par + t_perp == t:", np.allclose(t_par + t_perp, t))   # True
print("직교성 확인 (t_perp)·r ≈ 0:", t_perp @ r)                 # ≈ 0
```

관찰 포인트:
- **기하(직교조건)와 미적분(최소화) 두 경로가 같은 β를 준다** — 서로 다른 방법이 같은 답에 수렴하면 개념이 맞다는 신호.
- **분해 검증은 항상 두 가지:** ① 성분의 합이 원래 벡터인가(재구성) ② 두 성분이 정말 직교하는가(내적 0). 둘 다 확인해야 "분해"라 부를 수 있다.

## 🆕 새 용어 정리
| 용어 | 한 줄 정의 |
|---|---|
| **분해** (Decomposition) | 하나의 대상을 더 단순한 조각들의 합/곱으로 표현 |
| **직교 투영** (Orthogonal Projection) | b에서 a 위로 내린 수선의 발 βa; β=aᵀb/aᵀa |
| **목표벡터·기준벡터** (Target·Reference Vector) | 분해될 벡터 t / 분해 기준이 되는 벡터 r |
| **평행 성분** $t_{\parallel r}$ | r 방향으로 스케일된 성분; $r\,t^\mathsf{T}r/r^\mathsf{T}r$ |
| **수직 성분** $t_{\perp r}$ | r과 직교하는 나머지; $t-t_{\parallel r}$ |
| **소인수분해** (Prime Factorization) | 정수를 소수의 곱으로; 책의 "분해" 도입 비유 |
