# 역행렬·연립방정식 — 특수 행렬·AX=B 풀이

> 📐 **수학 개념원자(SSoT)** — 선형대수. 정의·성질·증명·기하 직관. AI 적용·연결은 코스 렌즈가 끌어다 쓴다.
> 🔗 [[_수학 MOC]] ｜ 🌐 [[_지식 연결 허브]] ｜ 🎓 AI 렌즈: [[14 역행렬·연립방정식 — 특수 행렬·AX=B 풀이|14(AI)]] ｜ 토대: [[2.2 행렬 곱셈·행렬식|행렬식·비가역]] · 다음: (고유값·특성다항식 — 예정)

> 🎯 **핵심 (TL;DR)**
> - **항등 행렬(identity([aɪˈdentɪti], 아이덴티티, 아이덴티티) $I$):** 대각=1·나머지=0. 곱해도 안 변함(=숫자 1). *변환을 취소해 원래 기저로 돌아온 상태.*
> - **역행렬($A^{-1}$) = 변환 *되돌리기*.** $A^{-1}A=I$. **존재 조건: $\det(A)\ne 0$**(정방·가역). det=0이면 역행렬 없음.
> - **연립방정식 = $AX=B$** (계수행렬 $A$·미지수 $X$·상수 $B$). 행렬엔 나눗셈이 없으니 **$X=A^{-1}B$** (양변에 $A^{-1}$ 곱해 $A$를 취소).
> - **특수 행렬:** 정방(행=열)·대각(대각만)·삼각(한쪽만, 풀기 쉬움·LU)·희소(대부분 0, 메모리 절약) + [실전선대] **난수**(`randn`)·**비정방**(행>열=높다·열>행=넓다)·**영 행렬**(`zeros`).
> - ⚠️ 표기: [실전선대] **단위 행렬 = 항등 행렬**(identity, 같은 것·번역만 다름). 크기는 아래 첨자 $I_5$.
> - 코드: `np.linalg.inv(A)`(근사) / `sympy.Matrix(...).inv()`(정밀, 곱은 `*`). 풀이 `inv(A) @ B` 또는 `A.inv()*B`. 생성 전용 함수: `randn`·`diag`·`triu/tril`·`eye`·`zeros`.

> [!tip] 🗺️ "○○곱" 헷갈릴 때 → [[_용어집#🔢 곱(product) 연산 한눈 지도|곱 연산 한눈 지도(용어집)]]
> $A^{-1}A$·$AX$ 모두 *행렬곱*(변환 합성). `@`/`matmul`(numpy), `*`(sympy).

> [!question] 🎯 필요성 사슬 (결핍 → 필요 → 발명)
> *출발 문제(구체적 목표):* "변환을 잇고(행렬곱), det=0이면 정보를 잃어 못 푼다"까지 왔다. 이제 **변환을 실제로 *되돌려* 연립방정식의 답을 꺼내야** 한다.
> 1. $AX=B$ 에서 $X$를 어떻게 꺼내나? 실수면 양변 $\div A$ → 그런데 **행렬엔 나눗셈이 없다** → "곱하면 원상복구되는" 짝 필요 → **역행렬 $A^{-1}$** ($A^{-1}A=I$).
> 2. "원상복구된 상태"를 가리킬 기준이 필요 → **항등 행렬 $I$**(=숫자 1의 역할) → $X=A^{-1}B$.
> 3. $A^{-1}$ 가 *항상* 있나? det=0(차원 붕괴)이면 되돌릴 수 없음 → **가역 조건 $\det(A)\ne 0$**.
> 4. 큰 시스템을 *효율적으로* 풀고 저장하려면? → **특수 행렬**(삼각=풀기 쉬움·LU, 대각=스칼라, 희소=메모리 절약).
> **한 줄:** 나눗셈 없는 행렬 세계에서 *되돌리는 도구* = 역행렬($A^{-1}A=I$) → 가역 조건 det≠0 → $X=A^{-1}B$ 로 연립방정식 해결, 특수 행렬이 그 계산을 떠받친다.

> [!abstract] 📘 책 핵심 — [필수수학] §4.5 특수 행렬 · §4.6 연립방정식과 역행렬
> **① §4.5 특수 행렬**
> - **정방 행렬(square):** 행 수 = 열 수. 선형 변환 표현·고유값 분해(eigen decomposition) 등에 필수.
> - **항등 행렬(identity, $I$):** 대각선=1, 나머지=0인 정방 행렬. "1을 곱하는 것"과 같아 변환에 영향 없음. *항등으로 만들 수 있다 = 변환을 취소하고 원래 기저를 찾았다*(§4.6의 핵심).
> - **역행렬(inverse, $A^{-1}$):** 다른 행렬의 변환을 *취소*. $A^{-1}A=I$. 계산은 §4.6.
> - **대각 행렬(diagonal):** 대각선만 값, 나머지 0. 축마다 단순 스칼라 신축.
> - **삼각 행렬(triangular):** 대각선과 *한쪽*(위 또는 아래)만 값. 연립방정식 풀기 쉬워 수치해석에서 선호(LU 분해).
> - **희소 행렬(sparse):** 대부분 0. 0을 저장하지 않고 비영 원소만 기록 → 메모리·계산 효율(컴퓨팅 관점의 이득).
>
> **② §4.6 연립방정식과 역행렬**
> - 연립방정식을 **$AX=B$** 로 표현: 계수 → $A$, 우변 상수 → $B$, 미지수 → $X$.
> - 푸는 법: $A$의 변환을 취소해 $X$를 분리 → $A^{-1}AX=A^{-1}B$ → $IX=A^{-1}B$ → **$X=A^{-1}B$**.
> - $A^{-1}A=I$ 는 "1을 곱함"과 같아 $x,y,z$ 를 효과적으로 분리. 손계산은 **가우스 소거법(Gaussian elimination)** — 책은 컴퓨터에 맡김.
> - 예제 4-17~4-19: `sympy`(정밀·분수, 곱은 `*`) / `numpy`(부동소수 근사, `inv`). 예시 해 $x{=}2,\ y{=}34,\ z{=}-8$.

> [!abstract] 📘 책 핵심 — [실전선대] §4.1.2 특수 행렬
> - **프레임:** 행렬의 수는 무한하지만 **적은 특성만으로 '군/범주'에 속하게** 설명 가능. 특정 연산에서 등장하는 범주가 있고 범주마다 유용한 속성 → 꼭 알아야 함. 자주 쓰이는 범주는 **전용 NumPy 함수**가 있을 정도. (더 많은 특수 행렬은 뒷부분에서 — `(🔭 예고)`)
> - **난수 행렬(random numbers matrix):** 가우스(정규) 등 분포에서 무작위 추출한 원소. 임의의 크기와 계수(rank — 부록, `(🔭 예고)`)로 쉽고 빠르게 생성 가능해 **선형대수 코드 공부에 유용**. `A = np.random.randn(Mrows, Ncols)` (Mrows=shape 0, Ncols=shape 1).
> - **정방/비정방:** 정방 = 행 수 = 열 수 = $\mathbb{R}^{N\times N}$. 비정방은 **행>열 = 높다**, **열>행 = 넓다**.
> - **대각(diagonal):** 왼쪽 위에서 오른쪽 아래로 내려가는 원소들. 대각 행렬 = 비대각 원소 전부 0(대각만 0 아닌 값 가능). **`np.diag()` 이중 동작:** 행렬 입력 → 대각 원소 *벡터* 반환 / 벡터 입력 → 그 원소가 대각선인 *행렬* 반환. ⚠️ 대각 요소 *추출*을 '대각화'라고 **부르지 않는다** — 대각화 연산은 12장 `(🔭 예고 — 12장)`.
> - **삼각(triangular):** 주 대각선 위 또는 아래가 모두 0. 비영 원소가 위 = **상삼각**, 아래 = **하삼각**. 추출 전용 함수 `np.triu()` / `np.tril()`.
> - **단위 행렬(identity) = 특수 행렬 중 가장 중요.** 곱해도 동일(숫자 1과 동등). **모든 대각 원소가 1인 정방 대각 행렬.** 문자 $I$, 크기는 아래 첨자($I_5$ = 5×5); 숫자 없으면 문맥(방정식 성립)으로 추론. `np.eye()`.
> - **영 행렬(zeros):** 영벡터와 유사 — 전부 0, 굵은 $\mathbf{0}$ 로 표기. 벡터·행렬이 같은 기호인 **중의적 표현은 수학·과학 표기법에서 흔함** → 넘어가자. `np.zeros()`.
> - 그림 4-2: 6종(정사각·상삼각·하삼각·대각·단위·제로) — 숫자와 그레이스케일값 = 원소 값.

## 🧠 근본 이해 · 융합 · 통찰

### 🔑 역행렬 = "되돌리기" 버튼 (나눗셈의 빈자리)
- 숫자 세계: $ax=b \Rightarrow x=b/a = a^{-1}b$. **행렬 세계엔 나눗셈이 없다** → 그 자리를 **$A^{-1}$** 가 메운다. $a\cdot a^{-1}=1$ 의 행렬판이 $A A^{-1}=I$.
- 기하적으로: $A$ 가 공간(격자)을 비틀면, $A^{-1}$ 가 **정확히 거꾸로 비틀어 원래로** 되돌린다. 합치면 아무 일도 안 한 것($I$).
- 그래서 **$I$ = "변환을 취소한 상태" = 원래 표준기저**. "기저=공간의 자"와 직결.

![[MD4-inverse 역행렬 되돌리기 (재구성).png|860]]

🔄 **재구성** [§4.5~4.6 개념] — (좌) 원래 격자·기저 → (중) $A$ 적용해 격자가 비틀림 → (우) $A^{-1}$ 적용해 **원래로 복원**. $A^{-1}A=I$.
> [!quote]- 📖 책 원본 [§4.6 $A^{-1}A=I$ 식] *(캡처 시 attachments 추가)*

### 🔑 연립방정식 = $AX=B$, 답 = $A^{-1}B$ (Gosamy 두 그림의 결말)
- 연립방정식 $\begin{cases}4x+2y+4z=44\\5x+3y+7z=56\\9x+3y+6z=72\end{cases}$ → $AX=B$. **"$X$를 $A$로 변환하면 $B$"** 라는 뜻.
- 거꾸로 **"$B$에서 $A$를 벗겨내면 $X$"** → $X=A^{-1}B$. 계산하면 $X=[2,\,34,\,-8]^{\mathsf{T}}$.
- **[[행렬 곱셈의 두 관점 (Gosamy 원문 정리)|Gosamy]] 두 그림의 결말:** row picture는 "세 평면(직선)의 교점", column picture는 "$B$를 $A$ 열들로 만드는 가중치 $X$". 역행렬은 **그 교점/가중치를 한 번에 계산**하는 기계.
- ⚠️ **존재 조건:** $A^{-1}$ 는 $\det(A)\ne 0$ 일 때만. det=0(선형종속)이면 평면들이 겹치거나 평행 → 유일해 없음(역행렬 없음).

### 🔑 특수 행렬 = "형태가 곧 성질" (계산 효율의 언어)
- **삼각·대각**은 *형태만 봐도* 풀이가 쉽다: 대각=축별 스칼라(역행렬도 그냥 역수), 삼각=대입으로 순차 풀이(LU 분해의 토대).
- **희소**는 *수학적 성질*이 아니라 **컴퓨팅 기회**: 0을 안 저장 → 대부분 0인 거대 행렬을 다룰 수 있게 함.

![[MD4-special 특수행렬 6종 지도 (재구성).png|880]]

🔄 **재구성** [§4.5 6종] — 정방(행=열)·항등($I$, 대각1)·대각·삼각(한쪽만)·희소(대부분 0)·역행렬($A^{-1}A=I$, det≠0). *형태 = 성질.*
> [!quote]- 📖 책 원본 [§4.5 특수 행렬 예시들] *(캡처 시 attachments 추가)*

### 🔑 [실전선대] 렌즈 — 무한한 행렬을 '범주'로 길들이기 (§4.1.2 흡수)
- **군/범주 프레임:** 행렬은 무한하지만(숫자 구성 방법이 무한) *적은 특성*이 범주를 정한다. 범주 = **연산의 지름길** — 형태만 보고 성질(풀이 난이도·저장 비용)을 예측할 수 있고, 자주 쓰여 *전용 생성 함수*까지 있다.
- ⚠️ **번역 충돌 — 단위 행렬 = 항등 행렬:** [실전선대]의 "단위 행렬"과 [필수수학]의 "항등 행렬"은 **같은 identity matrix**다. *단위벡터(unit vector)의 unit과 무관* — 혼동 주의. 크기 표기 $I_5$(아래 첨자), 생략 시 방정식이 성립하도록 문맥 추론.
- **`np.diag` 이중 동작 = 이름 하나, 기능 둘:** 행렬→벡터(대각 *추출*), 벡터→행렬(대각 *생성*). 그리고 추출은 **대각화가 아니다** — 대각화(diagonalization)는 12장의 별개 연산. `(🔭 예고 — 12장)`
- **기호 중의성은 관례다:** 굵은 $\mathbf{0}$ 는 영벡터도 영행렬도 가리킨다. 기호가 문맥에 기대는 건 수학 표기법의 일상 — $I$ 크기 추론과 같은 원리.
- **난수 행렬의 실용성:** 임의 크기·계수(rank)를 즉시 만들 수 있는 *실험 재료* — 이후 장들의 코드 예제가 대부분 `randn` 위에서 돈다.

## 🧩 코드 실습 (§4.6)
> 🎯 목표: $AX=B$ 를 $X=A^{-1}B$ 로 풀기 + $A^{-1}A=I$·복원 확인. sympy(정밀)와 numpy(근사) 비교.
> 코드: `~/LOCAL/03-00_STUDIES/AI/01_linear_algebra/md4_inverse_solve.py` (검증: det=6, X=[2,34,−8], A⁻¹A=I, A⁻¹(Av)=v)

```python
import numpy as np
from numpy.linalg import inv, det
A = np.array([[4,2,4],[5,3,7],[9,3,6]], float)
B = np.array([44,56,72], float)
print(round(det(A)))            # 6  → ≠0, 가역
print(inv(A) @ A)               # 항등 I (반올림하면 대각 1)
print(inv(A) @ B)               # [ 2. 34. -8.]  ← 연립방정식 해
# sympy: 정밀(분수) — 행렬곱은 @ 아닌 *
from sympy import Matrix
A_, B_ = Matrix([[4,2,4],[5,3,7],[9,3,6]]), Matrix([44,56,72])
print((A_.inv() * B_).T)        # Matrix([[2, 34, -8]])
```
관찰: **det≠0이라 역행렬 존재 → $X=A^{-1}B$ 한 번에 해. sympy는 분수로 정밀, numpy는 부동소수 근사.**

## 🆕 새 용어 정리
| 용어 | 한 줄 정의 |
|---|---|
| **정방 행렬** (square) | 행 수 = 열 수; 선형변환·역행렬·고유값의 전제 |
| **항등 행렬** (identity, $I$) | 대각=1·나머지=0; "×1"=변환 취소; $AI=IA=A$ |
| **역행렬** ($A^{-1}$) | 변환을 되돌림; $A^{-1}A=I$; det≠0일 때만 존재 |
| **대각 행렬** (diagonal) | 대각만 값; 축별 스칼라 신축; 역행렬=대각 역수 |
| **삼각 행렬** (triangular) | 대각+한쪽만; 연립방정식 풀기 쉬움(LU 분해) |
| **희소 행렬** (sparse) | 대부분 0; 비영 원소만 저장(메모리·계산 효율) |
| **연립방정식 = AX=B** | 계수 A·미지수 X·상수 B; 해 X=A⁻¹B |
| **가우스 소거법** | 손으로 해/역행렬 구하는 소거 절차(책은 컴퓨터에 위임) |
| **난수 행렬** (random) | 분포(주로 가우스)에서 추출한 원소; `randn(행,열)`; 임의 크기·rank 실험 재료 |
| **비정방 행렬** (nonsquare) | 행≠열; 행>열=**높다**(tall)·열>행=**넓다**(wide) |
| **단위 행렬 = 항등 행렬** | 같은 identity matrix(번역 차); $I_5$=5×5; 단위벡터의 unit과 무관 |
| **np.diag 이중 동작** | 행렬→대각 벡터 추출 / 벡터→대각 행렬 생성; 추출≠대각화(12장) |
| **상/하삼각 추출** | `np.triu()`(위 유지) / `np.tril()`(아래 유지) |
| **영 행렬** (zeros) | 전부 0; 굵은 $\mathbf{0}$(영벡터와 같은 기호 — 중의적 관례); `np.zeros()` |

| 상/하삼각 추출 함수? | `np.triu()` / `np.tril()` |
| 난수 행렬 생성? | `np.random.randn(행, 열)` — 가우스; 임의 크기·rank 실험용 |
| 영 행렬 표기? | 굵은 **0** — 영벡터와 같은 기호(중의적 표기는 관례); `np.zeros()` |
