> 🎯 **핵심 (TL;DR)**
> - **행렬 공간 = 행렬의 서로 다른 특징들 사이의 선형 가중 결합** — 어렵지 않다, 결국 [[1.8 벡터 확장 — 집합·선형결합·선형독립·부분공간·생성·기저|1.8]]의 확장.
> - **열공간 $C(A)$ = 행렬을 열벡터 집합으로 보고, 무한한 실수 스칼라로 만든 *모든* 선형 가중 결합** = 열들의 생성(span). $\in$ = '에 포함됨'.
> - **소속 판별이 핵심 질문:** $v\in C(A)$? — 열들의 가중 결합으로 $v$를 만들 수 있나. 이 질문이 **선형 최소제곱법의 시작** `(🔭 예고 — 10장)`.
> - 📌 **N개 열 ≠ N차원 열공간** — 차원 = **선형 독립인 열의 수**(1.8 그대로). 종속 2열이면 여전히 1D 선.
> - 열공간은 전체 $\mathbb{R}^M$ 이 되거나 그 안의 더 작은 부분공간(선·평면 — "우주를 가로지르는 얇은 종이").
> - **행공간 $R(A)$ = 완전히 동일한 개념, 열 대신 행** — $R(A)=C(A^{\mathsf{T}})$ (전치 하나로 끝). **대칭 행렬이면 $R(A)=C(A)$.**
> - **영공간 $N(A)$ = $Ay=0$ 을 만족하는 0 아닌 $y$ 들의 집합** 📌 — 자명해 $y{=}0$ 은 제외. **열이 선형 독립 ⇔ $N(A)=\{\}$ (빈 집합).**
> - **영공간 ⊥ 행공간** — $Ay=0$ = 모든 행과의 내적 0. 영공간은 **고유벡터·특이벡터 찾기(12장)의 핵심** `(🔭)`.
> - 정량 판별 도구 = **행렬 계수(rank)** `(🔭 예고 — 장 후반)` · 영공간 차원↔열 독립성의 정확한 관계 = **계수-영공간 차원(nullity) 정리** `(🔭 다음 장)`.

> [!question] 🎯 필요성 사슬 (결핍 → 필요 → 발명)
> *출발 문제:* $Av$ 가 만들 수 있는 벡터는 어디까지인가 — 방정식 $Ax=b$ 에 해가 *있기는 한가*?
> 1. 행렬의 열들이 "도달할 수 있는 범위"를 부를 이름이 없음 → 열들의 모든 선형 가중 결합 → **열공간 $C(A)$** (1.8 생성에 행렬 표기를 입힘)
> 2. 어떤 벡터가 그 범위 안인지 밖인지 판별해야 함($Ax=b$ 해 존재 = $b\in C(A)$) → **소속 판별** — 추측·산술·시각화로는 고차원 불가
> 3. 열이 많다고 공간이 넓어지지 않음(종속 열 = 재탕) → 차원을 세는 잣대 필요 → **독립 열 수 = 차원** → 정량화 = **rank** `(🔭 예고)`
> 4. 열로 한 일을 행에도 적용하고 싶음 → 새 이론 불필요, 전치가 있으니 → **행공간 $R(A)=C(A^{\mathsf{T}})$** (2.3 재사용)
> 5. "결합으로 $b$ 를 만들 수 있나"의 쌍둥이 질문 — "결합으로 **0** 을 만들 수 있나(0 아닌 가중치로)?" → 있다면 열들이 종속(1.8 식 2-3 그대로) → 그 가중치 전부의 집합 = **영공간 $N(A)$** — 고유벡터 사냥의 무기 `(🔭 12장)`
> **한 줄:** 도달 범위(열공간·행공간) → 이 벡터가 안인가(소속 판별=최소제곱 시작) → 0을 만드는 결합이 있나(영공간=종속의 증거) → 정량화는 rank·nullity.

> [!abstract] 📘 책 핵심 — [실전선대] §5.2 행렬 공간 · §5.2.1 열공간
> **① §5.2 — 행렬 공간(열, 행, 영)**
> - 전통·응용 선대의 핵심 개념이지만 어렵지 않음 — **근본적으로 행렬의 서로 다른 특징들 사이의 선형 가중 결합**.
>
> **② §5.2.1 — 열공간**
> - 벡터의 선형 가중 결합에서 **두 가지 수정**: ⑴ 행렬을 **열벡터의 집합**으로 간주 ⑵ 특정 스칼라 집합 대신 **무한한 실수 스칼라** 적용 → 생성된 무한 벡터 집합 = **행렬의 열공간**. 표기 $C(A)$, $\in$='의 구성원/에 포함됨', λ=가능한 모든 실숫값.
> - **예 1 (열 1개):** $C\!\left(\begin{bmatrix}1\\3\end{bmatrix}\right)=\lambda\begin{bmatrix}1\\3\end{bmatrix}$ — 열벡터 방향으로 원점 통과 양방향 무한선 = **1차원 부분공간**(그림 5-1). *(각주 감각: 엄밀히 '무한대는 실수가 아니라' 문자 그대로 무한 확장은 아니지만, 어떤 목적에든 충분히 — 인간 지식으로 헤아릴 수 있는 것보다 — 길다.)*
>   - 소속 판별: $[1,3]^{\mathsf{T}}\in C$? 예(λ=1). $[-2,-6]^{\mathsf{T}}$? 예(λ=−2). $[1,4]^{\mathsf{T}}$? **아니오** — 만들 스칼라가 없다.
>   - ⚠️ **λ 신분 + 모양 혼동 주의:** 여기의 λ는 특정 값이 아니라 **전 실수를 훑는 자유 변수**(집합 생성 매개변수) — 위 식은 방정식이 아니라 집합 $\{\lambda[1,3]^{\mathsf{T}} \mid \lambda\in\mathbb{R}\}$ 의 압축 표기다. 생김새가 거의 같은 고윳값 식 $Av=\lambda v$(λ=**행렬이 정해주는 특정 상수** `🔭`)와 신분이 다름: 전자="v의 배수 **전부**", 후자="A가 v를 몇 배로 늘리나". λ 신분 지도 → [[1.3 벡터 연산 — 덧셈·뺄셈·스케일링·전치·브로드캐스팅|1.3]].
> - **예 2 (독립 2열):** $C\!\left(\begin{bmatrix}1&1\\3&2\end{bmatrix}\right)=\lambda_1[1,3]^{\mathsf{T}}+\lambda_2[1,2]^{\mathsf{T}}$ — 답: **$\mathbb{R}^2$ 전체**. 예: $[-4,3]^{\mathsf{T}}$ = 두 열을 11과 −15로 가중(스칼라 찾기 = 최소제곱법 `(🔭 예고 — 10장)`). 그림 5-2A.
> - **예 3 (종속 2열):** $C\!\left(\begin{bmatrix}1&2\\3&6\end{bmatrix}\right)$ — 두 열이 같은 직선상 → 열공간 = **여전히 선(1차원)**. 📌 **N개 열이라고 무조건 N차원이 아니다** — 열들이 **선형 독립 집합일 때만** 열 수 = 차원. *(2026-07-13 정정: 이 [1,3]/[2,6] 예시는 그림 5-2B가 아니라 §2.3 선형독립 예시 $S=\{[1,3]^{\mathsf{T}},[2,6]^{\mathsf{T}}\}$ 그 자체 — [[1.8 벡터 확장 — 집합·선형결합·선형독립·부분공간·생성·기저|1.8]]에서 재확인됨.)*
> - **예 4 (R³ 속 독립 2열):** $C\!\left(\begin{bmatrix}3&0\\5&2\\1&2\end{bmatrix}\right)$ — 두 열 독립 → 열공간은 2차원이지만 **$\mathbb{R}^3$ 에 포함된 2D 평면**(그림 5-3) — "무한하지만 3차원에선 지극히 작은 조각, **우주를 무한히 가로지르는 얇은 종이**".
> - 평면 안에도 많은 벡터가 있지만 **밖에 더 많다** — 열공간 안 vs 밖.
> - **벡터가 열공간에 있는지 어떻게 아나? = 선형 최소제곱법의 시작** (응용 수학·공학에서 중요성은 아무리 강조해도 지나치지 않음). 지금까지는 추측·산술·시각화 — 직관용일 뿐 고차원·복잡 문제는 불가 → **정량적 방법 = 행렬 계수(rank)** `(🔭 예고 — 장 후반)`.
> - 그 전 직관: 행렬의 열이 벡터 부분공간을 구성 — 전체 M차원 공간이 되거나 더 작은 부분공간. **중요한 것 = 다른 벡터가 그 부분공간 안에 있는가**(열들의 선형 가중 결합으로 표현 가능한가).

> [!abstract] 📘 책 핵심 — [실전선대] §5.2.2 행공간
> - 열공간을 이해했으면 행공간은 쉽다: **완전히 동일한 개념, 단지 열 대신 행**으로 가능한 모든 가중 결합.
> - 표기 $R(A)$. 전치가 행↔열 교환이므로 **행공간 = 전치된 행렬의 열공간: $R(A)=C(A^{\mathsf{T}})$.**
> - 열공간과의 차이 예: 행공간(열공간은 아님)은 행 축소 연산에 대한 성질이 다름 — 책은 여기서 안 다룸 `(🔭 10장)`. *(한글 대조 완료(2026-07-13, 사용자 제공 스캔): 번역서 원문이 실제로 "행공간(열공간은 아님)은 행 축소 연산의 **변형**입니다"라고 정확히 쓰여 있음 — 내 인용 오류 아님. 영문 원서는 UHD가 구할 수 없어 대조 자체를 종결(2026-07-13) — 대신 **수학으로 직접 검증**: 기본행연산 $E$(가역)로 $A'=EA$ 를 만들면 $A=E^{-1}A'$ 도 성립해 두 방향 다 서로의 선형결합 → $\mathrm{rank}([A;A'])=\mathrm{rank}(A)=\mathrm{rank}(A')$(수치 확인 완료) → **행공간은 행 축소에 불변(invariant)이 수학적 사실**. "변형"이라는 한글 표현은(번역 경위와 무관하게) 문자 그대로 "행공간 자체가 바뀐다"로 읽으면 틀렸다 — 바뀌는 건 그 공간을 표현하는 구체적 행벡터들이지, 공간(span) 자체가 아니다.)*
> - $R(A)=C(A^{\mathsf{T}})$ 이므로 **두 행렬 공간은 대칭 행렬일 때 동일**: $R(A)=C(A)$.

> [!abstract] 📘 책 핵심 — [실전선대] §5.2.3 영공간
> - 열공간 요약식 $Ax=b$: "A의 열의 가중 결합으로 $b$ 를 만드는 가중치 $x$ 있나?" — '예'면 $b\in C(A)$.
> - 영공간 요약식 $Ay=\mathbf{0}$: "**열의 가중 결합이 영벡터가 되는** 가중치 $y$ 있나?"
> - 모든 행렬에 성립하는 정답 $y=\mathbf{0}$(모든 열에 0 곱)은 **뻔한 답, 의미 없음** → 질문은 "**모든 값이 0이 아닌** 가중치 집합?" 이를 만족하는 $y$ 들이 **$A$ 의 영공간 $N(A)$**.
> - **예 1:** $\begin{bmatrix}1&-1\\-2&2\end{bmatrix}$ — 저자는 $[7.34, 7.34]$ 를 찾음; $[1,1]$·$[-1,-1]$·$[2,2]$ 도 됨 → $Ay=0$ 인 $y$ 는 **무한히 존재**, 전부 한 벡터의 크기 조정: $N(A)=\lambda\begin{bmatrix}1\\1\end{bmatrix},\ \lambda\in\mathbb{R}$ *(또 λ — 자유 변수 신분)*.
> - **예 2:** $\begin{bmatrix}1&-1\\-2&3\end{bmatrix}$ — 벡터를 못 찾는 이유는 **애초에 영공간이 없기 때문**(여러분을 높게 평가합니다!): $N(A)=\{\,\}$ (빈 집합).
> - **열의 독립성과의 관계(우연 아님):** 영공간 차원 수 ↔ 열의 선형 독립성 — 정확한 본질은 다음 장 **계수-영공간 차원(nullity([ˈnʌlɪti], 널리티, 널리티)) 정리** `(🔭)`. 핵심: **열이 선형 독립 집합이면 영공간이 비어 있다.** 재강조(매우 중요): **최대계수·최대열계수 행렬 = 빈 영공간, 축소계수 행렬 = 비어 있지 않음.** 📌
> - SciPy: `scipy.linalg.null_space(A)` → `[[0.70710678],[0.70710678]]`, `null_space(B)` → `[]`. 왜 1이 아니라 0.70710678? 무한 후보 중 파이썬은 **단위 벡터**($\sqrt{1/2}\approx.70711$)를 반환 — 단위벡터는 수치적 안정성 등 좋은 특성 → **컴퓨터 알고리즘은 종종 단위벡터를 부분공간의 기저로 반환**(고유벡터·특이벡터에서 재이용 `🔭`).
> - **그림 5-4(행벡터로 그린 이유): 영공간은 행공간과 직교!** $Ay=0$ 을 행별로 쓰면 각 행 $a_i$ 에 대해 $a_i y=0$ — **영공간 벡터와 각 행의 내적이 전부 0**.
> - 왜 다들 영공간에 관심? "곱해서 영벡터"가 이상해 보이지만 — **12장 고유벡터·특이벡터를 찾는 데 핵심 역할** `(🔭)`.
> - **네 부분공간:** 모든 행렬엔 4개의 연관 부분공간 — 열·행·영 + **왼쪽 영공간 = $N(A^{\mathsf{T}})$**(행의 영공간, 전치의 영공간). 전통 교과과정은 그 복잡성·관계에 많은 시간을 쓰지만 — 지금은 넘어감.

> [!question] 🎯 가설 → 검증 → 정정
> 질문(외부 대화): "열공간 속 각 벡터가 저마다 1차원 열공간을 가진다는 게 벡터 부분공간이라는 뜻일까?" — 이 "벡터별 1D 파편화" 가설은 외부 AI가 정확히 반박했다: 부분공간은 벡터 하나하나가 아니라 **열 전체의 결합이 만드는 영역 하나**다. 여기까진 맞다.
> **그런데 외부 AI의 결론 — "부분공간은 새 개념이 아니라 열공간 그 자체를 부르는 또 다른 이름" — 은 과잉 일반화다.** 정확한 관계는 동의어가 아니라 **상위 범주(부분공간) ⊃ 특정 사례(열공간)**: "열공간은 부분공간의 한 예"이지 "부분공간=열공간"이 아니다.
> **결정적 반증은 이 노트 자체다:** 아래 "📋 네 부분공간 지도" 표에서 이 장 하나에만 **"부분공간"이라 불리는 것이 넷**(열공간·행공간·영공간·왼쪽 영공간)이다. 넷 다 부분공간이면서 서로 명백히 다른 대상 — "부분공간=열공간"이 참이라면 나머지 셋은 애초에 성립 불가능하다. [[1.8 벡터 확장 — 집합·선형결합·선형독립·부분공간·생성·기저|1.8]]의 정의도 "덧셈·스칼라곱에 닫힌(원점 포함) 부분집합"이라는 **구조적 성질**이지 "열공간"이라는 특정 대상을 가리키는 고유명사가 아니다.
> **책 원문(위 60행 인용)은 실제로 정확했다:** "행렬의 열이 벡터 부분공간을 **구성한다**" — '구성한다'는 "이 결합이 부분공간 자격을 만족하는 하나의 사례를 만든다"는 뜻이지 "부분공간=열공간"이라는 재정의가 아니다. 외부 AI 자신의 "대한민국(전체공간)/부산(열공간=부분공간)" 비유도 자기모순적: 서울도 대한민국의 도시이듯 행공간·영공간도 "부분공간"이라는 같은 범주의 다른 사례이므로, 그 비유를 끝까지 밀면 "도시=부산"이 아니라 **"부산은 도시의 한 예"**가 맞는 결론이었어야 한다.
>
> 한 줄: 부분공간은 **범주**(닫힘·원점포함 조건을 만족하는 모든 것), 열공간은 그 범주에 속하는 **한 특정 사례**(행공간·영공간도 마찬가지) — "구성한다" ≠ "~이다".

> - **책의 의도:** (A)=원재료(두 열 그 자체) → (B)=그 둘을 결합하면 새 점(2,5)에 닿는다는 것을 보여줘서, 바로 다음 "λ를 바꿔가며 결합하면 $\mathbb{R}^2$ 전체에 닿는다"는 주장을 시각적으로 준비시키는 그림.

### 🔑 선형 결합 = 평균이 아니다 — 후속 대화로 검증 (2026-07-13)

> [!question] 🎯 막힌 점 → 해소 여정 (2단계)
> **막힌 점 1:** "벡터마다 결합하면 기울기가 계속 달라지는데, 어떻게 고정된 열공간(평면 전체)을 갖는다는 거지?" → **해소:** 열공간은 "하나의 도형·기울기"가 아니라 **λ를 튜닝해 도달 가능한 점 전체(영역)**. λ가 연속으로 바뀌면 도달점도 연속으로 바뀌고, 그 궤적 전부가 영역(=열공간)이 되는 것 — 애초에 "고정된 기울기"를 찾으려 한 게 잘못된 질문틀이었다.
> **막힌 점 2 (더 근본적):** "선형 결합 = 두 벡터를 평균/중간 기울기로 섞는 것"이라는 오해. → **해소:** 실제로는 **순차 이동(꼬리 물기)** — $\lambda_1$ 방향으로 $\lambda_1$ 만큼 간 뒤, 그 지점에서 $\lambda_2$ 방향으로 $\lambda_2$ 만큼 더 간다. "평균처럼 보인" 건 $\lambda_1{=}\lambda_2{=}1$ 특수 케이스의 **최종 결과 벡터**(기울기 2.5, 원래 2·3의 중간)만 보고 과정을 거꾸로 추측했기 때문 — 결과가 우연히 중간값 근처였을 뿐 메커니즘은 평균이 아니다.

> [!example] 📐 일반해로 증명 — 임의의 (a,b)에 항상 도달 가능
> $\lambda_1[1,3]+\lambda_2[1,2]=(a,b)$ 를 풀면 $\lambda_1=b-2a,\ \lambda_2=3a-b$ — 분모 0이 되는 경우가 없어 **어떤 $(a,b)$든 유일해가 항상 존재**. 직접 검산(무작위 5쌍 + 역대입)해서 확인함 ✓ — 열공간=$\mathbb{R}^2$ 전체라는 주장의 대수적 버전.
> 🔭 **"유일해가 항상 존재"= $A_2$가 가역(det≠0)**이라는 말과 동치: $\det(A_2)=1\cdot2-1\cdot3=-1\neq0$. [[2.2 행렬 곱셈·행렬식|2.2]]·[[2.4 역행렬·연립방정식 — 특수 행렬·AX=B 풀이|2.4]]에서 정식으로 배울 개념을 손으로 먼저 겪은 것.

> [!success] 🧪 반증 실험 — "평균"이 아님을 결정적으로 보여주는 극단 사례
> 목표점을 원래 두 기울기(2, 3) 범위 **밖**인 $(1,0)$(기울기 0, x축 위)으로 잡으면 $\lambda_1=-2,\ \lambda_2=3$. 검산: $-2[1,3]+3[1,2]=[-2,-6]+[3,6]=[1,0]$ ✓(직접 재계산 완료).
> **음수 가중치**로 $v_1$ 반대 방향 2배 간 뒤(=임시 원점 $(-2,-6)$), 거기서 $v_2$ 방향 3배 전진해 두 원래 기울기와 무관한 지점에 정확히 착지 — "평균/중간"이었다면 애초에 불가능했을 결과. 순차 이동이 진짜 메커니즘이라는 최종 확인.

> 한 줄: 다중 벡터의 열공간이 헷갈렸던 이유는 "고정된 기울기 하나"를 찾으려 했기 때문 — 실제로는 **λ의 자유도가 만드는 도달 가능 영역**이고, 그 도달 메커니즘은 평균이 아니라 **음수 포함 자유 이동 + 순차 결합**이다.

### 🔑 $Ax=b$ 재해석 — "해가 있다 ⇔ $b\in C(A)$"
- [[2.2 행렬 곱셈·행렬식|2.2]] 열 시점: $Ax$ = A의 열들을 $x$ 계수로 결합 → **$Ax$ 가 도달 가능한 모든 결과 = 정확히 $C(A)$**. 따라서 연립방정식 $Ax=b$ 에 해가 존재한다는 말 = **$b$ 가 열공간 안에 있다**는 말의 동의어.
- [[2.4 역행렬·연립방정식 — 특수 행렬·AX=B 풀이|2.4]]의 column picture("b를 만드는 A열 결합 계수")가 이 언어로 완성된다. $b\notin C(A)$ 면 정확한 해가 없고 — 그때 "가장 가까운 열공간 안 점"을 찾는 것이 **최소제곱** `(🔭 10장)`.

### 🔑 소속 판별의 도구 사다리 — 눈→손→rank
- 책의 진행이 정직하다: 예제들은 추측(스칼라 있나)·산술(방정식 풀기)·시각화(그림)로 판별했지만, **셋 다 고차원에서 죽는다**. 그래서 정량 도구(rank)가 *필요해진다* — 다음 절의 필요성 사슬이 이 절에서 이미 만들어졌다.

### 🔑 행공간 = 전치 하나로 반값 획득 (2.3 회수)
- 새 이론이 0줄이다: [[2.3 행렬 전치 — 내적·외적 표기법·LIVE EVIL·대칭 행렬|2.3]]의 전치가 행↔열을 바꾸므로 $R(A)=C(A^{\mathsf{T}})$ — **열공간에서 증명한 모든 것이 행공간에 자동 상속**된다. "개념을 새로 사지 말고 전치로 재사용하라"의 교과서적 사례.
- 따름정리도 공짜: 대칭 행렬($A^{\mathsf{T}}=A$, 2.3)이면 $R(A)=C(A^{\mathsf{T}})=C(A)$ — 행 이야기와 열 이야기가 같은 이야기가 된다.

### 🔑 영공간 = 1.8 식 2-3의 해집합 — '종속의 증거 보관소'
- $Ay=0$ (0 아닌 $y$)을 열 시점으로 읽으면 $y_1 a_1 + \cdots + y_N a_N = 0$ (비자명해) — 이것은 [[1.8 벡터 확장 — 집합·선형결합·선형독립·부분공간·생성·기저|1.8]] **식 2-3(선형 종속의 정식 정의) 그 자체**다. 즉 **영공간 = 종속 증명서($\lambda$들)의 보관소**: 비어 있지 않다 ⇔ 열들이 종속 ⇔ 어떤 열은 재탕.
- 그래서 "독립 열 ⇔ 빈 영공간"은 새 정리가 아니라 **정의의 재진술**이고, 영공간 차원은 "종속의 개수"를 세는 잣대가 된다(nullity `🔭`) — 열공간 차원(독립 열 수)과 합치면 전체 열 수가 될 수밖에 없다는 직감까지 이미 생긴다.
- 자명해 $y=0$ 제외도 1.8·[[1.3 벡터 연산 — 덧셈·뺄셈·스케일링·전치·브로드캐스팅|1.3]] 영벡터 규약("선대는 비자명해를 찾아라")의 재등장.

### 🔑 영공간 ⊥ 행공간 — 내적 0의 기하 (1.6 회수)
- $Ay=0$ 을 **행 시점**으로 읽으면 각 행과 $y$ 의 내적이 0: $a_i\cdot y=0\ \forall i$ — [[1.6 벡터 내적 — 점곱·기하·직교·아다마르곱·외적|1.6]]의 "내적 0 = 직교"가 곧바로 **영공간 ⊥ 행공간**을 준다(그림 5-4). 같은 식 $Ay=0$ 이 열 시점(종속 증거) / 행 시점(직교)의 **두 얼굴**을 가진다 — 2.2 행렬곱 두 시점의 재활용.
- 예 1의 수치로: 행들 $[1,-1]$·$[-2,2]$ 는 같은 직선(종속·rank 1), 영공간 $[1,1]$ 방향은 그 직선과 수직 — $\mathbb{R}^2$ 가 행공간(1D)⊕영공간(1D)으로 쪼개진다는 예감 `(🔭 네 부분공간의 관계)`.

> 가설: "기울기가 같으면서 **방향이 반대**(가중치가 서로 상쇄)이면 영공간이 형성된다" — 예 1의 열 $c_1{=}[1,{-}2],\,c_2{=}[{-}1,2]{=}{-}c_1$(정확히 반대 방향)과 행 $[1,{-}1],[{-}2,2]{=}{-}2\times[1,{-}1]$을 보면 그럴듯하다.
> **"방향이 반대"는 예 1에만 있는 우연이고, 진짜 조건이 아니다.** 이 노트 예 3의 $C\!\left(\begin{bmatrix}1&2\\3&6\end{bmatrix}\right)$(열: $c_2{=}{+}2c_1$, **같은 방향**)으로 반증: `null_space` 계산 결과 $y\approx[-0.894,\,0.447]$ — 영공간이 **똑같이 존재한다**(성분 부호는 반대: $-2\times0.447=-0.894$). 열이 같은 방향인데도 영공간을 만드는 $y$ 자체는 **서로 다른 부호**를 갖는다 — "열 방향이 반대 ⇒ $y$ 성분은 같은 부호", "열 방향이 같음 ⇒ $y$ 성분은 다른 부호"라는 **부호 관계**만 성립하고, "영공간이 형성되려면 방향이 반대여야 한다"는 명제 자체는 성립하지 않는다.
> **진짜 조건은 기울기(=방향 무시한 비율)가 같다는 것, 그뿐이다.** 두 벡터가 같은 직선 위(스칼라 배수 — 배수가 양수든 음수든 무관)에 있으면 그 직선과 수직인 방향이 영공간이 된다. "방향이 반대"는 예 1처럼 배수가 마침 음수(−1, −2)일 때 곁다리로 따라오는 **결과**이지, 영공간의 **원인·조건**이 아니다.
>
> 한 줄: 영공간을 결정하는 건 "같은 직선 위(=종속)"라는 사실 하나뿐 — 방향이 같은지 반대인지는 영공간의 존재·방향 어느 쪽에도 영향을 못 준다, $y$ 성분의 부호 패턴만 바꿀 뿐.

### 🔑 $(A-\lambda I)v=0$ — 2.1 '이동'과 12장 예고가 여기서 만난다 (반보 선행)
- 책은 "영공간이 고유벡터 찾기에 핵심"이라고만 말하지만, 이미 가진 부품으로 반 발 먼저 조립할 수 있다: 고윳값 식 $Av=\lambda v$ 를 옮기면 $(A-\lambda I)v=0$ — **고유벡터 $v$ = '이동된 행렬 $A-\lambda I$'의 영공간 원소**다.
- [[2.1 행렬 연산 — 덧셈·뺄셈·이동·스칼라곱·아다마르곱|2.1]]이 예고한 응용 ①("이동은 고윳값을 찾는 메커니즘")의 실체가 이 문장이고, λ 신분 ②(이동량)와 ③(고윳값)이 **한 식에서 동시 출연**한다. 영공간이 비어 있지 않으려면 $A-\lambda I$ 가 축소계수여야 하고 — 그 λ를 찾는 것이 고윳값 문제 `(🔭 12장 — 수신 시 이 절 회수)`.
- ✅ **후속 확인(2026-07-14):** "$A-\lambda I$가 축소계수여야" 한다는 조건의 **정량적 판별식**이 [[2.8 행렬식 — 계산·선형종속성·특성다항식|2.8]]에서 나왔다 — $\det(A-\lambda I)=0$(특성다항식). "영공간이 안 비려면"이 결국 "행렬식이 0이 되는 λ를 찾는 문제"로 구체화된다.

### 🔑 0.70710678의 정체 — 단위 기저 벡터 관행 (1.2 회수)
- $[1,1]$ 이 더 읽기 쉬운데 왜 $[0.7071, 0.7071]$? — 무한 후보 중 **노름 1짜리**([[1.2 벡터 크기와 단위벡터 (노름)|1.2]] 단위벡터)를 골라 주는 것. $\sqrt{1/2}\approx0.70711$, 검산 $0.7071^2+0.7071^2=1$.
- 단위벡터는 수치 안정성 등 좋은 특성 → **알고리즘의 부분공간 출력 = 단위 기저 벡터**가 관행(고유벡터·특이벡터도 같은 모습으로 온다 `🔭`) — "컴퓨터가 주는 답의 생김새"를 미리 알아두는 것.

### 📋 네 부분공간 지도 (이름표만 — 관계는 🔭)
| 부분공간 | 정의 | 사는 곳 |
|---|---|---|
| 열공간 $C(A)$ | 열들의 span | $\mathbb{R}^M$ |
| 행공간 $R(A)=C(A^{\mathsf{T}})$ | 행들의 span | $\mathbb{R}^N$ |
| 영공간 $N(A)$ | $Ay=0$ 의 비자명해 | $\mathbb{R}^N$ — **행공간과 직교** |
| 왼쪽 영공간 $N(A^{\mathsf{T}})$ | $A^{\mathsf{T}}z=0$ (행의 영공간) | $\mathbb{R}^M$ — 이름만(책도 넘어감) |

## 🧩 코드 (§5.2.1~5.2.3)
> 🎯 목표: 소속 판별 3례 + "종속 2열=1D" + 영공간(단위벡터·빈 집합·행과의 직교) 확인.
> 코드: `~/LOCAL/03-00_STUDIES/AI/01_linear_algebra/la5_spaces.py`

```python
import numpy as np
A = np.array([[1.], [3.]])                     # 열 1개
# 소속 판별: v = λ·col 이 성립하는 λ가 있나
for v, expect in [([1,3], True), ([-2,-6], True), ([1,4], False)]:
    v = np.array(v, float)
    lam = v[0] / A[0, 0]                       # 첫 성분으로 λ 후보
    print(v, "∈ C(A)?", np.allclose(A.ravel()*lam, v), "(기대:", expect, ")")
# 독립 2열 → R² 전체: [-4,3] = 11·[1,3] − 15·[1,2]  (책 수치 검증)
c1, c2 = np.array([1,3.]), np.array([1,2.])
print(11*c1 - 15*c2)                           # [-4. 3.] ✓
# 종속 2열 → 여전히 1D (rank 예고)
D = np.array([[1,2],[3,6.]])
print("독립 열 수(rank):", np.linalg.matrix_rank(D))   # 1

# §5.2.3 영공간 — 책 예제 두 개
from scipy.linalg import null_space
A1 = np.array([[1.,-1.],[-2.,2.]])                 # 종속 열 → N(A) 존재
ns = null_space(A1)                                # [[0.70710678],[0.70710678]]
print(ns.ravel(), "= 단위벡터(√½)")                  # 노름 1 확인: 0.7071²×2 = 1
print(A1 @ ns)                                     # ≈ [[0],[0]] — Ay=0
print(A1[0] @ ns, A1[1] @ ns)                      # 각 행과 내적 0 — 영공간⊥행공간
B1 = np.array([[1.,-1.],[-2.,3.]])                 # 독립 열(가역, det=1)
print(null_space(B1))                              # [] — 빈 영공간
```

| 영공간은 어디와 직교? | **행공간** — $Ay=0$ = 각 행 $a_i$ 와의 내적 전부 0 (그림 5-4) |
| 영공간이 중요한 진짜 이유? | 고유벡터·특이벡터 찾기의 핵심(12장) — $(A-\lambda I)v=0$: 고유벡터=**이동된 행렬의 영공간** |
| `null_space` 가 0.7071을 주는 이유? | 무한 후보 중 **단위벡터**(√½) 반환 — 알고리즘은 단위 기저 벡터가 관행 |
| 네 부분공간? | 열 $C(A)$ · 행 $R(A)=C(A^{\mathsf{T}})$ · 영 $N(A)$ · **왼쪽 영 $N(A^{\mathsf{T}})$** |
| 선형 결합 ≠ 평균? 📌 | 그렇다 — 실제 메커니즘은 **순차 이동(꼬리 물기)**; "평균처럼 보임"은 λ₁=λ₂=1인 특수 결과만 본 착시 |
| 열공간=ℝ² 증명(대수)? | $\lambda_1[1,3]+\lambda_2[1,2]=(a,b)$ 풀면 $\lambda_1{=}b{-}2a,\lambda_2{=}3a{-}b$ — 항상 유일해 ⇔ **det≠0(가역)** |
| 부분공간 = 열공간? 📌 | **아니다** — 범주(부분공간) ⊃ 사례(열공간). 행공간·영공간도 부분공간의 다른 사례(네 부분공간 지도) — "구성한다"≠"~이다", 동의어 아님 |
| 영공간 형성 조건 = "방향이 반대"? 📌 | **아니다** — 조건은 오직 "기울기가 같음(=종속)". 방향(같은/반대)은 무관 — 같은 방향(예3: c₂=+2c₁)도 영공간 생김([-0.894,0.447]); 방향은 y 성분의 부호 패턴만 바꿈 |
