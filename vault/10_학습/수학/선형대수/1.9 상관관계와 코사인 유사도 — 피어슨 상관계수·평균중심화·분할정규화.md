# 상관관계와 코사인 유사도 — 피어슨 상관계수·평균중심화·분할정규화

> 📐 **수학 개념원자(SSoT)** — 선형대수/통계 교차. 정의·성질·증명·기하 직관. AI 적용·연결은 코스 렌즈가 끌어다 쓴다.
> 🔗 [[_수학 MOC]] ｜ 🌐 [[_지식 연결 허브]] ｜ 🎓 AI 렌즈: [[07 상관관계와 코사인 유사도 — RAG 검색엔진|07(AI)]]

> **상관관계(correlation)** 는 두 데이터열이 "함께 움직이는 정도"를 −1~+1 사이 수 하나로 압축한 것이다. 이 절은 상관계수(피어슨)를 **선형대수의 언어(내적·노름)** 로 다시 쓰고, [[1.6 벡터 내적 — 점곱·기하·직교·아다마르곱·외적|1.6]]에서 이미 배운 **코사인 유사도**와 얼마나 닮았고 어디서 갈라지는지 규명한다.

> 🎯 **핵심 (TL;DR)**
> - **피어슨 상관계수(식 3-1, 통계 표기):** ρ = Σ(xᵢ−x̄)(yᵢ−ȳ) / [√Σ(xᵢ−x̄)² · √Σ(yᵢ−ȳ)²]
> - **선형대수 표기(식 3-2):** x̃=x−x̄𝟙(평균중심화된 벡터)라 하면, ρ = x̃ᵀỹ / (‖x̃‖‖ỹ‖) — **평균중심화된 두 벡터의 코사인 유사도.**
> - **코사인 유사도(1.6에서 이미 정의):** cos θ = xᵀy / (‖x‖‖y‖) — **평균을 빼지 않은 원본** 벡터로 계산.
> - **두 정규화:** ① 평균중심화(첫 번째, 위치를 원점으로) ② 분할정규화(두 번째, 노름 곱으로 나눠 크기 제거).
> - **검증된 예제:** x=[0,1,2,3], y=[100,101,102,103] (y=x+100) → **ρ≈1.0000**(완벽한 선형 추세) 이지만 **cos θ≈0.808**(원점 기준 각도는 다름) — 상관계수와 코사인 유사도는 **다른 질문에 답한다.**
> - **공통 뿌리:** 상관계수도 코사인 유사도도 결국 **내적을 노름으로 나눈 값** — 둘 다 선형(내적 기반) 연산.

> [!question] 🎯 필요성 사슬 (결핍 → 필요 → 발명)
> *출발 문제:* 두 데이터열(예: 두 종목 주가, 두 문서의 단어 빈도)이 "관련 있다"는 걸 숫자 하나로 표현하고 싶다.
> 1. 내적은 있지만 크기(스케일)에 너무 민감하다([[1.6 벡터 내적 — 점곱·기하·직교·아다마르곱·외적|1.6]]) → 방향만 비교하려면 크기를 나눠 없애야 함 → **분할정규화 → 코사인 유사도**($x^\mathsf{T}y/\lVert x\rVert\lVert y\rVert$).
> 2. 그런데 코사인 유사도는 원점을 기준으로 각도를 재기 때문에, 두 데이터열에 **같은 상수를 더해도(오프셋)** 값이 달라진다 — "두 종목이 같이 오르내리는가"를 보고 싶은데 "평균 주가가 비슷한가"까지 섞여버림 → **위치(평균)를 먼저 제거**해야 순수한 "경향의 일치"만 남는다 → **평균중심화**($\tilde x=x-\bar x$).
> 3. 평균중심화 + 코사인 유사도를 합치면? → **피어슨 상관계수** — "경향(위치 무관) × 방향(크기 무관)"만 남은, 통계학이 100년 넘게 써온 표준 측도.
> **한 줄:** 코사인 유사도(1.6)에서 평균(위치)까지 제거하면 상관계수 — 상관계수는 **평균중심화된 벡터의 코사인 유사도.**

> [!abstract] 📘 책 핵심 — [실전선대] §3.1 상관관계와 코사인 유사도
> **① 상관관계란 (그림 3-1):** 두 변수 x,y의 관계를 산점도로 보면 강한 양의 선형(우상향)·강한 음의 선형(우하향)·무관(흩어짐)·비선형(곡선, 그러나 결정론적) 패턴이 구분된다. 상관계수는 이 중 **선형** 관계의 강도·방향만 하나의 수(−1~+1)로 요약한다.
>
> **② 식 3-1 — 피어슨 상관계수(통계 표기):**
> $$\rho=\frac{\sum_i (x_i-\bar x)(y_i-\bar y)}{\sqrt{\sum_i(x_i-\bar x)^2}\;\sqrt{\sum_i(y_i-\bar y)^2}}$$
>
> **③ 평균중심화(mean-centering([miːn ˈsɛntərɪŋ], 민 센터링, 민 센터링)) — 첫 번째 정규화:** $\tilde x = x-\bar x\mathbf{1}$ (모든 원소에서 평균을 뺌) → 데이터의 "위치"를 원점으로 옮긴다. 분자 $\sum(x_i-\bar x)(y_i-\bar y)$ = 평균중심화된 두 벡터의 **내적** $\tilde x^\mathsf{T}\tilde y$.
>
> **④ 분할정규화(divisive normalization([daɪˈvɪsɪv ˌnɔːrməlaɪˈzeɪʃən], 디바이시브 노멀라이제이션, 디바이시브 정규화)) — 두 번째 정규화:** 분모(두 노름의 곱)로 나눠 **크기**를 제거 — [[1.6 벡터 내적 — 점곱·기하·직교·아다마르곱·외적|1.6]]의 코사인 유사도 유도와 동일한 조작.
>
> **⑤ 식 3-2 — 선형대수 표기:**
> $$\rho=\frac{\tilde x^\mathsf{T}\tilde y}{\lVert\tilde x\rVert\,\lVert\tilde y\rVert}$$
>
> **⑥ 코사인 유사도와의 관계:** $\cos\theta=\dfrac{x^\mathsf{T}y}{\lVert x\rVert\lVert y\rVert}$ 와 식 3-2를 나란히 놓으면 **형태가 완전히 같다** — 차이는 오직 **평균중심화를 했는가(ρ) 안 했는가(cos θ)** 뿐. 그래서 "상관계수 = 평균중심화된 벡터들 사이의 코사인 유사도"라고 부른다.
>
> **⑦ 워크드 예제 — 책의 수치:** $x=[0,1,2,3]$, $y=[100,101,102,103]$ ($y=x+100$, 완전한 선형 이동) → **ρ=1**(완벽 상관) 인데 **cos θ≈0.808**(직교에서 꽤 멀지만 1은 아님). 책의 해석: 두 지표는 **"관련성"에 대해 서로 다른 가정**을 담는다 — 상관계수는 "같이 움직이는가"(추세), 코사인 유사도는 "원점에서 같은 방향을 가리키는가"(원본 벡터의 절대 위치까지 포함).
>
> **⑧ 마무리:** 상관계수·코사인 유사도 **둘 다 결국 내적을 노름으로 정규화한 연산** — 근본적으로 **선형(내적 기반)** 이다. (책은 이 절 뒤에 §3.1 관련 코딩 연습 문제 4개를 예고 — 아직 미제공, 제공되면 이 원자에 통합 예정.)

### 🔍 왜 ρ=1인데 cos θ=0.808인가 — 직접 유도
$y=x+100$ 은 순수한 **이동(shift)** 이다. 평균중심화를 하면:
$$\tilde x = x-\bar x = [-1.5,-0.5,0.5,1.5],\qquad \tilde y = y-\bar y = [-1.5,-0.5,0.5,1.5]$$
**평균을 빼는 순간 x̃와 ỹ는 완전히 같은 벡터가 된다** — 상수 오프셋(+100)이 통째로 사라지기 때문. 그래서 $\rho=\tilde x^\mathsf{T}\tilde y/(\lVert\tilde x\rVert\lVert\tilde y\rVert)=\lVert\tilde x\rVert^2/\lVert\tilde x\rVert^2=1$ 이 대수적으로 강제된다.

반면 원본 $x,y$ 는 평균을 안 뺐으므로 $\lVert y\rVert=203.01$ 이 **거의 전부 +100 오프셋에서 나온 크기**다(원소 간 실제 변화폭은 0~3인데, 노름은 100 근처에서 결정됨). 코사인은 이 큰 절대 위치를 각도 계산에 그대로 포함시키므로 $\cos\theta\ne1$:
$$\cos\theta=\frac{x^\mathsf{T}y}{\lVert x\rVert\lVert y\rVert}=\frac{614}{3.742\times203.01}\approx0.808$$

![[LA3-2 피어슨vs코사인 예제 (재구성).png|760]]
🔄 **재구성** — (왼쪽) x,y는 인덱스에 대해 완벽한 직선(y=x+100) → ρ=1. (오른쪽) 그런데 ‖y‖(203.0)가 ‖x‖(3.74)보다 압도적으로 큰 건 대부분 +100 오프셋 때문 — 코사인은 이 절대 크기 차이를 각도에 그대로 반영해 0.808로 낮아진다.

> [!tip] 🪂 심화 — 상관계수의 범위가 −1~+1인 이유(책이 생략) — 코시-슈바르츠   `#사이드/심화`
> 코시-슈바르츠 부등식(Cauchy–Schwarz inequality([ˈkoʊʃiː ˈʃwɑːrts], 코시 슈바르츠, 코시 슈바르츠)): 임의의 벡터 $u,v$ 에 대해 $|u^\mathsf{T}v|\le\lVert u\rVert\lVert v\rVert$. 양변을 $\lVert u\rVert\lVert v\rVert$ 로 나누면 $|\cos\theta|\le1$ — **코사인 유사도가 항상 [−1,1]에 갇히는 이유**다. 상관계수 ρ는 정의상 **평균중심화된 벡터들의 코사인**이므로, 같은 부등식을 $u=\tilde x,\,v=\tilde y$ 에 적용하면 **ρ도 자동으로 [−1,1]에 갇힌다** — 별도 증명이 필요 없다(코사인의 성질을 그대로 상속).

> [!warning] ⚠️ 정확성 — ρ=1은 "같은 값"이 아니라 "완벽한 선형 관계"   `#정확성`
> $x=[0,1,2,3]$ 과 $y=[100,101,102,103]$ 는 원소 하나도 같지 않다. 그런데도 ρ=1인 이유는 상관계수가 **"값의 근접"이 아니라 "변화 패턴(기울기 방향과 비율)의 일치"** 를 재기 때문 — [[1.6 벡터 내적 — 점곱·기하·직교·아다마르곱·외적|1.6]]의 "내적은 값의 근접이 아니라 부호 정렬×크기"라는 경고와 같은 함정의 상관계수 버전.

> [!quote] 🌉 다리 — 상관계수는 왜 "정규화된 공분산"인가   `#연결/통계`
> 분자 $\sum(x_i-\bar x)(y_i-\bar y)=\tilde x^\mathsf{T}\tilde y$ 를 $(n-1)$로 나누면 **공분산(covariance([koʊˈvɛəriəns], 코베리언스, 코베리언스))** $\text{Cov}(x,y)$ 이 된다. 상관계수는 공분산을 각 변수의 표준편차 곱으로 나눈 것과 정확히 같다($\rho=\text{Cov}(x,y)/(\sigma_x\sigma_y)$) — 이 절의 "분할정규화"가 통계학에서 부르는 이름이 바로 "표준편차로 나누기"다. 여러 변수 쌍의 공분산을 한 행렬에 모으면 **공분산 행렬** — 이 절의 스칼라 연산을 행렬로 확장한 것(🔭 예고).

## 🧩 코드 실습 (§3.1)
> 🎯 목표: 식3-1(직접)·식3-2(선대 표기)·numpy 내장이 셋 다 같은 ρ를 주는지, 코사인 유사도와 왜 다른지 수치로 검증.
> 코드: `~/LOCAL/03-00_STUDIES/AI/01_linear_algebra/la3_1_correlation.py`

```python
import numpy as np

x = np.array([0, 1, 2, 3], dtype=float)
y = np.array([100, 101, 102, 103], dtype=float)

# 식 3-1: 피어슨 상관계수 (직접, 통계 표기)
xbar, ybar = x.mean(), y.mean()
num = np.sum((x - xbar) * (y - ybar))
den = np.sqrt(np.sum((x - xbar)**2)) * np.sqrt(np.sum((y - ybar)**2))
rho_direct = num / den

# 식 3-2: 선형대수 표기 (평균중심화 벡터의 코사인)
xt, yt = x - xbar, y - ybar
rho_la = (xt @ yt) / (np.linalg.norm(xt) * np.linalg.norm(yt))

# numpy 내장 교차검증
rho_builtin = np.corrcoef(x, y)[0, 1]

# 코사인 유사도 (평균중심화 없음 — 원본 그대로)
cos_sim = (x @ y) / (np.linalg.norm(x) * np.linalg.norm(y))

print("피어슨(식3-1 직접):", rho_direct)     # 0.9999999999999998
print("피어슨(식3-2 선대):", rho_la)         # 0.9999999999999998 — 같음
print("numpy corrcoef:   ", rho_builtin)     # 1.0
print("코사인 유사도:      ", cos_sim)        # 0.8083174787557303

print("\n평균중심화 후 x̃, ỹ:", xt, yt)       # 완전히 동일한 벡터!
```

관찰 포인트:
- **식3-1·식3-2·numpy 내장 셋 다 정확히 일치** — 통계 표기와 선형대수 표기가 같은 연산의 다른 서술임을 수치로 확인.
- **평균중심화 후 x̃=ỹ**(완전히 같은 벡터) — ρ=1이 우연이 아니라 y=x+100이 순수 이동이라는 대수적 필연.
- **코사인 유사도(0.808)는 원본 벡터의 절대 위치(+100 오프셋)에 여전히 민감** — 상관계수(1.0)는 그 오프셋을 이미 제거한 뒤의 값.

## 🆕 새 용어 정리
| 용어 | 한 줄 정의 |
|---|---|
| **피어슨 상관계수** (Pearson Correlation Coefficient) | 두 데이터열의 선형 관계 강도·방향; −1~+1; ρ=x̃ᵀỹ/(‖x̃‖‖ỹ‖) |
| **평균중심화** (Mean-Centering) | x̃=x−x̄𝟙 — 위치(평균)를 원점으로 옮기는 첫 번째 정규화 |
| **분할정규화** (Divisive Normalization) | 노름의 곱으로 나눠 크기를 제거하는 두 번째 정규화 |
| **코사인 유사도** (Cosine Similarity) | cos θ=xᵀy/(‖x‖‖y‖) — 평균중심화 없이 원본 벡터로 계산 (1.6 원자 소관) |
| **공분산** (Covariance) | Σ(xᵢ−x̄)(yᵢ−ȳ)/(n−1) — 상관계수의 분자를 표본크기로 나눈 것 |
| **코시-슈바르츠 부등식** (Cauchy–Schwarz Inequality) | \|uᵀv\|≤‖u‖‖v‖ — 코사인·상관계수가 [−1,1]에 갇히는 근거 |
