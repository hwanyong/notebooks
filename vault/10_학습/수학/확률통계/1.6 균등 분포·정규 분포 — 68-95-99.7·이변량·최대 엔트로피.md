> 🎯 **핵심 (TL;DR)**
> - **균등 분포**: 구간 $(x_{\min}, x_{\max})$ 안의 모든 값이 뽑힐 확률이 같다 — pdf는 **상수** $f(x)=\frac{1}{x_{\max}-x_{\min}}$(그림 2-8). 실제 데이터(키·체중)는 균등하지 않다 — 컴퓨터 난수 생성기의 **기반**으로 유용.
> - **정규 분포(가우스 분포)**: 평균 $\mu$([mjuː], 뮤, 뮤) 주변에 종 모양으로 몰리고 표준 편차 $\sigma$([ˈsɪɡmə], 시그마, 시그마)만큼 대칭적으로 퍼진다 📌. **68-95-99.7 규칙**(평균의 1·2·3×표준편차 이내에 각각 약 68%·95%·99.7%) — 직접 검증: **68.27% / 95.45% / 99.73%**.
> - 단변량 pdf: $g(x;\mu,\sigma)=\frac{1}{\sqrt{2\pi\sigma^2}}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$ — 표준정규($\mu{=}0,\sigma{=}1$) 피크값 $=\frac{1}{\sqrt{2\pi}}=0.3989$(직접 검증, 책의 "0.40"과 일치).
> - **이변량 정규**(그림 2-10·2-11): 두 확률 변수 $x,y$의 결합 분포. 원시형(각 파라미터 나열) vs **공분산 행렬 Σ**(대문자 시그마)를 쓴 선형대수형 — 코드로 **두 수식이 실제로 같은 값**임을 3자 대조(원시형=Σ형=scipy) + **R² 전체 적분=1.000000**까지 확인.
> - **어떤 분포를 써야 할지 모르면 정규 분포가 합리적**이다 📌 — 동일한 분산을 가진 모든 분포 중 정규 분포가 **불확실성(엔트로피)이 가장 큰 선택**, 즉 가장 적은 사전 지식을 인코딩. 정규 분포는 **중심 극한 정리(CLT)**의 이론적 배경이기도 하다 `(🔭 예고)`.

> *출발 문제:* [[1.5 연속·이산 분포와 결합 확률 밀도 함수 — pdf·pmf·차원의 저주|1.5]]에서 "슬라이스하면 사후 확률에 비례한다"는 **기술**은 얻었지만, 정작 슬라이스할 **구체적인 함수**를 하나도 정의하지 않았다.
> 1. 가장 단순한 경우부터: 구간 안 모든 값이 똑같이 뽑힐 가능성이라면? → **균등 분포**(상수 pdf) — 그러나 실제 데이터(키)는 평균 근처에 몰리지 균등하지 않다(2단계 [[19 실제 vs 시뮬레이션 데이터 — 키-체중·조건부 층화·노이즈|19]]에서 이미 확인).
> 2. 평균 근처에 몰리고 대칭적으로 퍼지는 분포가 필요 → **정규 분포**(종 모양) — 퍼짐의 정도를 조절하는 파라미터($\sigma$)까지 필요.
> 3. 확률 변수가 하나(키)가 아니라 둘(키+체중)이면 정규 분포는 어떻게 확장되나 → **이변량 정규**, 두 변수의 관계(상관 $\rho$)까지 담는 **공분산 행렬 Σ**가 필요.
> 4. 데이터에 대한 사전 지식이 전혀 없다면 어떤 분포를 골라야 하나 → **정규 분포**(동일 분산에서 불확실성 최대 = 가정을 가장 적게 인코딩) — "모르면 정규 분포"의 수학적 근거.
> **한 줄:** 균등(가장 단순) → 정규(현실의 종 모양, σ 필요) → 이변량(변수 2개, Σ 필요) → 무정보 시 기본값(정규 = 최대 불확실성).

> [!abstract] 📘 책 핵심 — [필수수학AI] §2.11 균등 분포
> - 균등 분포를 직관적으로 이해하려면 비균등 분포의 예를 먼저 보는 게 좋다 — 실제 키-체중 데이터셋에서는 균등 분포로 키를 모델링할 수 없었다. 사람의 키와 체중이 균일하게 분포되어 있지 않기 때문이다. 실제로 키가 2미터인 사람과 160cm, 180cm인 사람을 만날 확률은 경험적으로 같지 않다.
> - 균등 분포는 말 그대로 균등하게 분포된 데이터만 모델링한다. 모든 데이터가 최솟값 $x_{\min}$과 최댓값 $x_{\max}$ 사이의 연속적인 구간에서 균등하게 분포되어 있다면 그 구간 안에서 임의의 값을 추출할 확률은 모두 같다. 예를 들어 구간이 $(0,1)$이라면 0.2를 추출할 확률과 0.75를 추출할 확률은 같다.
> - 따라서 균등 분포의 확률 밀도 함수는 **상수**다. 구간 $(x_{\min}, x_{\max})$에 있는 확률 변수 $x$에 대해:
> $$f(x;x_{\min},x_{\max}) = \frac{1}{x_{\max}-x_{\min}} \ \text{for } x_{\min}<x<x_{\max}, \quad \text{구간 외에서는 모두 } 0.$$
> - 균등 분포의 확률 밀도 함수 그래프는 직선(계단) 형태다 — 실제·시뮬레이션 데이터 모두 균등하게 분포된 데이터가 전체 구간에 골고루 분산되기 때문. 구간 내에서는 모든 데이터 값이 공평하다.
> - 균등 분포는 **컴퓨터 시뮬레이션에서 랜덤한 숫자를 생성하는 데 매우 유용**하다 — 파이썬 난수 생성기의 기본 알고리즘 어딘가에 균등 분포가 쓰인다.

> [!abstract] 📘 책 핵심 — [필수수학AI] §2.12 정규 분포
> - 사람의 키 데이터(특정 성별)를 모델링하는 데 더 적합한 연속 확률 분포는 가우스 분포라고도 하는 종 모양의 **정규 분포**다. 정규 분포에서 추출한 샘플은 평균값 $\mu$ 주변에서 분포의 정점을 이루는 곳에 모이는 경향이 있고 평균에서 멀어질수록 대칭적으로 줄어든다. 분포가 평균에서 멀어질수록 얼마나 퍼져 있는지는 정규 분포의 두 번째 파라미터인 **표준 편차** $\sigma$에 의해 조절된다.
> - **68-95-99.7 규칙**: 데이터의 약 68%가 평균의 1×표준 편차 이내에 있고, 데이터의 95%가 평균의 2×표준 편차 이내, 데이터의 약 99.7%가 평균의 3×표준 편차 이내에 속한다.
> - 정규 분포에서 데이터를 샘플링할 때는 평균 근처의 값이 선택될 확률이 높으며 아주 작은 값($-\infty$)이나 아주 큰 값($\infty$)이 선택될 확률은 매우 낮다. 평균값 근처에서 정점을 형성하고 분포의 외곽에서 감소하는 성질 때문에 종 모양을 형성한다. 정규 분포는 확률 이론에서 매우 중요한 정리인 **중심 극한 정리(CLT)**의 이론적 배경이 된다.
> - **중심 극한 정리**는 동일한 분포(반드시 정규 분포일 필요는 없음)를 가진 많은 독립 확률 변수들의 평균이 정규 분포를 따른다는 것을 나타낸다. 이는 정규 분포가 사회와 자연계 어디에서나 나타나는 이유를 설명해준다 — 신생아의 체중 분포, 학생의 성적 분포, 국가의 소득 분포, 혈압 측정값 분포 등.
> - 만약 어떤 분포를 사용해야 할지 확신이 없고 데이터에 관해 사전 지식도 없는 상황이라면 정규 분포를 선택하는 것이 합리적이다. **사실 동일한 분산을 가지는 모든 분포 중에서 정규 분포는 불확실성이 가장 큰 선택이다.** 즉, 정규 분포를 선택하는 것은 모델이 가장 적은 사전 지식을 인코딩하는 것과 같다.
> - 단변량(하나의 확률 변수 $x$)이 정규 분포를 따를 때 pdf: $g(x;\mu,\sigma)=\dfrac{1}{\sqrt{2\pi\sigma^2}}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$
> - 두 확률 변수 $x,y$(다변량)가 정규 분포를 따를 때 pdf(원시형):
> $$g(x,y;\mu_1,\sigma_1,\mu_2,\sigma_2,\rho)=\dfrac{1}{\sqrt{(2\pi)^2\det\!\begin{pmatrix}\sigma_1^2 & \rho\sigma_1\sigma_2\\ \rho\sigma_1\sigma_2 & \sigma_2^2\end{pmatrix}}}\ e^{-\left\{\frac12(x-\mu_1\ \ y-\mu_2)\begin{pmatrix}\sigma_1^2 & \rho\sigma_1\sigma_2\\ \rho\sigma_1\sigma_2 & \sigma_2^2\end{pmatrix}^{-1}\!\begin{pmatrix}x-\mu_1\\y-\mu_2\end{pmatrix}\right\}}$$
> - 선형대수를 사용하면 이변량 정규 분포의 확률 밀도 함수를 더 간결하게 표현할 수 있다: $g(x,y;\mu,\Sigma)=\dfrac{1}{\sqrt{(2\pi)^2\det(\Sigma)}}\ e^{-\left\{\frac12(u-\mu)^{\mathsf T}\Sigma^{-1}(u-\mu)\right\}}$
> - [그림 2-11]은 이변량 정규 분포에서 6,000개의 데이터 포인트를 랜덤 추출하여 그린 그래프다. 중심 근처의 점들은 선택될 확률이 높고 중심에서 멀리 떨어진 점들은 선택될 확률이 낮다. 그림의 선을 보면 정규 분포의 등고선을 따른다 — 추출한 데이터들이 어떤 분포에서 샘플링되었는지 모르고 관찰한다면 이 선들은 정규 분포의 등고선이라고 추정할 수 있다.

> 🔤 **표기(rule 13, [수학]→[책 1D]→[내 2D]):** 책은 $(x{-}\mu_1\ \ y{-}\mu_2)$를 **행벡터**처럼 나열해 쓰지만($1{\times}2$), 실제 이차형식 계산에선 이 대상이 **한 번은 행($1{\times}2$, 왼쪽), 한 번은 열($2{\times}1$, 오른쪽)**로 양쪽에 다르게 쓰인다 — $u,\mu$는 코드에서 **명시적 열벡터** `np.array([[x],[y]])` (또는 `.reshape(-1,1)`)로 다뤄야 `(u-\mu)^{\mathsf T}\Sigma^{-1}(u-\mu)$가 스칼라 하나로 맞아떨어진다(아래 코드 §에서 `d @ Sig_inv @ d`로 1D `(2,)` 벡터를 써도 넘파이가 자동으로 맞춰주지만, 수식과 1:1 대조하려면 `(2,1)` 열벡터가 정확하다).

## 🧠 근본 이해 · 융합 · 통찰

### 🔑 원시형 = Σ형 = scipy — 책의 "더 간결한 표현"이 진짜로 같은 값인지 3자 대조

- Σ가 **대칭**($\rho\sigma_1\sigma_2$가 비대각 양쪽에 동일)이라는 점은 [[2.3 행렬 전치 — 내적·외적 표기법·LIVE EVIL·대칭 행렬|2.3]]의 대칭 행렬, [[22 확률·통계 용어 지도 — 확률 변수·분포·기대값·공분산·마르코프|22]]의 "공분산 행렬은 대칭"과 정확히 같은 대상 — 여기서 처음으로 **수식을 직접 조립**해 재확인한 것.

### 🔑 68-95-99.7이 "약"인 이유 — 정밀값과 반올림의 차이
- 책은 "약 68%… 95%… 약 99.7%"라 쓴다. 직접 계산하면 **68.27% / 95.45% / 99.73%** — 책의 반올림은 정확하지만 셋째 자리까지 보면 68·95는 위로, 99.7은 반올림 방향이 다르다(69→68 아님, 68.27→68 내림 / 95.45→95 내림 / 99.73→99.7 내림). 실무에서 "68-95-99.7 규칙(empirical rule)"이라 부르는 이유.

### 🔑 최대 엔트로피 — "정규 분포가 불확실성이 가장 크다"의 정확한 의미
- 책의 "동일한 분산의 모든 분포 중 정규 분포가 불확실성 최대"라는 주장은 **정보 이론의 최대 엔트로피 원리**의 특수 사례다: 연속 확률 변수 중 **평균·분산이 고정된 조건**에서 (미분) 엔트로피 $H[X]=-\int f(x)\ln f(x)\,dx$를 최대화하는 분포가 정규 분포임이 증명되어 있다(라그랑주 승수법). "불확실성이 크다" = "이 분포를 고른다고 해서 분산 정보 외에 추가로 가정한 게 없다"는 뜻 — 사전 지식이 없을 때 정규 분포가 기본값인 이유가 여기서 완성된다.

### 🔑 정규 분포는 왜 파라미터가 μ,σ 둘뿐인가 — 균등 분포와의 대비

```python
# ② 표준정규 피크·면적·68-95-99.7
peak = stats.norm.pdf(0, 0, 1)
assert np.isclose(peak, 1 / np.sqrt(2 * np.pi))                      # 0.3989...
area, _ = quad(lambda x: stats.norm.pdf(x, 0, 1), -60, 60)
assert np.isclose(area, 1.0)
p1 = stats.norm.cdf(1) - stats.norm.cdf(-1)   # 68.27%
p2 = stats.norm.cdf(2) - stats.norm.cdf(-2)   # 95.45%
p3 = stats.norm.cdf(3) - stats.norm.cdf(-3)   # 99.73%

# ③ 이변량정규 — 원시형 vs Σ형 vs scipy 3자 대조
mu1, mu2, s1, s2, rho = 0.0, 0.0, 1.3, 1.3, 0.5
Sigma = np.array([[s1**2, rho*s1*s2], [rho*s1*s2, s2**2]])

def g_raw(x, y):                                    # 책 원시형(파라미터 나열)
    Sig = np.array([[s1**2, rho*s1*s2], [rho*s1*s2, s2**2]])
    Sig_inv = np.linalg.inv(Sig)
    d = np.array([x - mu1, y - mu2])                 # (2,) — 수식상 (2,1) 열벡터에 대응
    qf = d @ Sig_inv @ d
    return np.exp(-0.5*qf) / np.sqrt((2*np.pi)**2 * np.linalg.det(Sig))

def g_matrix(x, y):                                  # 책 Σ형(선형대수)
    u, mu = np.array([x, y]), np.array([mu1, mu2])
    Sig_inv = np.linalg.inv(Sigma)
    qf = (u - mu) @ Sig_inv @ (u - mu)
    return np.exp(-0.5*qf) / np.sqrt((2*np.pi)**2 * np.linalg.det(Sigma))

rv = stats.multivariate_normal(mean=[mu1, mu2], cov=Sigma)
for (x, y) in [(0,0), (1,1), (-1,2), (0.5,-0.5), (2,2)]:
    assert np.isclose(g_raw(x,y), g_matrix(x,y)) and np.isclose(g_raw(x,y), rv.pdf([x,y]))

# ④ R^2 전체 이중적분 = 1 (유효한 pdf인지 검증)
vol, err = dblquad(lambda y, x: g_raw(x, y), -12, 12, lambda x: -12, lambda x: 12)
assert abs(vol - 1.0) < 1e-4
print(f"68/95/99.7 정밀값: {p1*100:.2f}% {p2*100:.2f}% {p3*100:.2f}%")
print(f"이변량정규 중심 피크={g_raw(0,0):.4f}  이중적분={vol:.6f}(오차 {err:.1e})")
```
출력: `68/95/99.7 정밀값: 68.27% 95.45% 99.73%` / `이변량정규 중심 피크=0.1087  이중적분=1.000000(오차 9.6e-09)`.

| **정규 분포·가우스 분포** (normal/gaussian distribution) | 평균 μ 주변에 종 모양으로 몰리는 분포 — 파라미터 μ(위치)·σ(퍼짐) 둘; 68-95-99.7 규칙; 동일 분산 중 불확실성(엔트로피) 최대 |
| **표준 편차** σ([ˈsɪɡmə], 시그마, 시그마) | 정규 분포가 평균에서 얼마나 퍼졌는지 조절하는 파라미터 |
| **공분산 행렬** Σ (대문자 시그마) | 이변량 이상 정규 분포의 분산·상관을 담는 대칭 행렬 — 대각=각 분산, 비대각=$\rho\sigma_i\sigma_j$; [[22 확률·통계 용어 지도 — 확률 변수·분포·기대값·공분산·마르코프\|22]] 공분산 행렬 일반론의 정규분포 특수형 |
| **상관계수** ρ([roʊ], 로우, 로) | 두 변수의 선형 상관 정도, $-1{\le}\rho{\le}1$ — 이변량 정규 pdf의 다섯 번째 파라미터 |
| **68-95-99.7 규칙** (empirical rule) | 정규 분포에서 평균 ±1·2·3σ 이내에 각각 약 68%·95%·99.7% — σ 크기와 무관하게 항상 성립(표준화 불변) |
| **중심 극한 정리** (central limit theorem, CLT) | 동일 분포(정규 아니어도 됨)를 가진 다수의 독립 확률 변수의 **평균**이 정규 분포로 수렴 — 정규 분포가 자연계에 흔한 이유; [[22 확률·통계 용어 지도 — 확률 변수·분포·기대값·공분산·마르코프\|22]] 예고의 첫 실제 진술(정식 증명은 아직 🔭) |
