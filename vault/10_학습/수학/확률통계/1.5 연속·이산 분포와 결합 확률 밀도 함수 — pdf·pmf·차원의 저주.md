# 연속·이산 분포와 결합 확률 밀도 함수 — pdf·pmf·차원의 저주

> 📐 **수학 개념원자(SSoT)** — [[22 확률·통계 용어 지도 — 확률 변수·분포·기대값·공분산·마르코프|22 용어 지도]](MA §2.8)가 예고만 하고 넘어간 "연속 확률 변수 → pdf" "여러 확률 변수 → 결합 분포"를 실제로 가동한다. 핵심 전환: 연속에서는 **점의 확률이 항상 0**이라 질문 자체를 '값'에서 '구간'으로 바꿔야 하고, 확률 변수가 여럿이면 결합 분포를 전부 아는 대신 **한 변수를 고정해 슬라이스**하는 실전 기술이 필요하다.

> 🎯 **핵심 (TL;DR)**
> - **연속 분포에서 정확한 값을 관찰할 확률은 0** 📌 — 무한한 정밀도 때문에 '정확히 180cm'는 의미가 없다. 대신 **179.95<키<180.05 같은 구간**으로 질문하고 그 구간의 **면적**(pdf를 적분)으로 확률을 정량화한다.
> - **이산은 각 값이 명확히 구분**(주사위 5, 동전 앞면)되어 정확한 값의 확률을 그대로 말할 수 있다(1/6) — 숫자가 아닌 결과도 가능. 이 구분이 **pdf(연속)와 pmf(이산)** 두 함수로 갈라지는 이유다.
> - **결합 확률 밀도 함수 f(x₁,x₂)** = 여러 확률 변수를 함께 다루는 분포. **독립이면 결합=각 분포의 곱**(쉬움), **종속이면(키·체중처럼) 결합을 구하기가 매우 어렵다** — 종속 변수가 늘수록 저장·계산 차원이 기하급수적으로 느는 **차원의 저주(curse of dimensionality)** 📌.
> - 결합 pdf에서 **한 확률 변수를 특정 값으로 고정**하면(x₁=a) 그 슬라이스 f(a,x₂)는 **사후 확률 분포 f(x₂│x₁=a)에 비례**한다 📌(그림 2-7) — 결합 분포를 전부 아는 건 사치스러울 정도로 강력한 정보이고, 보통은 베이즈 정리+사전분포+가능도로 같은 사후분포에 도달한다([[1.3 조건부 확률·베이즈 정리 — 방향·뒤집기·조건부 결합·합|1.3]]).
> - 결합 분포를 **곱셈 규칙으로 조건부 확률들의 곱으로 분해**해 학습하면 생성 모델이 된다 — 딥마인드의 오디오 생성 모델 **WaveNet**이 이 원리를 쓴다 `(🔭 예고)`.

> [!question] 🎯 필요성 사슬 (결핍 → 필요 → 발명)
> *출발 문제:* [[1.4 이항 분포·베타 분포 — pmf·CDF·확률의 확률|1.4]]까지는 이산 확률 변수(k=성공 횟수)만 다뤘다. 그런데 키·시간·소득처럼 **실수값**을 갖는 데이터는 이산처럼 값을 셀 수 없다 — 179.95cm와 180.05cm 사이엔 무한히 많은 값이 있다.
> 1. 실수 범위의 값 하나를 "관찰할 확률"을 어떻게 정의하나 → 무한한 정밀도 때문에 **정확한 값의 확률은 0** → 질문을 '점'에서 '**구간**'으로 바꾸고 **확률 밀도 함수(pdf) + 구간 적분**을 발명.
> 2. 확률 변수가 하나가 아니라 여럿(키+체중)이면? → **결합 확률 밀도 함수** f(x₁,x₂) — 모든 변수 관계를 한 번에 표현.
> 3. 그런데 결합 분포를 통째로 구하는 건 종속 변수가 늘수록 기하급수적으로 어려워진다(**차원의 저주**) → 전부 알 필요 없이 **한 변수를 고정한 슬라이스**만으로도 다른 변수의 사후 확률을 (비례해서) 얻는 실전 기술이 필요.
> 4. 슬라이스조차 못 구하면? → [[1.3 조건부 확률·베이즈 정리 — 방향·뒤집기·조건부 결합·합|1.3]] **베이즈 정리**(사전분포+가능도)가 결합 분포 없이 같은 사후분포에 도달하는 우회로.
> **한 줄:** 점의 확률=0이라 구간으로(pdf) → 변수가 여럿이면 결합분포(사치) → 못 구하면 슬라이스(비례) → 그마저 없으면 베이즈(우회).

> [!abstract] 📘 책 핵심 — [필수수학AI] §2.9 연속 분포와 이산 분포
> - 연속 분포를 다룰 때는 '정확한 값'을 관찰·샘플링하는 대신 **'특정 값 주변의 데이터 포인트'**를 관찰·샘플링한다고 표현하는 것이 중요하다 — **연속 분포에서 정확한 값을 관찰할 확률은 0**이다.
> - 숫자가 연속적인 범위 안에 있을 때 어느 한 값과 다음 값 사이에는 이산적인 구분이 없다 — 실수는 무한하다. 예: 남자 키를 측정해 180cm를 얻었어도 정확히 180cm인지 180.0000000785cm인지 179.9999111134255cm인지 알 수 없다. 이 경우엔 **179.95<관찰한 키<180.05**와 같이 범위를 설정하고 범위 내에서 키가 관찰될 확률을 정량화하는 것이 낫다.
> - 반면 이산 확률 변수는 각각의 값을 쉽게 구분할 수 있다(주사위 5가 나올 확률=1/6이라 자신 있게 말할 수 있음). 이산 확률 변수는 숫자가 아닌 결과도 가질 수 있다(동전의 앞면·뒷면) — 연속 확률 변수는 숫자만 가질 수 있다.
> - 이 차이 때문에 연속 확률 변수는 **확률 밀도 함수(pdf)**, 이산 확률 변수는 **확률 질량 함수(pmf)**를 정의한다. 밀도는 지정된 길이·면적·공간(차원에 따라 다름)의 부피 내에 존재하는 물질의 양 — 질량을 구하려면 밀도에 해당 영역의 길이·면적·부피를 곱한다. 무한히 작은 단위당 밀도가 주어진 경우 적분은 무한히 많고 작은 영역의 합과 유사하므로, 질량을 구하려면 전체 영역에 대해 적분해야 한다.
> - 유의점(지금 단계, Chapter 11에서 자세히 `🔭`):
>   - 연속 확률 변수 1개 → 1차원 pdf: $P(179.95 < \text{height} < 180.05) = \int_{179.95}^{180.05} f(x_1)\,dx_1$
>   - 연속 확률 변수 2개(키·체중) → 2차원 결합 pdf $f(x_1,x_2)$의 이중 적분: $P(179.95{<}\text{height}{<}180.05,\ 70{<}\text{weight}{<}75) = \int_{70}^{75}\int_{179.95}^{180.05} f(x_1,x_2)\,dx_1 dx_2$
>   - 3개 이상(키·체중·혈압) → 고차원 결합 pdf의 삼중(이상) 적분: $P(a{<}x_1{<}b,\ c{<}x_2{<}d,\ e{<}x_3{<}f) = \int_e^f\int_c^d\int_a^b f(x_1,x_2,x_3)\,dx_1 dx_2 dx_3$
> - **실수의 무한함이 낳는 역설**: 모든 집합에 확률을 부여하고 모든 집합이 독립이라 가정하면, 집합들의 확률의 합이 1보다 큰 집합(프랙탈 모양의 집합, 유리수 집합을 변형해 구성한 집합 등)을 만들 수 있다는 모순이 생긴다. 수학의 **측정 이론**이 이 모순을 해결하며 확률 밀도 함수를 다룰 수 있는 수학적 뼈대를 제공한다 — 측정값이 0인 집합을 정의하고 어디서나 확률을 계산할 수 있는 정리들을 제공한다. 응용에는 이것만으로 충분하다.

> [!abstract] 📘 책 핵심 — [필수수학AI] §2.10 결합 확률 밀도 함수의 힘
> - 확률 변수가 여러 개 있을 때 이들의 결합 확률 분포를 구할 수 있으면 가장 좋겠지만 그런 경우는 드물다. 결합 확률 분포를 안다는 것은 각 확률 변수의 분포(**주변 분포**)뿐 아니라 이 확률 변수 간의 영향(**조건부 확률**)을 모두 안다는 뜻이다 — 도시 전체를 위에서 내려다보는 것처럼 모든 정보를 다 아는 것과 같다. 도시 안에 있으면 관찰할 수 있는 정보가 제한된다.
> - 확률 변수들이 **독립이면** 결합 분포는 각 확률 분포의 **곱으로 쉽게** 구할 수 있다. 하지만 사람의 키와 체중처럼 변수들이 **독립적이지 않은 경우** 결합 분포를 구하는 것은 매우 어렵다. 확률 변수가 종속적일 때는 결합 분포에서 종속 변수 분포만 떼어 내서 확인할 수 없다 — 두 개 이상의 변수가 종속적일 때는 변수들의 모든 값을 저장해야 각각의 분포를 구할 수 있다. 종속적인 확률 변수의 수가 늘어날수록 저장해야 하는 값, 계산하고 검색해야 하는 차원이 기하급수적으로 늘어나는데, 이를 **차원의 저주**라고 한다.
> - 결합 확률 밀도 함수 $f(x_1,x_2)$의 확률 변수 중 하나를 특정한 값으로 고정하면 **사후 확률 분포**(관찰값이 주어졌을 때 다른 확률 변수의 분포)에 비례하는 분포를 구할 수 있다. 예를 들어 $f(x_1,x_2)$에 대해 $x_1{=}a$라면 $f(a,x_2)$는 쉽게 구할 수 있다. 이는 사후 확률 분포 $f(x_2\mid x_1{=}a)$에 **비례**한다(그림 2-7). 이 경우는 결합 확률 분포를 알고 있는 것 — 매우 사치스러울 정도로 많은 정보를 알고 있는 것이다. 대부분의 경우 **베이즈 정리와 사전 분포 및 가능도 함수**를 사용해 동일한 사후 확률 분포를 얻는다.
> - 인공지능을 활용한 일부 애플리케이션에서는 **확률의 곱셈 규칙**을 사용해 결합 확률 분포를 학습하기도 한다. 결합 확률 분포를 조건부 확률의 곱으로 분리하여 학습하는데, 학습 후에 이 분포로부터 새롭고 흥미로운 데이터를 생성한다. **딥마인드는 이 원리를 이용해 WaveNet이라는 오디오를 생성하는 모델을 내놓기도 했다.**

## 🧠 근본 이해 · 융합 · 통찰

### 🔑 "비례"의 정확한 의미 — 정규화 상수만큼의 차이, 수치로 확인
- 책은 "$f(a,x_2)$는 사후 확률 분포 $f(x_2\mid x_1{=}a)$에 **비례**한다"고만 말하고 왜 '같다'가 아니라 '비례'인지는 설명하지 않는다. 이유: $f(a,x_2)$는 $x_2$에 대해 적분해도 1이 될 필요가 없다(정규화 안 된 슬라이스) — $x_2$로 적분해 **정규화 상수로 나눠야** 진짜 조건부 확률 $f(x_2\mid x_1{=}a)$가 된다. [[22 확률·통계 용어 지도 — 확률 변수·분포·기대값·공분산·마르코프|22]]의 "가능도 함수가 분포가 아닌 이유(합이 1일 필요 없음)"와 **정확히 같은 구조**다.
- 검증: $x_1{=}a{=}1.5$로 고정한 슬라이스를 정규화한 값과, 이변량 정규 조건부 분포의 닫힌 형태 공식 $\mathcal{N}\!\left(\mu_2+\rho\frac{\sigma_2}{\sigma_1}(a-\mu_1),\ \sigma_2^2(1-\rho^2)\right)$가 전 구간에서 오차 $10^{-8}$ 이내로 일치(코드 §). 조건부 평균이 $a{=}1.5$일 때 $0.75$로 이동함 — $\rho{=}0$ 대조군은 조건부 평균이 항상 $\mu_2{=}0$으로 **고정**된다(코드 §) — **독립이면 조건화가 무의미하다**는 [[1.3 조건부 확률·베이즈 정리 — 방향·뒤집기·조건부 결합·합|1.3]] 독립 정의의 수치적 재확인.

### 🔑 결합 분포=도시 조망 비유 — 1.2·1.3의 "정보량 서열"이 완성된다
- [[1.2 확률 계산 — 주변·결합·합·덧셈 법칙|1.2]]의 결합(AND)·주변 확률과 [[1.3 조건부 확률·베이즈 정리 — 방향·뒤집기·조건부 결합·합|1.3]]의 조건부 확률은 사실 **결합 분포라는 하나의 객체에서 뽑아낼 수 있는 두 단면**이었다 — 결합을 다 알면 주변(가장자리 합/적분)도 조건부(슬라이스)도 공짜로 나온다. 책의 "도시 전체 조망 vs 도시 안에서 보는 제한된 정보" 비유가 1.2~1.3에서 각각 따로 다룬 도구들의 **상위 구조**를 드러낸다.

### 🔑 차원의 저주 — 왜 종속이 이산과도, 연속과도 다르게 어려운가
- [[1.2 확률 계산 — 주변·결합·합·덧셈 법칙|1.2]]에서 독립 가정 시 결합=곱(쉬움)이었던 것과 정확히 대비된다. 종속 변수는 "표"(이산이면 셀 하나하나, 연속이면 곡면 하나하나)를 **통째로 저장**해야 하는데, 변수가 $n$개면 저장 공간이 (해상도)$^n$으로 폭발한다 — 신경망 가중치 행렬의 차원이 커질수록 파라미터가 폭증하는 것과 동형 구조 `(🔭 예고 — 딥러닝 실전)`.

## 🧩 코드 — 슬라이스=사후 확률(비례) 검증
> 🎯 목표: ① 이변량정규 슬라이스 정규화 = 조건부 닫힌형 공식 일치 확인 ② ρ=0 대조군으로 "독립=조건화 무의미" 수치 확인.
> 코드: `~/LOCAL/03-00_STUDIES/AI/02_optimization/ma2_9-12_distributions.py` (그림 재구성·전체 검증 스크립트에 포함)

```python
import numpy as np
from scipy import stats
from scipy.integrate import quad

mu1, mu2, s1, s2, rho = 0.0, 0.0, 1.3, 1.3, 0.5
Sigma = np.array([[s1**2, rho*s1*s2], [rho*s1*s2, s2**2]])
rv = stats.multivariate_normal(mean=[mu1, mu2], cov=Sigma)

a = 1.5  # x1 = a 고정 (슬라이스)
norm_const, _ = quad(lambda x2: rv.pdf([a, x2]), -30, 30)
slice_normalized = lambda x2: rv.pdf([a, x2]) / norm_const

cond_mean = mu2 + rho * (s2 / s1) * (a - mu1)          # 이변량정규 조건부 닫힌형
cond_var = s2**2 * (1 - rho**2)
cond_true = lambda x2: stats.norm.pdf(x2, cond_mean, np.sqrt(cond_var))

for x2 in [-2, -1, 0, 1, 2, 3]:
    assert np.isclose(slice_normalized(x2), cond_true(x2), atol=1e-8)
print(f"조건부 평균(a=1.5) = {cond_mean}")   # 0.75 — rho>0이라 a 쪽으로 이동

cond_mean_rho0 = mu2 + 0.0 * (s2 / s1) * (a - mu1)
print(f"rho=0 대조군 조건부 평균 = {cond_mean_rho0}")  # 0.0 — a와 무관, 독립=조건화 무의미
```
출력: `조건부 평균(a=1.5) = 0.75` / `rho=0 대조군 조건부 평균 = 0.0` — 슬라이스 정규화 == 조건부 공식 전 지점 일치(assert 통과).
