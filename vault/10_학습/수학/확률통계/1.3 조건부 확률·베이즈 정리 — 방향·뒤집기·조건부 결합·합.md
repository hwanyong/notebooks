# 조건부 확률·베이즈 정리 — 방향·뒤집기·조건부 결합·합

> 🎯 **핵심 (TL;DR)**
> - **조건부 확률(conditional probability([kənˈdɪʃənəl], 컨디셔널, 컨디셔널)) = 사건 B가 발생했을 때 A가 발생할 확률** — 표기 $P(A \text{ GIVEN(given([ˈɡɪvən], 기븐, 기븐)) } B)$ 또는 $P(A \mid B)$. 확률에서 **가장 헷갈리는 개념**(책).
> - **조건의 방향이 전부다** 📌: $P(\text{커피} \mid \text{암}) = 0.85 \ne P(\text{암} \mid \text{커피}) = 0.0065$ — "암 환자가 커피를 마실 확률"과 "커피를 마시는 사람이 암에 걸릴 확률"은 다른 질문. 방향 착각이 "암 환자의 85%가 커피를…" 류 엉터리 기사의 원료.
> - **베이즈 정리(Bayes' Theorem([ˈθiːərəm], 시어럼, 시어럼(정리))) = 조건 뒤집기** 📌: $P(A \mid B) = \dfrac{P(B \mid A)\,P(A)}{P(B)}$ — 반대 방향 조건부만 있을 때 원하는 방향을 계산. 새 정보로 믿음을 업데이트하는 연결 고리로도 사용.
> - **조건부 결합(일반 곱셈 법칙)** 📌: $P(A \text{ AND } B) = P(A \mid B) \times P(B)$ — **종속 여부와 무관하게 성립**(1.2 곱셈 정리의 독립 딱지 제거). 두 방향이 같은 값: $P(\text{커피}\mid\text{암})P(\text{암}) = P(\text{암}\mid\text{커피})P(\text{커피}) = 0.00425$.
> - **조건부 합(덧셈 법칙 일반형)** 📌: $P(A \text{ OR } B) = P(A) + P(B) - P(A \mid B) \times P(B)$ — 상호 배타면 $P(A \mid B) = 0$이라 자동으로 단순 합 환원.
> - **독립의 정식 정의**(1.2 예고 회수): $P(A \mid B) = P(A)$ — B가 일어나도 A의 가능성에 아무 차이가 없다.

> [!abstract] 📘 책 핵심 — [필수수학] §2.2.3 조건부 확률과 베이즈 정리
> **① 정의와 방향**
> - 확률에서 가장 헷갈리는 개념 = **조건부 확률**: 사건 B가 발생했을 때 사건 A가 발생할 확률. 표기 $P(A \text{ GIVEN } B)$ 또는 $P(A \mid B)$.
> - 예: "암 환자의 85%가 커피를 마신다"는 연구 주장 = $P(\text{커피} \mid \text{암}) = 0.85$ — *암에 걸린 사람이* 커피를 마실 확률. 이걸 듣고 커피를 끊어야 하나?
> - 미국 기준 비교: $P(\text{커피}) = 0.65$ (statista.com) · $P(\text{암}) = 0.005$ (cancer.gov, 0.5%).
> - 생각해 보기: 커피가 암 발생에 기여한다면 암 발생률이 0.5%보다 훨씬 높아야(65%에 가까워야) 하지 않나?
> - **비율 숫자의 교묘함**: 맥락 없이 중요해 보이게 착각시킬 수 있고 뉴스 클릭 유도에 악용됨 — *일반적인 속성*(커피)을 *흔치 않은 속성*(암)과 연관시킨 엉터리 기사.
> - 혼동의 원인 = **조건의 방향**이 중요한데 두 조건을 동일하다고 착각: '커피를 마시는 사람이 암에 걸릴 확률' ≠ '암에 걸린 사람이 커피를 마실 확률'. 커피 마시는 사람 중 암 환자는 거의 없지만, 암 환자의 다수는 커피를 마신다.
>
> **② 베이즈 정리 — 조건 뒤집기**
> - 커피-암 연관을 연구하려면 $P(\text{암} \mid \text{커피})$가 필요한데 가진 건 $P(\text{커피} \mid \text{암}) = 0.85$ → **베이즈 정리**로 뒤집는다:
> $$P(A \mid B) = \frac{P(B \mid A)\,P(A)}{P(B)}, \qquad P(\text{암} \mid \text{커피}) = \frac{P(\text{커피} \mid \text{암}) \times P(\text{암})}{P(\text{커피})} = \frac{0.85 \times 0.005}{0.65} = 0.0065$$
> - [예제 2-1] 파이썬 계산: 세 변수 대입 → `0.006538461538461539` (아래 코드 §에서 분수로 재검증).
> - **해석: 커피를 마시는 사람이 암에 걸릴 확률은 0.65%에 불과** — 암 환자가 커피를 마실 확률 85%와는 매우 다르다. 조건의 방향이 중요한 이유. 베이즈 정리는 조건 뒤집기 + **새 정보로 믿음을 계속 업데이트**하기 위해 여러 조건부 확률을 연결할 때도 사용.
> - 베이즈 정리 이면의 직관 심화는 **부록 A.4** 참조 `(🔭 예고)` — 지금은 "조건부 확률을 뒤집는 도구"라는 것만.
> - [TIP] **나이브 베이즈(Naive([naɪˈiːv], 나이이브, 나이브) Bayes)**: 베이즈 정리가 중요한 역할을 하는 유명 ML 알고리즘 `(🔭 예고 — 조엘 그루스 『밑바닥부터 시작하는 데이터 과학』; 옮긴이: 『실무로 통하는 ML 문제 해결 with 파이썬』 18장)`.

> [!abstract] 📘 책 핵심 — [필수수학] §2.2.3 [박스] '커피를 마시는 사람'을 정의하는 기준은?
> - 매일 마시는 사람과 한 달에 한 번 마시는 사람을 똑같이 '커피를 마시는 사람'으로? 20년 마신 사람과 한 달 전 시작한 사람은?
> - **데이터가 전체 스토리의 일부만 이야기하는 원인** — 예/아니요 환자 목록이면 특정 임곗값을 정의해야 하고, 아니면 '지난 3년 커피 마신 횟수' 같은 가중 지표가 필요.
> - 예제는 자격 정의 없이 간단화했지만 **실제 현장에서는 항상 데이터에서 실마리를 찾아라** `(🔭 예고 — 3장 상세)`.

> [!abstract] 📘 책 핵심 — [필수수학] §2.2.4 결합 및 합 조건부 확률
> **① 조건부 결합 — 어느 확률을 곱하나**
> - "전체 인구 중 **암에 걸렸으면서 커피를 마시는** 사람의 확률"? 옵션 1: $P(\text{커피}) \times P(\text{암}) = 0.65 \times 0.005 = 0.00325$ / 옵션 2: $P(\text{커피} \mid \text{암}) \times P(\text{암}) = 0.85 \times 0.005 = 0.00425$.
> - 커피 확률이 *암에 걸린 사람에게* 적용되는 것이므로 더 구체적이고 현재 조건에 부합하는 $P(\text{커피} \mid \text{암})$을 써야 한다 — **결합 확률에 이미 $P(\text{암})$이 포함**되어 있으므로:
> $$P(\text{커피 AND 암}) = P(\text{커피} \mid \text{암}) \times P(\text{암}) = 0.85 \times 0.005 = 0.00425 \;(0.425\%)$$
> - **다른 방향으로도 동일**: $P(\text{암} \mid \text{커피}) \times P(\text{커피}) = 0.0065 \times 0.65 = 0.00425$ — 같은 값! (정확분수 검산은 코드 § · ⚠️ 반올림 함정은 🔑)
> - 조건부 확률이 없다면 $P(\text{커피}) \times P(\text{암}) = 0.00325$가 최선.
> - **독립의 뜻**: B가 A에 영향을 주지 않으면 $P(A \mid B) = P(A)$ — B가 발생해도 A의 가능성에 아무 차이 없음. 따라서 **종속 여부와 관계없이** 결합 공식을 일반화할 수 있다:
> $$P(A \text{ AND } B) = P(A \mid B) \times P(B)$$
>
> **② 조건부 합 — 덧셈 법칙의 일반형**
> - A가 B의 확률에 영향을 줄 수 있는 경우 덧셈 법칙을 수정:
> $$P(A \text{ OR } B) = P(A) + P(B) - P(A \mid B) \times P(B)$$
> - **상호 배타적인 사건에도 적용**: A와 B가 동시에 발생할 수 없으면 $P(A \mid B) \times P(B) = 0$.

## 🧠 근본 이해 · 융합 · 통찰

### 🔑 조건부 = 분모(모집단)의 교체 — 85%와 0.65%는 같은 교집합을 다른 분모로 나눈 것
- 두 조건부는 **동일한 결합 확률 $P(\text{커피 AND 암}) = 0.00425$** 를 서로 다른 모집단으로 나눈 값이다: $\frac{0.00425}{P(\text{암})=0.005} = 0.85$(분모=희귀한 암 환자) vs $\frac{0.00425}{P(\text{커피})=0.65} = 0.0065$(분모=흔한 커피 인구). **분자는 같고 분모가 130배 다르니 결과도 130배 차이** — "조건의 방향" 혼동의 정체는 분모 착각이다.
- [[1.1 확률 — 정의·가능도·오즈|1.1]] 오즈의 "분모 교체" 통찰의 재등장: 확률의 얼굴은 분모가 결정한다.

### 🔑 베이즈 정리는 새 공리가 아니다 — 결합의 두 방향 분해를 재배열한 것
- §2.2.4가 수치로 실연한 $P(A \mid B)P(B) = P(B \mid A)P(A)\;(=P(A \text{ AND } B))$ — 이 **한 등식을 $P(B)$로 나누면 그대로 베이즈 정리**다. 책은 §2.2.3(정리 제시)→§2.2.4(동일성 실연) 순서지만, 논리적 유래는 역방향: 결합의 대칭성이 먼저고 베이즈는 그 재배열. *(§2.3 도입부에서 확인된 옮긴이 각주 3도 같은 등식 — "P(커피|암)×P(암) = P(커피)×P(암|커피)" — 을 명시한다.)*
- 그래서 외울 건 사실상 하나 — "결합은 어느 쪽으로 분해해도 같다". 뒤집기 공식은 여기서 30초 만에 복원된다.
- ✅ **책의 확인(2026-07-16, [필수수학AI] §2.8.5~2.8.6 다권 보강):** MA가 이 통찰을 공식 서술로 뒷받침 — "**모든 확률 변수의 결합 확률 분포 전체를 알 수 있다면 베이즈 규칙은 불필요**하다. 베이즈 규칙은 결합 분포를 알지 못할 때 원하는 조건부 확률을 계산하는 데 활용된다"(2.8.6 NOTE). 또 §2.2.4의 일반 곱셈 법칙이 "조건부 확률의 곱 또는 **연쇄 규칙**(chain rule)"이라는 정식 이름을 얻었다(2.8.5) — 렌즈 03의 시퀀스 확률 $\prod P(w_t \mid w_{<t})$ 가 바로 그 연쇄. 결합 분포를 한 변수 고정으로 분할하면 조건부 분포([그림 2-7] 미수신)라는 분포-레벨 서술도 추가 — [[22 확률·통계 용어 지도 — 확률 변수·분포·기대값·공분산·마르코프|22 용어 지도]].

### 🔑 ⚠️ 반올림 함정 — "0.0065 × 0.65 = 0.00425"는 표기상으로만 참
- 실제로 $0.0065 \times 0.65 = 0.004225 \ne 0.00425$. 등식이 성립하는 건 **반올림 전 정확값** $P(\text{암}\mid\text{커피}) = \frac{17}{2600} = 0.006538\ldots$ 을 쓸 때뿐: $\frac{17}{2600} \times \frac{13}{20} = \frac{17}{4000} = 0.00425$ (코드 §에서 분수 검증).
- 교훈: **동일성 검증은 반올림 값이 아니라 정확값으로** — 중간 반올림이 등식을 조용히 깨뜨린다([[T09 확률은 왜 0.0~1.0인가 — 공리·곱셈 닫힘·부동소수점 팩트체크·로그 공간|T09]]의 float 정밀도 이야기와 같은 계열).

### 🔑 ⚠️ "0.65%에 불과"의 반쪽 — 올바른 비교 대상은 85%가 아니라 기저율 0.5%다
- 책의 서사는 "85% vs 0.65%"의 대비로 안심시키지만, 역학적으로 의미 있는 비교는 $P(\text{암} \mid \text{커피})$ vs $P(\text{암})$ — **이 수치대로라면 $\frac{0.0065}{0.005} = \frac{17}{13} \approx 1.31$, 커피 인구의 암 확률이 기저율보다 31% 높다**(리프트). 그 이유는 $\frac{P(\text{커피}\mid\text{암})}{P(\text{커피})} = \frac{0.85}{0.65}$ 가 1보다 크기 때문(두 비는 항상 같다 — 베이즈에서 즉시).
- 단, 이걸 "커피가 암을 늘린다"로 읽으면 그 또한 오독: 책 수치는 **서로 다른 출처·모집단**(cancer.gov/statista/가상의 연구)의 조합인 데다, [박스]가 지적하듯 '커피를 마시는 사람'의 정의조차 모호하다. 인과는커녕 연관조차 이 숫자만으론 확정 불가 → 데이터 검증의 영역 `(🔭 예고 — 3장 가설 검정)`.
- **정리: 조건부 확률을 읽는 순서 = ① 방향 확인 ② 기저율과 비교 ③ 데이터의 자격(정의·출처) 심문.** 책이 ①을, 이 절이 ②③을 채운다.

### 🔑 1.2 ⚠️의 회수 — 조건부 형태가 '진짜 일반형'이다
- [[1.2 확률 계산 — 주변·결합·합·덧셈 법칙|1.2]]에서 "곱 대입형 $-P(A){\times}P(B)$는 독립 전용(배타에 쓰면 $\frac{11}{36}$ 오답)"이라 경고했다 — §2.2.4가 그 빈자리를 채운다: $-P(A \mid B){\times}P(B)$ 는 $P(A \text{ AND } B)$ 그 자체라 **독립이든 종속이든 배타든 항상 옳다**(배타면 $P(A \mid B) = 0$ → 단순 합 환원, 검산: $P(4 \text{ OR } 6) = \frac{1}{6}+\frac{1}{6}-0 = \frac{1}{3}$ ✓).
- 공식 계보 완성: 열거(진리) → 독립 특수형 $P(A)P(B)$(1.2) → **조건부 일반형 $P(A \mid B)P(B)$(1.3)** — 독립은 $P(A \mid B) = P(A)$인 특수 사례로 강등.

## 🧩 코드 (§2.2.3~2.2.4)
> 🎯 목표: ① 베이즈 뒤집기 $\frac{17}{2600}$(예제 2-1 값) ② 결합 두 방향이 정확분수로 동일($\frac{17}{4000}$) ③ 반올림 곱 $0.004225 \ne 0.00425$ 함정 확인 ④ 리프트 $\frac{17}{13}$ 두 경로 일치 ⑤ 배타 사건에서 조건부 합의 자동 환원.
> 코드: `~/LOCAL/03-00_STUDIES/AI/02_optimization/md2_3_conditional_bayes.py`

```python
from fractions import Fraction as F

p_coffee, p_cancer = F(65, 100), F(5, 1000)     # 0.65, 0.005
p_coffee_g_cancer = F(85, 100)                  # P(커피|암) = 0.85

# ① 베이즈 정리 — 조건 뒤집기 (예제 2-1의 분수판)
p_cancer_g_coffee = p_coffee_g_cancer * p_cancer / p_coffee
print(p_cancer_g_coffee, float(p_cancer_g_coffee))   # 17/2600 0.006538...

# ② 결합의 두 방향 분해 = 동일 (베이즈의 유래)
j1 = p_coffee_g_cancer * p_cancer               # P(커피|암)P(암)
j2 = p_cancer_g_coffee * p_coffee               # P(암|커피)P(커피)
print(j1, j2, j1 == j2)                          # 17/4000 17/4000 True (= 0.00425)

# ③ ⚠️ 반올림 함정 — 표기값 곱은 등식을 깬다
print(0.0065 * 0.65)                             # 0.004225 ≠ 0.00425

# ④ 리프트(기저율 대비) — 두 경로가 같은 17/13 ≈ 1.31
print(p_cancer_g_coffee / p_cancer, p_coffee_g_cancer / p_coffee)   # 17/13 17/13

# ⑤ 조건부 합의 배타 자동 환원: P(4|6)=0 → 단순 합
p4_or_6 = F(1,6) + F(1,6) - 0 * F(1,6)           # P(4|6)=0
print(p4_or_6)                                   # 1/3 — 1.2의 11/36 오답 문제 해소
```

## 🆕 용어
| 용어 | 한 줄 정의 |
|---|---|
| **조건부 확률** (conditional probability, $P(A \mid B)$) | 사건 B가 발생했을 때 A가 발생할 확률 — $P(A \text{ GIVEN } B)$; **방향이 다르면 다른 질문**($P(\text{커피}\mid\text{암}) \ne P(\text{암}\mid\text{커피})$) |
| **베이즈 정리** (Bayes' Theorem) | $P(A \mid B) = \frac{P(B \mid A)P(A)}{P(B)}$ — 조건부 확률의 방향 뒤집기; 믿음 업데이트의 연결 고리; 유래는 결합의 두 방향 분해 동일성 |
| **조건부 결합 (일반 곱셈 법칙)** | $P(A \text{ AND } B) = P(A \mid B) \times P(B)$ — 종속 여부 무관 일반형; 두 방향 분해가 같은 값 |
| **조건부 합 (덧셈 법칙 일반형)** | $P(A \text{ OR } B) = P(A) + P(B) - P(A \mid B) \times P(B)$ — 배타면 $P(A \mid B)=0$으로 자동 환원 |
| **독립 (정식 정의)** | $P(A \mid B) = P(A)$ — B가 발생해도 A의 가능성에 차이 없음 (1.2 "가정" 예고의 회수) |
| **리프트** (lift, 사이드) | $\frac{P(A \mid B)}{P(A)} = \frac{P(B \mid A)}{P(B)}$ — 기저율 대비 조건부의 배율; 1이면 독립, 커피-암 수치로는 $\frac{17}{13} \approx 1.31$ |
