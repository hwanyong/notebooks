### [[13 행렬 전치 — 내적·외적 표기법·LIVE EVIL·대칭 행렬|13 행렬 전치 — 표기법·LIVE EVIL·대칭 행렬]]

### [[15 행렬 노름·대각합 — 프로베니우스·ℓ2·tr(AᵀA)|15 행렬 노름·대각합 — 프로베니우스·ℓ2·tr(AᵀA)]]
| 앞면 | 뒷면 |
|---|---|
| 행렬 노름은 하나? | **아니다** — 여러 개; 각각 행렬을 특징짓는 한 숫자; 표기 ‖A‖ |
| 행렬 노름 두 계열? | **원소별**(원소 크기) vs **유도**(변환의 크기 조정 배율, 6·13장 예고) |
| 프로베니우스 공식? | ‖A‖F=√(ΣᵢΣⱼa²ᵢⱼ) — 전 원소 제곱합의 제곱근(첨자 F) |
| 프로베니우스의 정체? | 행렬을 펴서(vectorize) 벡터 ℓ2 — 새 공식 아님(뿌리: 거대해진 피타고라스 → 벡터 한 뿌리) |

| 정규화의 기본 생각? | 노름을 비용 함수에 추가 — 가중치 크기 벌금 |
| L2 vs L1? | L2=릿지(비대 방지) / L1=라쏘(희소 방지) |
| 이동량 λ 기준(회수)? | ‖A‖F 등 행렬 자체 값의 일부 — 스케일 비례 |
| 대각합 tr(A) 정의·조건? | 대각 원소의 합; **정방 행렬만** |

| ‖A‖F 를 tr로? 📌 | √tr(AᵀA) — AᵀA 대각=각 열의 자기 내적(제곱합) |
| √tr(AᵀA)로 계산하면? | 개념은 아름답지만 낭비 — 실전은 `(A**2).sum()` |
| '대각'=주대각선(i=j) 하나뿐인 이유? | ① 자기→자기 성분(yᵢ 중 aᵢᵢxᵢ 직행분) ② 기저 불변 — tr(P⁻¹AP)=tr(A) |

### [[16 행렬 공간 — 열공간·행공간·영공간|16 행렬 공간 — 열공간·행공간·영공간]]
