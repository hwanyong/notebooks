# 3단계. 현대 AI의 코어 엔진 정복 — 고급 행렬 분해 & 트랜스포머/LoRA 생코딩

> **목표:** LLM과 미세튜닝(LoRA) 아키텍처를 순수 코드로 밑바닥부터 제작. 시중 파인튜닝 기법이 사실은 **'SVD 기반 행렬의 낮은 계수 근사화'** 를 가중치 레이어에 이식한 것임을 이해하며 거품 걷어내기.

🔙 코스 허브: [[_커리큘럼 MOC]]

- [ ] 직교 행렬·그람-슈미트 과정 — **[실전선대] 8.1~8.2**
- [ ] 가우스 소거법·가우스-조던 행 축소의 수치해석 원리 — **[실전선대] 9.1~9.2**
- [ ] 고윳값/고유벡터 기하·정방행렬 대각화·대칭행렬 직교 고유벡터 — **[실전선대] 12.1~12.5 / [필수수학] 4.7**

- [ ] **특잇값 분해(SVD)** 구성요소·기하 메커니즘, 고윳값 분해와의 차이 증명 — **[실전선대] 13.1~13.4 / [필수수학AI] 6.1~6.8**

- [ ] 선형판별분석(LDA) — **[실전선대] 14.2** _(2026-07-15 전수조사 발견)_
- [ ] SVD의 소셜 미디어 응용 — **[필수수학AI] 6.13** _(2026-07-15 전수조사 발견)_
- [ ] 자연어 벡터 표현·지프의 법칙·어텐션 수리 해석 — **[필수수학AI] 7.1~7.8**
- [ ] 시계열 데이터를 위한 CNN·RNN(GRU/LSTM)·자연어 예제·금융 인공지능 — **[필수수학AI] 7.9~7.12** _(2026-07-15 전수조사 발견 — 특히 7.10 RNN/LSTM 비중 있음)_

- [ ] 트랜스포머/GPT 아키텍처 구조 + 데이터셋 텐서 흐름 조망 — **[밑바닥LLM] 1.4~1.6**

## 🔭 확장 참조 (아키텍처 부채 — 2026-07 갭 분석)
> [!info] 🔭 왜 이 표가 있나
> 밑바닥LLM은 GPT-2식(절대위치임베딩·LayerNorm·GELU·바닐라 MHA)을 만든다. 2023년 이후 대부분 오픈모델(Llama·Mistral·Qwen·DeepSeek)의 표준 구성과는 다르다. 지금 배우지 않고, GPT 블록이 실제로 돌아간 뒤에만 연다 — 컴포넌트를 하나씩 교체하며 대조하는 방식으로.

| 트리거 | 내용 | 자원 | 규모 |
|---|---|---|---|
| GPT 블록 완성 직후 | 절대위치임베딩 → RoPE 교체 | Modernizing nanoGPT(GQA·RoPE·SwiGLU) | 3~5h |
| 〃 | LayerNorm→RMSNorm, GELU→SwiGLU 교체 | 〃 | 3~5h |
| 〃 | 바닐라 MHA → GQA 교체 | 〃 | 3~5h |
| 〃 (선택) | FFN → 토이 MoE(전문가 N개+라우터) | — | 5~10h |
| [밑바닥LLM] 5.4~5.5(사전훈련 가중치 로드) 직후 | 스케일링 법칙 — 파라미터·데이터·연산량 최적 배분 | Kaplan 2020·Hoffmann 2022(Chinchilla) 논문 | 미정 |
| 〃 | 해석가능성 — 모델 내부 회로 분석 입문 | Anthropic Transformer Circuits(transformer-circuits.pub)·Neel Nanda 튜토리얼 | 미정 |

🔗 정렬·추론모델(DPO·GRPO) 확장 참조는 [[_4단계 - 커리어 분기]]에.
