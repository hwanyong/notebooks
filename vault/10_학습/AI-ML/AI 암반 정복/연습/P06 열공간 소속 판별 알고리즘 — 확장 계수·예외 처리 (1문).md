# P06 열공간 소속 판별 알고리즘 — 확장 계수·예외 처리 (1문)

> ✏️ **연습 세트 (책 공식 [연습 문제 5-9]).** [[2.6 행렬 공간 — 열공간·행공간·영공간|2.6]]이 던진 질문("$v\in C(A)$?")과 [[2.7 행렬 계수(rank) — 정보 차원·최대계수·이동·확장|2.7]]이 준 도구(확장 계수 비교)를 실제로 코드화하는 문제. 정답은 접이식. 운영 규칙: [[_연습 MOC]]
> 🎯 **이 세트가 시험하는 것:** ① 확장(augmenting) 계수 비교로 소속 판별 ② 계수-3인 $4{\times}3$ 행렬에서 임의의 $v\in\mathbb{R}^4$ 가 열공간에 있을 확률(측도 0) ③ 최대계수 $4{\times}4$ 행렬은 **항상** $v\in C(A)$ 인 이유(각주 7) ④ 입력 검증(예외 처리).

## 문제

**P1 (책 [연습 5-9]).** 정규분포 난수로 계수-3 행렬 $A\in\mathbb{R}^{4\times3}$ 과 벡터 $v\in\mathbb{R}^4$ 를 만들고, $v\in C(A)$ 인지 판별하는 알고리즘을 구현하라. 여러 번 실행해서 일관된 패턴이 보이는지 확인하라. 그런 다음 $A\in\mathbb{R}^{4\times4}$ 계수-4 행렬을 쓰면 $A$ 가 $4{\times}4$ 난수 행렬일 때 **항상** $v\in C(A)$ 를 찾을 수 있다는 데 확신한다 — 왜 그런가? 마지막으로 True/False를 반환하고 벡터 크기가 안 맞을 때 예외를 던지는 함수로 정리하라.
- 시험 개념: 확장 계수 비교 = 소속 판별 — [[2.7 행렬 계수(rank) — 정보 차원·최대계수·이동·확장|2.7 §5.4.1]] · 최대계수 열공간 = 전체 공간 — [[2.6 행렬 공간 — 열공간·행공간·영공간|2.6]]

---

> [!quote]- ✅ 정답·풀이
> **알고리즘:** $v\in C(A) \iff \mathrm{rank}(A) = \mathrm{rank}([A\mid v])$ — $v$ 를 열로 덧붙였는데도(확장) 독립 방향 수가 그대로면, $v$ 는 이미 $A$ 의 열들이 만드는 결합으로 표현 가능하다는 뜻([[2.7 행렬 계수(rank) — 정보 차원·최대계수·이동·확장|2.7]] 의 확장 판별 그대로).
> ```python
> import numpy as np
> def in_column_space(A, v, tol=1e-8):
>     A = np.asarray(A, dtype=float); v = np.asarray(v, dtype=float).reshape(-1)
>     if v.shape[0] != A.shape[0]:
>         raise ValueError(f"v의 길이({v.shape[0]})가 A의 행 수({A.shape[0]})와 다릅니다 — Av 곱이 정의되지 않음")
>     r_A = np.linalg.matrix_rank(A, tol=tol)
>     r_aug = np.linalg.matrix_rank(np.hstack([A, v.reshape(-1,1)]), tol=tol)
>     return r_A == r_aug
> ```
> **일관된 패턴 (4×3, 계수-3):** 30회 반복 시행 결과 **0/30**이 True — $v\in C(A)$ 가 사실상 한 번도 성립하지 않는다. 이유: $C(A)$ 는 $\mathbb{R}^4$ 안의 딱 **3차원** 부분공간("얇은 종이", [[2.6 행렬 공간 — 열공간·행공간·영공간|2.6]])인데, $v$ 는 연속분포에서 뽑은 **4차원 전체**의 임의의 점이다 — 3차원 평면이 4차원 공간에서 차지하는 "부피"는 0이라, 임의의 연속 확률 변수가 정확히 그 평면 위에 떨어질 확률은 **이론상 정확히 0**(측도 0 사건). 유한 소수점 연산이라 완전히 0은 아니지만 30회에서 단 한 번도 관측되지 않을 만큼 극히 드물다.
> **일관된 패턴 (4×4, 계수-4):** 30회 반복 시행 결과 **30/30**이 True — 항상 성립. 이유(각주 7 재확인): $4{\times}4$ **최대계수** 행렬의 열공간은 $\mathbb{R}^4$ 전체를 생성한다 — 독립인 열이 4개면 그 4개가 $\mathbb{R}^4$ 의 완전한 기저([[1.8 벡터 확장 — 집합·선형결합·선형독립·부분공간·생성·기저|1.8]])가 되므로, $\mathbb{R}^4$ 의 **모든** 벡터가 예외 없이 그 기저의 선형결합으로 표현된다. "얇은 종이" 밖이 존재하지 않는 특수 케이스.
> **예외 처리:** 길이가 안 맞는 $v$(예: $A\in\mathbb{R}^{4\times3}$ 에 $v\in\mathbb{R}^5$)를 넣으면 `ValueError`를 던진다 — $Av$ 자체가 정의되지 않는 입력이므로 조용히 틀린 답(또는 브로드캐스팅 에러)을 내기 전에 명시적으로 막는다.
> ```python
> import numpy as np
> rng = np.random.default_rng(3)
> res43 = [in_column_space(rng.standard_normal((4,3)), rng.standard_normal(4)) for _ in range(30)]
> assert sum(res43) == 0                      # 4x3 계수-3: 항상 False
> res44 = [in_column_space(rng.standard_normal((4,4)), rng.standard_normal(4)) for _ in range(30)]
> assert sum(res44) == 30                     # 4x4 계수-4: 항상 True
> try:
>     in_column_space(rng.standard_normal((4,3)), rng.standard_normal(5)); assert False
> except ValueError:
>     pass                                     # 예외 정상 발생
> ```
