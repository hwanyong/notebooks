# P03 프로베니우스 노름·대각합 — √tr(AᵀA) 두 우회로 (9문)

> ✏️ **연습 세트 (P1~P6 자체 출제 + P7~P9 책 공식 연습).** 정답은 접이식 — 펼치기 전에 손으로 풀 것. 운영 규칙: [[_연습 MOC]]
> 🎯 **이 세트가 시험하는 것:** ① 식 5-1 직접 계산 ② $A^{\mathsf{T}}A$ 의 대각 = **열** 제곱합 확인 ③ 비정방에서도 공식이 성립하는 이유(tr(A) 자체는 불가) ④ $AA^{\mathsf{T}}$ 우회로(**행** 제곱합)와 tr 일치 — 책의 "동일한 행에 대한 내적" 문구 해명 ⑤ "노름=대각 구하기" 오독 격파 ⑥ 코드 4중 검산 + `np.trace` 함정 ⑦ 노름의 스칼라 비례성·비음성(5-1) ⑧ 거리 수렴을 닫힌 식으로(5-2) ⑨ 이동이 노름·상관·거리에 미치는 서로 다른 영향(5-4).

## A. 손계산 — 공식 그대로 (§5.1.1)

**P1.** $A=\begin{bmatrix}2&1\\0&3\end{bmatrix}$ — ⑴ 식 5-1로 $\lVert A\rVert_F$ ⑵ $A^{\mathsf{T}}A$ 를 계산하고 **각 대각 원소가 어느 열의 제곱합인지** 짚어라 ⑶ $\sqrt{tr(A^{\mathsf{T}}A)}$ 가 ⑴과 같은지 확인.
- 시험 개념: 식 5-1 = √tr(AᵀA) 두 경로 일치 — [[2.5 행렬 노름·대각합 — 프로베니우스·ℓ2·tr(AᵀA)|2.5]]

**P2.** $R=\begin{bmatrix}1&2\\3&4\\5&6\end{bmatrix}$ ($3{\times}2$ 비정방) — ⑴ $R^{\mathsf{T}}R$ 로 $\lVert R\rVert_F$ ⑵ 식 5-1 직접 계산과 대조 ⑶ $tr(R)$ 자체는 왜 계산 대상이 아닌가?
- 시험 개념: 비정방에도 노름은 존재하지만 **대각합은 정방 전용** — 우회로가 성립하는 건 $R^{\mathsf{T}}R$ 이 *항상* 정방이기 때문 ([[2.3 행렬 전치 — 내적·외적 표기법·LIVE EVIL·대칭 행렬|2.3 곱셈 기법]])

## B. 두 우회로 — AᵀA(열) vs AAᵀ(행)

**P3.** 같은 $R$ 로 $RR^{\mathsf{T}}$ ($3{\times}3$)의 **대각만** 계산하라 — ⑴ 각 대각 원소가 어느 **행**의 제곱합인지 ⑵ $tr(RR^{\mathsf{T}})$ 와 P2의 $tr(R^{\mathsf{T}}R)$ 비교 ⑶ 두 행렬은 크기부터 다른데($2{\times}2$ vs $3{\times}3$) 대각합이 같은 이유를 한 문장으로.
- 시험 개념: 📌 $tr(A^{\mathsf{T}}A)=tr(AA^{\mathsf{T}})$ — 전 원소 제곱합을 **열로 묶나 행으로 묶나** 총합 동일. 책 §5.1.1의 "각 대각 원소는 동일한 **행**에 대한 내적" 문구는 $AA^{\mathsf{T}}$(행) 시점 — $A^{\mathsf{T}}A$ 는 **열** 시점이고 공식은 양쪽 다 성립.

## C. 오독 격파 (막힌 점 회수)

**P4.** 누가 주장한다: "$A=\begin{bmatrix}1&2\\3&4\end{bmatrix}$ 의 프로베니우스 노름은 대각 원소로 $\sqrt{1^2+4^2}=\sqrt{17}$." — 수치로 반박하고, 이 주장이 **어느 행렬의 대각**을 혼동했는지 지적하라.
- 시험 개념: "노름 = 대각 구하기" 오독 — 정의(식 5-1)는 **전 원소**; 대각은 $A$ 가 아니라 $A^{\mathsf{T}}A$ 의 것 ([[2.5 행렬 노름·대각합 — 프로베니우스·ℓ2·tr(AᵀA)|2.5 막힌 점]])

**P5.** $\sqrt{tr(A^{\mathsf{T}}A)}$ 는 개념 통합엔 아름답지만 계산으로는 낭비다 — $N$ 열 행렬에서 **무엇을 계산하고 버리는지** 구체적으로 말하고, 실전 한 줄 코드를 써라.
- 시험 개념: 비대각 = 서로 다른 열들의 내적(쌍 수 $N(N{-}1)/2$)을 만들고 버림 — 실전은 원소별 직행

## D. 코드 검산

**P6.** NumPy로 ⑴ `(A**2).sum()**0.5` ⑵ `np.linalg.norm(A,'fro')` ⑶ `np.linalg.norm(A.ravel(),2)` ⑷ `np.sqrt(np.trace(A.T@A))` 4중 일치를 확인하고, ⑸ **함정**: 비정방 `R`(3×2)에 `np.trace(R)` 를 호출하면 에러가 날까, 값이 나올까? 나온다면 그 값은 수학적으로 무엇인가?
- 시험 개념: 4중 검산 + `np.trace` 는 비정방에서 **조용히** 존재하는 대각 원소만 더함(수학적 대각합 아님) — 1D `.T` 먹통([[T04 NumPy 벡터 내부 — shape·strides·열벡터 (N,1)|T04]])과 같은 계열의 "조용한 함정"

> [!quote]- ✅ 정답·풀이 (펼치기 전에 손으로!)
> **P1.** ⑴ $\sqrt{4+1+0+9}=\sqrt{14}\approx 3.742$. ⑵ $A^{\mathsf{T}}A=\begin{bmatrix}4&2\\2&10\end{bmatrix}$ — 대각 $4=2^2{+}0^2$(**1열**), $10=1^2{+}3^2$(**2열**). ⑶ $tr=4{+}10=14$ → $\sqrt{14}$ ✔️ 같은 값 (덤: §5.1.1 책 예제의 14와 우연히 동일).
> **P2.** ⑴ $R^{\mathsf{T}}R=\begin{bmatrix}35&44\\44&56\end{bmatrix}$, $tr=91$ → $\sqrt{91}\approx 9.539$. ⑵ $1{+}4{+}9{+}16{+}25{+}36=91$ ✔️. ⑶ $R$ 은 $3{\times}2$ **비정방 — 대각합은 정방 전용이라 $tr(R)$ 은 정의되지 않는다.** 그런데도 공식이 성립하는 이유: $R^{\mathsf{T}}R$ 은 크기와 무관하게 **항상 정방**($N{\times}N$).
> **P3.** ⑴ $RR^{\mathsf{T}}$ 대각 = $5{=}1^2{+}2^2$(**1행**), $25{=}3^2{+}4^2$(**2행**), $61{=}5^2{+}6^2$(**3행**). ⑵ $5{+}25{+}61=91=tr(R^{\mathsf{T}}R)$ ✔️. ⑶ **둘 다 "전 원소 제곱합"의 재포장이기 때문** — 열로 나눠 담느냐($R^{\mathsf{T}}R$, $2{\times}2$) 행으로 나눠 담느냐($RR^{\mathsf{T}}$, $3{\times}3$)만 다르고 총합은 같다. 그래서 $\lVert A\rVert_F=\sqrt{tr(A^{\mathsf{T}}A)}=\sqrt{tr(AA^{\mathsf{T}})}$ — 책의 "동일한 **행**에 대한 내적" 문구는 이 중 $AA^{\mathsf{T}}$(또는 $A^{\mathsf{T}}$ 의 행=원래 열) 시점의 서술.
> **P4.** 정의대로면 $\sqrt{1{+}4{+}9{+}16}=\sqrt{30}\approx 5.477 \ne \sqrt{17}\approx 4.123$ — 반박 끝. 혼동의 정체: **$A$ 자신의 대각**을 봤다. 공식의 대각은 $A^{\mathsf{T}}A=\begin{bmatrix}10&14\\14&20\end{bmatrix}$ 의 대각($10{=}1^2{+}3^2$, $20{=}2^2{+}4^2$)이고, 여기엔 $A$ 의 **전 원소**가 열 단위로 재포장돼 있다($10{+}20{=}30$).
> **P5.** $A^{\mathsf{T}}A$ 의 **비대각 = 서로 다른 열끼리의 내적** $N(N{-}1)/2$ 쌍(P4의 예에선 $14$ 두 칸)을 전부 계산해 놓고 대각만 쓰고 버린다. 실전: `np.sqrt((A**2).sum())` (또는 `np.linalg.norm(A,'fro')`).
> **P6.** ⑴~⑷ 전부 $\sqrt{30}\approx 5.4772$ 로 일치. ⑸ **에러 없이 5가 나온다** — `np.trace(R)`=$1{+}4$, "존재하는 대각 원소"만 조용히 더한 값이고 **수학적 대각합이 아니다**(정방 전용). 조용히 틀린 값을 주는 함정이라 비정방엔 쓰지 말 것.
>
> ```python
> import numpy as np   # 검산 (전부 통과 2026-07-06 — la5_norms.py에도 등재)
> A1 = np.array([[2.,1.],[0.,3.]])
> assert np.trace(A1.T@A1) == 14 and np.isclose(np.sqrt(14), np.linalg.norm(A1,'fro'))
> R = np.array([[1.,2.],[3.,4.],[5.,6.]])
> assert np.trace(R.T@R) == np.trace(R@R.T) == 91          # P2·P3: 두 우회로 tr 동일
> assert (R.T@R).shape == (2,2) and (R@R.T).shape == (3,3) # 크기는 다름
> A = np.array([[1.,2.],[3.,4.]])
> four = [np.sqrt((A**2).sum()), np.linalg.norm(A,'fro'),
>         np.linalg.norm(A.ravel(),2), np.sqrt(np.trace(A.T@A))]
> assert np.allclose(four, np.sqrt(30))                    # P6: 4중 일치
> assert not np.isclose(np.sqrt((np.diag(A)**2).sum()), np.sqrt(30))  # P4: √17 반박
> assert np.trace(R) == 5.0                                # P6⑸: 비정방 조용한 함정
> ```

## E. 책 공식 연습 [5-1]·[5-2]·[5-4] — 실험·수렴·이동 (2026-07-14 수신)

**P7 (책 [연습 5-1]).** $10\times10$ 난수 행렬에 스칼라 $0\sim50$(40단계)을 곱하며 프로베니우스 노름을 그린다(10회 평균). ⑴ 그래프 모양을 예측하라(직선? 곡선?) ⑵ 왜 그런 모양인지 식 5-1로 설명 ⑶ 노름이 절대 음수가 될 수 없고 영행렬에서만 0이 되는 이유.
- 시험 개념: $\lVert cA\rVert_F=|c|\cdot\lVert A\rVert_F$(스칼라 비례성) — [[2.5 행렬 노름·대각합 — 프로베니우스·ℓ2·tr(AᵀA)|2.5]]

**P8 (책 [연습 5-2]).** 두 $N\times N$ 난수 행렬 $A,B$ 에 같은 스칼라 $s$(초기 1)를 곱해 프로베니우스 거리 $\lVert sA-sB\rVert_F$ 를 재고, 1 이상이면 $s\leftarrow 0.9s$ 를 반복(while)해 1 미만이 되는 지점을 찾는다. 몇 번 반복될지, $s$ 의 최종값을 결정하는 **닫힌 형태 식**은?
- 시험 개념: $\lVert sA-sB\rVert_F=s\lVert A-B\rVert_F$(선형성) → $s<1/\lVert A-B\rVert_F$ 를 만족하는 최소 $0.9^k$ — 반복 없이도 미리 계산 가능

**P9 (책 [연습 5-4]).** $10\times10$ 난수 행렬을 노름의 $0\sim1$ 비율(30단계)만큼 이동($A+\lambda I$, $\lambda=$비율$\times\lVert A\rVert_F$)시키며 ⑴ 노름 변화율(%) ⑵ 원본과의 (평탄화) 상관계수 ⑶ 원본과의 프로베니우스 거리를 각각 그린다. 세 그래프의 **모양이 서로 다른 이유**(하나는 휘어지고, 하나는 완만히 줄고, 하나는 직선인)를 설명하라.
- 시험 개념: 이동은 **대각에만** 가해지므로($A+\lambda I$) 전체 노름 대비 대각 비중이 커질수록 원본과의 상관은 깨지지만(비선형 감소), 이동량 자체(=거리)는 $\lambda\sqrt{N}$ 으로 **정확히 선형** — [[2.1 행렬 연산 — 덧셈·뺄셈·이동·스칼라곱·아다마르곱|2.1 이동]] · [[2.5 행렬 노름·대각합 — 프로베니우스·ℓ2·tr(AᵀA)|2.5]]

---

> [!quote]- ✅ P7~P9 정답·풀이 (책 공식 연습 5-1·5-2·5-4)
> **P7.** ⑴ **완전한 직선**(원점 통과). ⑵ 식 5-1의 노름은 각 원소에 선형으로 걸리는 제곱합의 제곱근이므로 $\lVert cA\rVert_F=\sqrt{\sum(ca_{ij})^2}=\sqrt{c^2\sum a_{ij}^2}=|c|\lVert A\rVert_F$ — $c\ge0$ 구간에서 완전한 직선. ⑶ 제곱합은 항상 $\ge0$(제곱은 음수 불가)이고, 그 합이 0이 되려면 **모든** 원소가 0이어야 하므로(하나라도 0이 아니면 양수 항 존재) 노름=0 ⇔ 영행렬.
> 재현 결과(2026-07-14, 10×10 난수 10회 평균): 스칼라·노름 상관계수 **1.0000**(완전 선형), 스칼라=0에서 노름=0 정확히 재현. → [[LA5-7 노름-스칼라 선형비례 (재구성).png|LA5-7]] (책 그림 5-7 재현).
>
> **P8.** $\lVert sA-sB\rVert_F=\lVert s(A-B)\rVert_F=s\lVert A-B\rVert_F$(P7의 스칼라 비례성 재사용, $s>0$) — 원 거리를 $d_0=\lVert A-B\rVert_F$ 라 하면 조건은 $s\,d_0<1 \Leftrightarrow s<1/d_0$. $s=0.9^k$ 이므로 $k>\log_{0.9}(1/d_0)=\dfrac{-\ln d_0}{\ln 0.9}$ — **반복문 없이 로그 한 줄로 계산 가능**, while 루프는 이 닫힌 식을 흉내내는 비효율 버전.
> 재현 결과(7×7 난수 예시): $d_0=12.439$ → 필요 $s\approx0.07977$(24회 반복 후 거리 0.992) — 닫힌식 $1/d_0=0.08039$ 와 일치($s\le 1/d_0$, 한 스텝 전 값 $s/0.9$ 는 조건 미달도 확인).
>
> **P9.** 세 그래프가 다른 모양인 이유는 **이동이 대각에만 가해지기 때문**:
> - **거리(직선):** $\lVert(A+\lambda I)-A\rVert_F=\lVert\lambda I\rVert_F=\lambda\sqrt{N}$ — $\lambda$ 에 대해 정확히 선형이므로 그래프는 완전한 직선.
> - **노름 변화율(위로 휨):** 대각 $N$ 칸만 $+\lambda$ 되므로 노름 자체는 $\sqrt{\lVert A\rVert_F^2+2\lambda\,tr(A)+N\lambda^2}$ 꼴(교차항 포함) — $\lambda^2$ 항이 지배하기 시작하면서 변화율(%)이 가속도적으로 커진다(재현: 0%→230%, 30단계).
> - **상관(완만히 감소):** 상관계수는 "원본과 얼마나 같은 패턴이냐"이지 크기가 아니다 — 대각 $N$ 칸만 커지고 비대각 $N^2{-}N$ 칸은 그대로라서, $\lambda$ 가 커질수록 "원본에 대각 신호가 덧씌워진" 형태가 되어 상관이 완만히 감소(재현: 1.0→0.315)하지만 비대각 정보가 남아 있어 0까지 떨어지진 않는다.
> 재현 결과(10×10 난수, 30단계): 거리 0→28.47(예측값 $\lambda\sqrt{10}$ 과 정확히 일치), 노름변화율 0%→230%, 상관 1.0→0.315. → [[LA5-8 이동 노름변화율·상관·거리 (재구성).png|LA5-8]] (책 그림 5-8 재현, 3패널).
>
> ```python
> import numpy as np
> rng = np.random.default_rng(0)
> A = rng.standard_normal((10,10)); norm0 = np.linalg.norm(A,'fro')
> for ratio in np.linspace(0,1,30):
>     lam = ratio*norm0
>     shifted = A + lam*np.eye(10)
>     dist = np.linalg.norm(shifted-A,'fro')
>     assert np.isclose(dist, lam*np.sqrt(10))          # P9: 거리=선형 증명
> ```
