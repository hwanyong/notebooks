# P01 행렬 곱셈 — 유효성·외적·스칼라 행렬 (5문)

> ✏️ **연습 세트.** 정답은 접이식 — 펼치기 전에 손으로 풀 것. 운영 규칙: [[_연습 MOC]]
> 🎯 **이 세트가 시험하는 것:** ① 표준 행렬곱 계산(칸 = 행·열 내적, [[2.2 행렬 곱셈·행렬식|2.2]]) ② **출력 크기 스스로 판정** — 답 그리드가 3×3으로 *일부러 크게* 주어져, 유효성 규칙 $(M{\times}N)(N{\times}K){\to}M{\times}K$ 로 빈칸을 남길 줄 아는지 본다 📌 ③ 특수 케이스 인식(외적·스칼라 행렬).

## 문제

**P1.** $E=\begin{bmatrix}3&5\\-1&1\end{bmatrix}$, $A=\begin{bmatrix}-2&2&3\\3&5&-2\end{bmatrix}$ — $H=EA$ 는?
- 시험 개념: 칸=내적 + 출력 크기 $(2{\times}2)(2{\times}3){\to}2{\times}3$ (3×3 그리드에서 **셋째 행은 비워야 함**)

**P2.** $D=\begin{bmatrix}-1&0\\2&-2\\1&3\end{bmatrix}$, $F=\begin{bmatrix}0&2\\3&-2\end{bmatrix}$ — $H=DF$ 는?
- 시험 개념: $(3{\times}2)(2{\times}2){\to}3{\times}2$ (**셋째 열을 비워야 함**) + 0이 낀 내적

**P3.** $C=\begin{bmatrix}-1\\-2\\2\end{bmatrix}$, $D=\begin{bmatrix}2&1\end{bmatrix}$ — $H=CD$ 는?
- 시험 개념: **열벡터×행벡터 = 외적**($(3{\times}1)(1{\times}2){\to}3{\times}2$, rank-1) — [[04 벡터 내적 — 점곱·기하·직교 + 아다마르곱·외적|04 외적 $vw^{\mathsf{T}}$]]. 내적(스칼라)과 혼동하면 여기서 걸린다.

**P4.** $E=\begin{bmatrix}1&-1&1\\2&3&4\end{bmatrix}$, $A=\begin{bmatrix}2&4\\2&-2\\-1&5\end{bmatrix}$ — $H=EA$ 는?
- 시험 개념: $(2{\times}3)(3{\times}2){\to}2{\times}2$ — 내부 3 소거, 원소 3개짜리 내적(부호 실수 조심)

**P5.** $B=\begin{bmatrix}5&-2\\-1&5\end{bmatrix}$, $A=\begin{bmatrix}2&0\\0&2\end{bmatrix}$ — $H=BA$ 는?
- 시험 개념: $A=2I$ = **스칼라 행렬** — $BA=2B$(전체 2배 스케일링, [[11 행렬 연산 — 덧셈·뺄셈·이동·스칼라곱·아다마르곱|11 스칼라곱]]·[[14 역행렬·연립방정식 — 특수 행렬·AX=B 풀이|14 항등]]). 보너스: $2I$ 는 **교환 가능**($AB=BA$) — 비교환 규칙의 특수 예외.

> [!quote]- ✅ 정답·풀이 (펼치기 전에 손으로!)
> **P1.** $H=\begin{bmatrix}9&31&-1\\5&3&-5\end{bmatrix}$ (2×3, 셋째 행 공백) — ✅ **입력한 답 정확** (2026-07-03). 예: $h_{11}=3{\cdot}({-2}){+}5{\cdot}3=9$, $h_{12}=3{\cdot}2{+}5{\cdot}5=31$.
> **P2.** $H=\begin{bmatrix}0&-2\\-6&8\\9&-4\end{bmatrix}$ (3×2, 셋째 열 공백). 예: $h_{21}=2{\cdot}0{+}({-2}){\cdot}3=-6$.
> **P3.** $H=\begin{bmatrix}-2&-1\\-4&-2\\4&2\end{bmatrix}$ — 외적: 모든 열이 $C$의 배수(rank-1). 각 원소 $h_{ij}=c_i d_j$.
> **P4.** $H=\begin{bmatrix}-1&11\\6&22\end{bmatrix}$. 예: $h_{11}=1{\cdot}2{+}({-1}){\cdot}2{+}1{\cdot}({-1})=-1$, $h_{22}=2{\cdot}4{+}3{\cdot}({-2}){+}4{\cdot}5=22$.
> **P5.** $H=2B=\begin{bmatrix}10&-4\\-2&10\end{bmatrix}$ — 계산 없이 "$2I$ 곱 = 2배"로 즉답 가능해야 한다.
>
> ```python
> import numpy as np   # 검산 (전부 통과 2026-07-03)
> assert np.array_equal(np.array([[3,5],[-1,1]]) @ np.array([[-2,2,3],[3,5,-2]]), [[9,31,-1],[5,3,-5]])
> assert np.array_equal(np.array([[-1,0],[2,-2],[1,3]]) @ np.array([[0,2],[3,-2]]), [[0,-2],[-6,8],[9,-4]])
> assert np.array_equal(np.array([[-1],[-2],[2]]) @ np.array([[2,1]]), [[-2,-1],[-4,-2],[4,2]])
> assert np.array_equal(np.array([[1,-1,1],[2,3,4]]) @ np.array([[2,4],[2,-2],[-1,5]]), [[-1,11],[6,22]])
> B, A = np.array([[5,-2],[-1,5]]), 2*np.eye(2)
> assert np.array_equal(B@A, 2*B) and np.array_equal(B@A, A@B)   # 스칼라 행렬은 교환!
> ```
