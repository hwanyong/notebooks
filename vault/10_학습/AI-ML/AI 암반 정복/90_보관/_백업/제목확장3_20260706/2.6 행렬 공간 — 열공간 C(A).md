# 행렬 공간 — 열공간 C(A)

> 📐 **수학 개념원자(SSoT)** — 선형대수. 행렬 공간(열·행·영)의 단일 주인 — 열공간부터. [[1.7 벡터 확장 — 집합·선형결합·선형독립·부분공간·생성·기저|1.7]] 생성·부분공간의 행렬판.
> 🔗 [[_수학 MOC]] ｜ 🌐 [[_지식 연결 허브]] ｜ 🎓 AI 렌즈: [[12 행렬 공간 — 열공간 C(A)|12 열공간(AI)]] ｜ 토대: [[1.7 벡터 확장 — 집합·선형결합·선형독립·부분공간·생성·기저|생성·부분공간]] · [[2.2 행렬 곱셈·행렬식|행렬-벡터 곱(열 시점)]]

> 🎯 **핵심 (TL;DR)**
> - **행렬 공간 = 행렬의 서로 다른 특징들 사이의 선형 가중 결합** — 어렵지 않다, 결국 [[1.7 벡터 확장 — 집합·선형결합·선형독립·부분공간·생성·기저|1.7]]의 확장.
> - **열공간 $C(A)$ = 행렬을 열벡터 집합으로 보고, 무한한 실수 스칼라로 만든 *모든* 선형 가중 결합** = 열들의 생성(span). $\in$ = '에 포함됨'.
> - **소속 판별이 핵심 질문:** $v\in C(A)$? — 열들의 가중 결합으로 $v$를 만들 수 있나. 이 질문이 **선형 최소제곱법의 시작** `(🔭 예고 — 10장)`.
> - 📌 **N개 열 ≠ N차원 열공간** — 차원 = **선형 독립인 열의 수**(1.7 그대로). 종속 2열이면 여전히 1D 선.
> - 열공간은 전체 $\mathbb{R}^M$ 이 되거나 그 안의 더 작은 부분공간(선·평면 — "우주를 가로지르는 얇은 종이").
> - 정량 판별 도구 = **행렬 계수(rank)** `(🔭 예고 — 장 후반)`.

---

> [!question] 🎯 필요성 사슬 (결핍 → 필요 → 발명)
> *출발 문제:* $Av$ 가 만들 수 있는 벡터는 어디까지인가 — 방정식 $Ax=b$ 에 해가 *있기는 한가*?
> 1. 행렬의 열들이 "도달할 수 있는 범위"를 부를 이름이 없음 → 열들의 모든 선형 가중 결합 → **열공간 $C(A)$** (1.7 생성에 행렬 표기를 입힘)
> 2. 어떤 벡터가 그 범위 안인지 밖인지 판별해야 함($Ax=b$ 해 존재 = $b\in C(A)$) → **소속 판별** — 추측·산술·시각화로는 고차원 불가
> 3. 열이 많다고 공간이 넓어지지 않음(종속 열 = 재탕) → 차원을 세는 잣대 필요 → **독립 열 수 = 차원** → 정량화 = **rank** `(🔭 예고)`
> **한 줄:** 열들의 도달 범위(열공간) → 이 벡터가 그 안인가(소속 판별 = 최소제곱의 시작) → 범위의 실제 크기는 독립 열 수(rank).

> [!abstract] 📘 책 핵심 — [실전선대] §5.2 행렬 공간 · §5.2.1 열공간
> **① §5.2 — 행렬 공간(열, 행, 영)**
> - 전통·응용 선대의 핵심 개념이지만 어렵지 않음 — **근본적으로 행렬의 서로 다른 특징들 사이의 선형 가중 결합**.
>
> **② §5.2.1 — 열공간**
> - 벡터의 선형 가중 결합에서 **두 가지 수정**: ⑴ 행렬을 **열벡터의 집합**으로 간주 ⑵ 특정 스칼라 집합 대신 **무한한 실수 스칼라** 적용 → 생성된 무한 벡터 집합 = **행렬의 열공간**. 표기 $C(A)$, $\in$='의 구성원/에 포함됨', λ=가능한 모든 실숫값.
> - **예 1 (열 1개):** $C\!\left(\begin{bmatrix}1\\3\end{bmatrix}\right)=\lambda\begin{bmatrix}1\\3\end{bmatrix}$ — 열벡터 방향으로 원점 통과 양방향 무한선 = **1차원 부분공간**(그림 5-1). *(각주 감각: 엄밀히 '무한대는 실수가 아니라' 문자 그대로 무한 확장은 아니지만, 어떤 목적에든 충분히 — 인간 지식으로 헤아릴 수 있는 것보다 — 길다.)*
>   - 소속 판별: $[1,3]^{\mathsf{T}}\in C$? 예(λ=1). $[-2,-6]^{\mathsf{T}}$? 예(λ=−2). $[1,4]^{\mathsf{T}}$? **아니오** — 만들 스칼라가 없다.
>   - ⚠️ **λ 신분 + 모양 혼동 주의:** 여기의 λ는 특정 값이 아니라 **전 실수를 훑는 자유 변수**(집합 생성 매개변수) — 위 식은 방정식이 아니라 집합 $\{\lambda[1,3]^{\mathsf{T}} \mid \lambda\in\mathbb{R}\}$ 의 압축 표기다. 생김새가 거의 같은 고윳값 식 $Av=\lambda v$(λ=**행렬이 정해주는 특정 상수** `🔭`)와 신분이 다름: 전자="v의 배수 **전부**", 후자="A가 v를 몇 배로 늘리나". λ 신분 지도 → [[1.3 벡터 연산 — 덧셈·뺄셈·스케일링·전치·브로드캐스팅|1.3]].
> - **예 2 (독립 2열):** $C\!\left(\begin{bmatrix}1&1\\3&2\end{bmatrix}\right)=\lambda_1[1,3]^{\mathsf{T}}+\lambda_2[1,2]^{\mathsf{T}}$ — 답: **$\mathbb{R}^2$ 전체**. 예: $[-4,3]^{\mathsf{T}}$ = 두 열을 11과 −15로 가중(스칼라 찾기 = 최소제곱법 `(🔭 예고 — 10장)`). 그림 5-2A.
> - **예 3 (종속 2열):** $C\!\left(\begin{bmatrix}1&2\\3&6\end{bmatrix}\right)$ — 두 열이 같은 직선상(그림 5-2B) → 열공간 = **여전히 선(1차원)**. 📌 **N개 열이라고 무조건 N차원이 아니다** — 열들이 **선형 독립 집합일 때만** 열 수 = 차원(2장 연결).
> - **예 4 (R³ 속 독립 2열):** $C\!\left(\begin{bmatrix}3&0\\5&2\\1&2\end{bmatrix}\right)$ — 두 열 독립 → 열공간은 2차원이지만 **$\mathbb{R}^3$ 에 포함된 2D 평면**(그림 5-3) — "무한하지만 3차원에선 지극히 작은 조각, **우주를 무한히 가로지르는 얇은 종이**".
> - 평면 안에도 많은 벡터가 있지만 **밖에 더 많다** — 열공간 안 vs 밖.
> - **벡터가 열공간에 있는지 어떻게 아나? = 선형 최소제곱법의 시작** (응용 수학·공학에서 중요성은 아무리 강조해도 지나치지 않음). 지금까지는 추측·산술·시각화 — 직관용일 뿐 고차원·복잡 문제는 불가 → **정량적 방법 = 행렬 계수(rank)** `(🔭 예고 — 장 후반)`.
> - 그 전 직관: 행렬의 열이 벡터 부분공간을 구성 — 전체 M차원 공간이 되거나 더 작은 부분공간. **중요한 것 = 다른 벡터가 그 부분공간 안에 있는가**(열들의 선형 가중 결합으로 표현 가능한가).

### 🔑 $Ax=b$ 재해석 — "해가 있다 ⇔ $b\in C(A)$"
- [[2.2 행렬 곱셈·행렬식|2.2]] 열 시점: $Ax$ = A의 열들을 $x$ 계수로 결합 → **$Ax$ 가 도달 가능한 모든 결과 = 정확히 $C(A)$**. 따라서 연립방정식 $Ax=b$ 에 해가 존재한다는 말 = **$b$ 가 열공간 안에 있다**는 말의 동의어.
- [[2.4 역행렬·연립방정식 — 특수 행렬·AX=B 풀이|2.4]]의 column picture("b를 만드는 A열 결합 계수")가 이 언어로 완성된다. $b\notin C(A)$ 면 정확한 해가 없고 — 그때 "가장 가까운 열공간 안 점"을 찾는 것이 **최소제곱** `(🔭 10장)`.

### 🔑 소속 판별의 도구 사다리 — 눈→손→rank
- 책의 진행이 정직하다: 예제들은 추측(스칼라 있나)·산술(방정식 풀기)·시각화(그림)로 판별했지만, **셋 다 고차원에서 죽는다**. 그래서 정량 도구(rank)가 *필요해진다* — 다음 절의 필요성 사슬이 이 절에서 이미 만들어졌다.

## 🧩 코드 (§5.2.1)
> 🎯 목표: 소속 판별 3례 재현 + "종속 2열=1D" 확인.
> 코드: `~/LOCAL/03-00_STUDIES/AI/01_linear_algebra/la5_spaces.py`

```python
import numpy as np
A = np.array([[1.], [3.]])                     # 열 1개
# 소속 판별: v = λ·col 이 성립하는 λ가 있나
for v, expect in [([1,3], True), ([-2,-6], True), ([1,4], False)]:
    v = np.array(v, float)
    lam = v[0] / A[0, 0]                       # 첫 성분으로 λ 후보
    print(v, "∈ C(A)?", np.allclose(A.ravel()*lam, v), "(기대:", expect, ")")
# 독립 2열 → R² 전체: [-4,3] = 11·[1,3] − 15·[1,2]  (책 수치 검증)
c1, c2 = np.array([1,3.]), np.array([1,2.])
print(11*c1 - 15*c2)                           # [-4. 3.] ✓
# 종속 2열 → 여전히 1D (rank 예고)
D = np.array([[1,2],[3,6.]])
print("독립 열 수(rank):", np.linalg.matrix_rank(D))   # 1
```
