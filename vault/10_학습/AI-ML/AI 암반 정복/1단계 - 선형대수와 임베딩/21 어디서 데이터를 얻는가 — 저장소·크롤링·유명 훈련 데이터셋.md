# 어디서 데이터를 얻는가 — 저장소·크롤링·유명 훈련 데이터셋

> 📊 [[18 인공지능 유형과 데이터 — 인지 AI·복잡성 4종·정형 비정형|18]]의 "ground truth 부재·데이터가 연료" 문제의 실전편 — **실데이터는 얻기 힘들다**는 저자의 경험담 + 공개 저장소·크롤링·유명 AI들의 훈련 데이터셋 목록.

> 🗺️ [[_1단계 - 선형대수와 임베딩|1단계 허브]] ｜ ⬅️ 이전: [[20 수학 모델 — 선형·비선형·시뮬레이션과 AI 3중 관계|20 수학 모델]] ｜ ➡️ 다음: [[22 확률·통계 용어 지도 — 확률 변수·분포·기대값·공분산·마르코프|22 용어 지도]]

> 🎯 **핵심 (TL;DR)**
> - **실데이터 얻기는 매우 힘들다** 📌 — 저자(전쟁 폐허국 출신 수학 박사)가 기업·정부·군에 무료 해결까지 제안하며 배운 교훈: 규제·개인 정보·기관 심의 위원회 장애물 + 통과해도 비공개 경향(자기 데이터를 최대 활용 못 하면서도). "거의 구걸" — 이 분야의 보편 경험이니 **거절에 낙담 말고 계속 요청하라**(누군가는 믿음의 도약을 한다).
> - 공개 경로: **캐글**, '최고의 데이터 저장소' 검색, 특화 저장소(비전·NLP·오디오·과학) — 옮긴이 추천: 구글 데이터셋 서치·UCI ML 저장소·데이터 월드.
> - **웹 크롤링**: 사이트 규칙 준수(일부 서면 허가 필요)·방법을 배워야 — "데이터 과학자는 해킹할 줄 안다"가 통계학자와의 차이라는 말까지. SNS 크롤링으로 수십만 기록 데이터셋 가능.
> - **성공한 AI의 훈련 데이터셋을 둘러보는 습관** 📌(다운로드 불필요 — 메타데이터·피처·레이블 확인만으로 익숙해짐): WaveNet(딥마인드, 음성·음악 생성) = CSTR Voice Cloning Toolkit(109명 화자 44시간)+영어 24h·만다린 34.8h+피아노 솔로 60h·MagnaTagATune(29초 클립·188개 태그 약 200h) / AlexNet(2012) = **ImageNet**(수백만 이미지·수천 클래스, **크라우드소싱** 레이블) — 전부 **비정형 데이터**.
> - **레이블 데이터의 가치 = 가설 함수의 결과를 측정할 기준 제공** 📌([[18 인공지능 유형과 데이터 — 인지 AI·복잡성 4종·정형 비정형|18]] ground truth의 재등장). 훈련 데이터 비공개면 논문·문서에서 획득 방법을 읽는 것만으로도 많이 배움.
> - WaveNet의 "특정 인물 목소리로 변환" = 신경망의 **조건화** — 지금은 "문제에 인위적 제한을 가해 결과를 특정 집합으로 제한"으로 이해 `(🔭 예고 — MA 7장 NLP·금융 AI에서 수학적 의미)`.
> - 기억할 3가지 📌: ① AI엔 디지털 데이터가 필요 ② 데이터는 얻기 어려울 수 있음 ③ 전 세계 디지털화 진행 중.

---

> [!question] 🎯 왜 이걸 배우나 — 필요성 사슬 (결핍 → 필요 → 발명)
> *출발 문제:* [[19 실제 vs 시뮬레이션 데이터 — 키-체중·조건부 층화·노이즈|19]]까지 "데이터가 있다면"을 전제했다 — 정작 **그 데이터를 어디서 구하나**가 비어 있다.
> 1. 실데이터 소유 기관은 규제·프라이버시·심의로 잠겨 있음(결핍) → 끈질긴 요청 + **공개 저장소**(캐글·UCI 등)가 우회로.
> 2. 저장소에 없는 데이터는? → **크롤링**(규칙 준수 전제) — 웹 자체를 데이터원으로.
> 3. 내 문제에 뭘 모아야 할지 감이 없다면? → **성공 사례의 훈련 데이터셋 역추적**(WaveNet·ImageNet) — 메타데이터·레이블 구조만 봐도 "AI에 데이터가 어떻게 들어가는가"의 직관이 생김.
> **한 줄:** 잠긴 실데이터(결핍) → 저장소·크롤링(우회) → 성공 사례의 데이터 역추적(감각) — 레이블이 곧 측정 기준.

> [!abstract] 📘 책 핵심 — [필수수학AI] §2.7 어디서 데이터를 얻는가
> - **저자의 경험**: 전쟁으로 폐허가 된 나라에서 성장(공급망 중단·전력망 파괴·교량 폭격·테러·밀수·인신매매·인플레이션·빈곤 — 수학이 풀 수 있는 문제 무궁무진) → 미국에서 수학 박사·대학 재직 시절 기업·정부·군에 실데이터 프로젝트를 찾아다니며 **무료 해결까지 제안** → 어렵게 배운 사실: **실데이터 얻기가 매우 힘들다**. 수많은 규제·개인 정보 문제·기관 심의 위원회, 그 후에도 비공개 경향(가진 데이터를 최대 활용 못 하면서도). 거의 구걸 — 이 분야 많은 사람의 공통 경험.
> - 낙담시키려는 얘기가 아님 — **데이터 소유처가 주저·거절해도 놀라거나 낙담하지 말 것. 계속 요청하면 누군가는 믿음의 도약을 한다.** 📌
> - **웹 공개 데이터**: 캐글(각주 26) 등. '최고의 데이터 저장소' 키워드 검색; 비전·NLP·오디오 생성·과학 연구 특화 저장소 존재. 옮긴이(27): 구글 데이터셋 서치·UCI 머신러닝 저장소·데이터 월드.
> - **웹 크롤링**: 대상 사이트 규칙 준수(일부는 사전 서면 허가), 방법 학습 필요. "데이터 과학자는 해킹하는 방법을 안다"를 통계학자와의 차이로 꼽는 사람들도. SNS(페이스북·인스타그램·유튜브·플리커·링크드인) 크롤링으로 사용자 통계(친구 수·좋아요·댓글·활동) 수집 → 수십만 기록의 대형 데이터셋.
> - **성공적 AI가 학습한 데이터셋을 살펴보는 습관** — 다운로드 없이 메타데이터·피처·레이블만 확인해도 데이터에 익숙해짐:
>   - **WaveNet**(딥마인드, `🔭 MA 7장`): 실감 나는 목소리·음악 생성 신경망(텍스트→오디오·특정 인물 목소리 변환). 자연스러운 변환의 비결 = **신경망이 특정 인물의 목소리를 조건화** — 조건화의 수학적 의미는 7장에서; 지금은 *문제에 인위적 제한을 가해 결과를 특정 집합으로 제한*으로 이해. 훈련 데이터: (텍스트 비조건) 다중 화자 오디오 생성용 **CSTR Voice Cloning Toolkit**(109명 화자·44시간), 텍스트→음성용 북미 영어 24시간·중국어 만다린 34.8시간, 음악 생성용 유튜브 피아노 솔로 60시간·**MagnaTagATune** 약 200시간(29초 클립, 장르·악기 편성·템포·볼륨·분위기 등 **188개 태그 레이블**).
>   - **레이블 지정 데이터가 매우 유용한 이유 = 가설 함수의 결과를 측정할 기준 제공**(이어지는 절들에서).
>   - **AlexNet**(2012, 이미지 분류/컴퓨터 비전): **ImageNet**으로 훈련 — 인터넷 스크랩 수백만 이미지·수천 클래스, 레이블은 **크라우드소싱(crowdsourcing)** 으로 사람이 직접 지정.
>   - 지금까지의 예시 전부 **비정형 데이터**.
> - 훈련 데이터가 비공개면 해당 AI의 **논문·문서에서 데이터 획득 방법을 읽기** — 그것만으로 많이 배움.
> - 본격 수학 전 기억할 핵심: **AI 시스템에는 디지털 데이터가 필요하다 · 때로는 필요한 데이터를 얻기 어렵다 · 전 세계를 디지털화하려는 움직임이 곳곳에서 일어나고 있다.**

## 🧠 근본 이해 · 융합 · 통찰

### 🔑 데이터셋 역추적 = ground truth 문제의 실전 해법 카탈로그
- [[18 인공지능 유형과 데이터 — 인지 AI·복잡성 4종·정형 비정형|18]]의 복잡성 ②(성능 — ground truth 부재)가 여기서 실물로: MagnaTagATune의 **188개 태그**, ImageNet의 **크라우드소싱 레이블** — 유명 AI의 뒤에는 예외 없이 *레이블 확보의 공학*이 있다. "레이블 = 가설 함수 측정 기준"이라는 문장이 두 노트를 잇는 못.
- 크라우드소싱은 [[1.1 확률 — 정의·가능도·오즈|MD 1.1]]의 통계(데이터 없이 존재 불가)를 사람의 노동으로 사는 행위 — 데이터의 "이론 대 데이터" 이분법에서 데이터 쪽 비용이 구체화된다.

### 🔑 WaveNet의 '조건화' 예고 = MD 1.3 조건부의 세 번째 재등장
- "특정 인물의 목소리를 **조건화**" — [[1.3 조건부 확률·베이즈 정리 — 방향·뒤집기·조건부 결합·합|MD 1.3]]의 조건부($P(\cdot \mid \text{화자})$), [[19 실제 vs 시뮬레이션 데이터 — 키-체중·조건부 층화·노이즈|19]]의 Index 층화에 이은 세 번째 등장이다. 책의 임시 정의("결과를 특정 집합으로 제한")는 조건부 확률의 "분모 교체 = 모집단 정밀화" 그대로. 산점도 층화(19) → 생성 모델 조건화(21) → 수학적 의미(MA 7장 `🔭`)로 이어지는 한 가닥.

## 막힌 점 · 추후 보강

## 🆕 용어
| 용어 | 한 줄 정의 |
|---|---|
| **웹 크롤링** (web crawling) | 웹 페이지를 자동 수집해 데이터셋을 만드는 기법 — **사이트 규칙 준수**(일부 서면 허가) 전제 |
| **크라우드소싱** (crowdsourcing) | 다수의 사람에게 작업(레이블링)을 분산 — ImageNet 레이블의 제작 방식 |
| **조건화** (conditioning, 🔭 MA 7장) | 문제에 인위적 제한을 가해 결과를 특정 집합으로 제한(임시 정의) — WaveNet의 화자 지정; 수학적 실체는 조건부 확률([[1.3 조건부 확률·베이즈 정리 — 방향·뒤집기·조건부 결합·합|MD 1.3]]) |
| **ImageNet / MagnaTagATune / CSTR VCTK** | 유명 훈련 데이터셋 — 비전(수백만 이미지·크라우드소싱) / 음악(29초 클립·188 태그) / 다중 화자 음성(109명·44h) |
