# 수학 모델 — 선형·비선형·시뮬레이션과 AI 3중 관계

> 📐 "수학 모델:"이라는 제목을 단 두 절(§2.3·§2.6)의 통합. **선형 = 세상의 평평함**을 모델링하는 함수 클래스의 정식 정의 + 수학 모델·시뮬레이션·AI가 서로를 키우는 3중 관계.

> 🗺️ [[_1단계 - 선형대수와 임베딩|1단계 허브]] ｜ ⬅️ 이전: [[19 실제 vs 시뮬레이션 데이터 — 키-체중·조건부 층화·노이즈|19 실제 vs 시뮬 데이터]] ｜ ➡️ 다음: [[21 어디서 데이터를 얻는가 — 저장소·크롤링·유명 훈련 데이터셋|21 데이터 획득]]

> 🎯 **핵심 (TL;DR)**
> - **선형 종속성(linear dependency) = 세상의 평평함** 📌 — 1차원 직선(line)·2차원 평면(plane)·고차원 초평면(hyperplane([ˈhaɪpərpleɪn], 하이퍼플레인, 초평면)). 선형 함수의 그래프는 한없이 평평하고 절대 구부러지지 않는다. 평평하지 않은 모든 것 = 비선형.
> - ⚠️ **용어 충돌 주의**: 여기의 "선형 종속성"(출력이 피처에 선형으로 의존)은 [[1.8 벡터 확장 — 집합·선형결합·선형독립·부분공간·생성·기저|선형대수의 선형종속]](벡터들이 서로 중복)과 **완전히 다른 개념** — 용어집 ⚠️ 표 참조.
> - **선형 함수의 문법** 📌: 피처는 **그 자체로만**(거듭제곱·루트 금지, 분모·sin·cos·exp·log 속 금지), **스칼라를 곱하고 더하고 빼는 것만** 허용 — $f(x_1,x_2,x_3) = w_0 + w_1 x_1 + w_2 x_2 + w_3 x_3$. 가중치(weight) $w_i$ = 스칼라 파라미터, $w_0$ = **편향 항(bias term)**. 결과 = 피처 간 선형 상호 작용 + 편향.
> - **비선형 함수의 문법**: 피처가 1 아닌 거듭제곱·서로 곱하거나 나눔·sin/cos/exp/log 안에 들어감 — 예: $w_1\sqrt{x_1} + w_2\frac{x_2}{x_3}$, $w_1 x_1^2 + \cdots$, $w_1 e^{x_1} + \cdots$. **신경망의 성공 = 피처 사이의 비선형 관계 포착 능력** 📌.
> - **AI ↔ 수학 모델·시뮬레이션의 3중 관계** 📌: ① 수학 모델·시뮬은 AI가 **학습할 데이터를 생성**(자율 주행 위험 상황·화성 탐사·신약·일기 예보) ② AI는 **기존 수학 모델·시뮬을 향상**(파라미터/분포 탐색·메시(mesh) 학습·더 큰 영역 확장) ③ **AI 자체가 하나의 거대한 수학 모델이자 시뮬레이션**(인간지능 개입 작업의 시뮬레이션 — 기계가 하는 일은 전부 '계산').
> - 모델 설계 = **현실 근접 ↔ 간결함·접근성의 절충** 📌. 일반화(학습한 능력을 새 상황에 적용) = 일반·좁은 AI 공통 목표.

> [!question] 🎯 왜 이걸 배우나 — 필요성 사슬 (결핍 → 필요 → 발명)
> *출발 문제:* [[19 실제 vs 시뮬레이션 데이터 — 키-체중·조건부 층화·노이즈|19]]에서 "선형 관계"를 눈으로 봤고 $w_0 + w_1 \times height$ 를 썼다 — 그런데 "선형"의 **정식 자격**과, 왜 이 어휘로 세상을 모델링하는지가 없다.
> 1. 직선처럼 보이는 패턴을 함수로 쓰려면 → "선형"의 문법 필요 → **피처 그 자체 × 스칼라 + 덧셈만** = 선형 함수(직선·평면·초평면).
> 2. 세상 대부분은 평평하지 않음(구부러진 곡선·곡면) → **비선형 함수** — 무한한 가능성, 신경망 성공의 원천.
> 3. 모델을 어디에 쓰나 → AI와의 3중 관계: 시뮬이 AI의 훈련장(데이터 생성), AI가 시뮬의 보강재, **AI 자체가 시뮬레이션** — 모델링이 커리큘럼의 목적지임을 좌표로 박는다.
> **한 줄:** 평평함의 문법(선형) → 구부러짐의 어휘(비선형, 신경망) → 모델·시뮬·AI가 서로를 키우는 삼각 순환.

> [!abstract] 📘 책 핵심 — [필수수학AI] §2.3 수학 모델: 선형과 비선형
> - **선형 종속성은 세상의 평평함을 모델링** — 1차원 직선(line), 2차원 평면(plane), 고차원 **초평면(hyperplane)**. 선형 함수의 그래프는 한없이 평평하고 절대 구부러지지 않음. 테이블·막대·천장 같은 직선·평평한 면의 대표 함수 = 선형. **평평하지 않은 것 = 전부 비선형** — 구부러진 곡선·곡면의 데이터 포인트는 비선형 함수가 생성.
> - **선형 함수 수식의 자격**: 피처는 거듭제곱·루트 없이 **그 자체로만** 등장, 분모·사인(sine)·코사인(cosine)·지수(exponential)·로그(logarithmic) 등 다른 함수에 포함되지 않음. 피처엔 **오로지 스칼라**(벡터·행렬이 아닌 실수/복소수)를 **곱하고, 서로 더하거나 뺄 수만** 있음:
> $$f(x_1, x_2, x_3) = w_0 + w_1 x_1 + w_2 x_2 + w_3 x_3$$
> - 파라미터 또는 **가중치(weight)** $w_1, w_2, w_3$ = 스칼라, 피처와 선형으로 결합. **편향 항(bias term)** $w_0$ 을 더해 결과 생성 — 즉 결괏값 = **피처 간 선형 상호 작용 + 편향**.
> - **비선형 함수**: 하나 이상의 피처가 1이 아닌 거듭제곱, 다른 피처에 곱하거나 나눠지거나, sin·cos·지수·로그 등에 포함. 예 3종:
> $$f = w_0 + w_1\sqrt{x_1} + w_2\tfrac{x_2}{x_3}, \qquad f = w_0 + w_1 x_1^2 + w_2 x_2^2 + w_3 x_3^2, \qquad f = w_1 e^{x_1} + w_2 e^{x_2} + w_3\cos(x_3)$$
> - 비선형 함수는 무한히 다양 — 세상의 다양한 현상 모델링 가능. **실제로 신경망은 피처 사이의 비선형적 관계를 포착하는 능력 덕분에 성공**. 앞으로 선형 조합(linear combination)·가중치·피처·선형/비선형 관계 용어를 계속 사용.

> [!abstract] 📘 책 핵심 — [필수수학AI] §2.6 수학 모델: 시뮬레이션과 인공지능 (수신분)
> - 모델 설계자는 무엇을 넣을지 결정해 모델을 더 현실적으로 만들 수 있음 — 자연을 더 모방할수록 수학이 더 들어감. **현실 근접 ↔ 간결함·접근성의 절충점** 필요. 설계자마다 다른 모델, 어떤 모델은 특정 현상을 더 잘 포착 — 계산 능력의 비약적 발전 덕에 더 복잡·사실적인 모델을 만들고 테스트 가능.
> - 자연은 아원자 양자 영역(subatomic quantum realm)부터 은하계 규모까지 — 수학 모델은 숫자·함수·방정식만으로 자연의 세부를 설명하고 **불확실성 앞에서 확률로 정량화된 무작위성을 묘사**하는 탁월한 방법. 시뮬레이션으로 조사·시각화 → 인사이트가 모델 개선 + 깊은 수학적 통찰 → 긍정적 피드백 사이클.
> - 기술의 핵심이 수학이라는 것, 구체적으로 **0과 1의 연산·전송뿐**이라는 것이 인상적 — 숫자 이해의 일반화 = **지식의 일반화**, 인간지능의 특징. 일반 AI(인간 유사·슈퍼)와 좁은 AI(특정 작업)의 공통 목표 = **일반화**(학습한 능력을 새 상황에 적용). 좁은 AI 원리(데이터 학습 후 새 데이터에 양질의 예측) = Chapter 3 `(🔭 예고)`.
> - **AI와 수학 모델·시뮬레이션의 세 가지 상호 작용**:
>   ① **수학 모델·시뮬 → AI 학습 데이터 생성**: 자율 주행(절벽·보행자 충돌 등 *피해야 하는* 상황은 가상으로만 훈련 가능 — 도로에 내보내기 전 모든 위험 상황을 가상으로 훈련; 옮긴이 24: 학습=훈련), 화성 탐사·신약 개발·재료 설계·일기 예보·항공·군사 훈련.
>   ② **AI → 기존 수학 모델·시뮬 향상**: 적절한 파라미터 값·확률 분포 탐색, 방정식 이산화 시 **메시(mesh)** 모양·크기 학습(각주 25: 세밀한 메시가 다양한 공간·시간 스케일의 정보·민감한 행동 포착), 계산법을 더 긴 시간·복잡한 영역으로 확장. 항해·항공·금융·재료·유체역학·분자·핵·대기·해양·천체물리·사이버 보안이 수학 모델·시뮬에 크게 의존 — AI 통합이 긍정적 결과 `(🔭 예고 — 책 후반부 사례)`.
>   ③ **AI 자체 = 수학 모델이자 시뮬레이션**: AI의 큰 목표는 인간지능 복제 — 성공적 ML 시스템(구조 변형 신경망 포함)은 시각·패턴 인식·일반화·자연어·논리 추론 등 **인간지능이 개입되는 작업을 시뮬레이션하는 수학 모델**. 이해력·감정·공감·협업도 그 영역 — 범용 AI엔 이들의 복제 방법도 필요(연구 진행 중). **모든 영역에서 기계가 하는 일 = '계산'**: 문서 의미 계산(NLP)·픽셀 결합 계산(비전)·오디오→숫자 벡터 변환·새 오디오 계산. 소프트웨어 AI = 하나의 거대한 수학 모델이자 시뮬레이션.

## 🧠 근본 이해 · 융합 · 통찰

### 🔑 선형 함수 = 0단계 퍼셉트론 = 1.8 선형결합 — 세 노트가 한 식이다
- $f = w_0 + w_1 x_1 + \cdots + w_n x_n$ 은 (a) [[05 퍼셉트론·편향·은닉층|0단계 퍼셉트론]]의 $y = wx + b$ (활성화 전 선형부, $w_0 = b$!) (b) [[1.8 벡터 확장 — 집합·선형결합·선형독립·부분공간·생성·기저|원자 1.8]] 선형결합 $\sum \lambda_i v_i$ 의 "피처 버전" (c) 내적 $\mathbf{w} \cdot \mathbf{x} + w_0$ ([[1.6 벡터 내적 — 점곱·기하·직교·아다마르곱·외적|원자 1.6]]) — **전부 같은 수학**이다. 책의 "스칼라만 곱하고 더한다"는 문법이 곧 선형결합의 정의.
- "구부러지지 않는다"의 대가: 선형만으론 곡면 세상을 못 담음 → 신경망 = **선형(행렬곱) + 비선형(활성화)의 교대 적층** — 비선형 성공담의 구체적 형태 `(🔭 예고 — [필수수학AI] 4장)`.

### 🔑 ⚠️ "선형 종속성" 용어 충돌 — 같은 한국어, 다른 두 개념
- **여기(§2.3)**: linear *dependency* = 출력이 피처에 **선형으로 의존**한다(함수의 성질). / **[[1.8 벡터 확장 — 집합·선형결합·선형독립·부분공간·생성·기저|1.8]]·[[2.8 행렬식 — 계산·선형종속성·특성다항식|2.8]]**: linear *dependence* = 벡터들이 서로 **중복된 정보**를 가진다(벡터 집합의 성질, det=0). 한국어 번역이 둘 다 "선형 종속"이라 충돌 — 문맥이 "출력↔피처"면 전자, "벡터들끼리"면 후자. 용어집 ⚠️ 표에 등재.

### 🔑 3중 관계의 비대칭 — ①은 데이터, ②는 파라미터, ③은 존재론
- ①(시뮬→AI)은 [[19 실제 vs 시뮬레이션 데이터 — 키-체중·조건부 층화·노이즈|19]]의 "함수를 알면 데이터를 만든다"의 응용(위험한 ground truth를 공짜로) — [[18 인공지능 유형과 데이터 — 인지 AI·복잡성 4종·정형 비정형|18]] ground truth 부재 문제의 한 처방. ②(AI→시뮬)는 "적합(fit)" 능력을 전통 수치해석의 파라미터·메시에 돌려쓰는 것. ③(AI=시뮬)은 범주 자체의 선언 — "기계가 하는 일은 전부 계산"은 [[T06 벡터=상태 vs 행렬=연산자 (기저 재정의·SIMD·GEMM)|T06]]·GPU 하드웨어 관점과 같은 결론을 철학 쪽에서 도달한 것.
- 이 커리큘럼의 위치 감각: 우리가 배우는 수학(선형대수·확률·최적화)은 ③의 부품 목록이다 — "AI를 이해한다 = 이 거대한 수학 모델의 부품을 안다".

| **선형 종속성** (linear dependency, 모델) | 출력이 피처에 선형으로 의존 — 직선·평면·초평면의 평평함; ⚠️ 벡터의 선형종속(1.8)과 다른 개념 |
| **초평면** (hyperplane) | 고차원의 "평면" — 선형 함수 그래프의 일반형(1D 직선→2D 평면→nD 초평면) |
| **가중치** (weight, $w_i$) | 피처에 곱하는 스칼라 파라미터 — ML 학습의 대상(0단계 가중치와 동일 개념) |
| **편향 항** (bias term, $w_0$) | 피처 없이 더해지는 상수 — 퍼셉트론의 $b$ 와 동일(기준점 오프셋) |
| **선형 함수의 문법** | 피처 그 자체 × 스칼라 + 덧셈/뺄셈만 — 거듭제곱·루트·분모·sin/cos/exp/log 금지 |
| **비선형 함수** | 위 문법 위반 전부(제곱·피처 간 곱/나눗셈·초월함수 포함) — 신경망 성공의 원천 |
| **메시** (mesh, 🔭) | 방정식 이산화의 격자 — 세밀할수록 다양한 스케일의 정보 포착(각주 25); AI가 모양·크기를 학습 가능 |
| **일반화** (generalization) | 학습한 능력을 새로운 상황에 적용 — 일반·좁은 AI 공통 목표; "지식의 일반화"는 인간지능의 특징 |
