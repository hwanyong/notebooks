# 확률·통계 용어 지도 — 확률 변수·분포·기대값·공분산·마르코프

> 🎯 **핵심 (TL;DR)**
> - **어휘 먼저 전략** 📌: 새 분야 입문 = 새 언어 학습('봉주르'의 뜻을 몰라도 쓰임새부터) — 분야마다 **같은 개념에 다른 용어**를 써서 생기는 혼란을 막는 게 목적. 정의는 등장할 때마다.
> - **확률 이론의 목표 = 랜덤한 숫자·확률적 사건을 결정론적 명제로 만드는 것** 📌 (인간은 불확실성을 싫어함).
> - **확률 변수 = 결과가 무작위인 "함수의 사촌"** 📌 — $x^2$에 3을 넣으면 9가 확실(결정론)이지만, 확률 변수는 보기 전까지 결과를 모름 → 목표를 확실성에서 **가능성의 정량화**로 교체(주사위 4는 1/6, 그러나 결과는 예측 불가 — 가능하다면 카지노 폐업). 이산/연속 집합에서 결과 반환.
> - **분포 어휘**: 이산→**pmf**·연속→**pdf**([[1.4 이항 분포·베타 분포 — pmf·CDF·확률의 확률|1.4]] 예고의 명명 확정) · 확률 변수 여러 개 = **결합 확률 분포**에서 샘플링(전체를 아는 경우 드묾 — 알면 매우 강력) · **주변 확률 분포** = 결합의 가장자리, $p(x) = \sum_y p(x,y)$ (덧셈 규칙, → [[1.2 확률 계산 — 주변·결합·합·덧셈 법칙|1.2]]).
> - **조건부 계열은 전부 MD 원자와 접속** 📌: 조건부·베이즈·**연쇄 규칙**(→ [[1.3 조건부 확률·베이즈 정리 — 방향·뒤집기·조건부 결합·합|1.3]]) · [NOTE] **결합 분포 전체를 알면 베이즈 규칙 불필요**(모를 때의 도구) — 1.3의 "베이즈=결합 대칭성 재배열" 통찰을 책이 확인.
> - **사전 분포·사후 분포·가능도 함수** 📌: 데이터 관찰 *전* 가중치의 분포(사전) / 관찰 데이터가 주어졌을 때 가중치의 분포(사후) / 특정 가중치가 주어졌을 때 데이터를 관찰할 확률을 인코딩하는 **함수**(가능도) — [NOTE] 가능도가 분포가 아니라 **함수**인 이유: **합(적분)이 1이 될 필요가 없어서** → [[1.1 확률 — 정의·가능도·오즈|1.1]] 심화의 정식 확정.
> - **불확실성의 4대 요소**: 기대값·평균값·분산·표준 편차 — 앞 둘 = 평균의 정량화(미관찰→기대값, 관찰→표본 평균), 뒤 둘 = 퍼짐의 정량화. **목표 = 분산 제어로 불확실성 줄이기** 📌(이 분야 수학 명제 대부분이 기대값·분산 제어). **큰 수의 법칙** = 표본 크기 무한대에서 기대값=표본 평균의 근거.
> - **공분산·상관관계**: 두 변수의 **선형** 관계 — 분산은 공분산 행렬의 **대각선**, 행렬은 **대칭**, 대각화로 상관 제거(🔭 고유값) · 상관=정규화된 변수용(척도 무관)/공분산=비정규화 · **독립은 공분산 0보다 강함**(선형 너머까지) 📌.
> - 기타 예고: 균등·정규 분포(**중심 극한 정리**와 밀접 `🔭`) · 혼합 분포=**가우스 혼합 모델**([[19 실제 vs 시뮬레이션 데이터 — 키-체중·조건부 층화·노이즈|19]] 회수) · 확률 변수의 합/곱의 분포 `🔭` · 그래프(방향/비방향)로 결합 분포 분해 `🔭` · **마르코프 과정**(상태·행동·전이 확률·보상 — 강화 학습 핵심, `🔭 MA 11장`) · **정규화=스케일링=표준화**(빼고 나누기 — 평균/표준편차→표준화, 최소/범위→0~1, 노름→단위벡터).

> [!question] 🎯 왜 이걸 배우나 — 필요성 사슬 (결핍 → 필요 → 발명)
> *출발 문제:* MD 2장에서 확률의 *사건* 어휘(P(X)·조건부·분포 둘)를 정복했는데, AI 문헌은 **확률 변수·분포·기대값** 같은 *변수-분포* 어휘로 말한다 — 같은 개념, 다른 언어.
> 1. 문헌을 읽으려면 어휘 대응표부터 → **용어 지도**(이 절): 사건 어휘(MD) ↔ 변수·분포 어휘(MA)의 다리.
> 2. 함수의 확실성이 무너지는 지점(같은 입력, 다른 결과) → **확률 변수** — 확실성 대신 가능성의 정량화.
> 3. 변수가 여러 개가 되는 순간 결합·주변·조건부·독립이 **분포 레벨**로 승격 → MD 원자들의 일반화 좌표.
> 4. 예측의 오차를 말하려면 평균과 퍼짐의 언어 필요 → **기대값·분산**(불확실성 인코딩) → 제어 목표.
> **한 줄:** 사건 어휘(MD) → 변수·분포 어휘(MA)로 번역 — 새 개념은 넷뿐(확률 변수·사전/사후·기대값/분산·공분산), 나머지는 이미 정복한 것의 새 이름.

> [!abstract] 📘 책 핵심 — [필수수학AI] §2.8 도입 (어휘 먼저)
> - 새 분야 입문의 첫걸음 = **용어** — 새 언어 학습과 같다(프랑스어 '봉주르'의 뜻을 몰라도 쓰임을 보고 익히듯; 때론 '봉수와' 자리에 '봉주르'를 잘못 쓰지만 오래 머물면 맥락에 맞게 됨).
> - 세부를 숙지하지 않고 어휘를 빨리 배우는 또 다른 장점: **분야마다 같은 개념을 다른 용어로 불러** 생기는 혼란·소통 장벽 방지 — 이미 아는 개념의 새 이름임을 알게 됨.
> - AI 이해에 필요한 확률·통계 용어는 많지 않다. **확률 이론의 목표 = 랜덤한 숫자·확률적 사건을 결정론적인 명제로 만드는 것**(인간은 불확실성을 싫어하고 세상을 통제·예측 가능하게 만들고 싶어 함). 용어의 정의까지 알 필요 없이 **개념이 다른 개념과 어떻게 연결되는지**만 이해하면 됨.

> [!abstract] 📘 책 핵심 — §2.8.1~2.8.3 확률 변수·분포·주변 확률
> - **2.8.1 확률 변수**: 모든 것은 확률 변수에서 시작. 함수의 결과는 결정론적($x^2$에 3 → $3^2=9$ 확실). **확률 변수의 결과는 불확실·예측 불가·확률적** — 보기 전까지 어떤 값이 반환될지 모름 → 확실성 대신 **특정 결과를 얻을 가능성의 정량화**가 목표. 주사위(조작 없다고 가정) 4가 나올 확률 1/6이라 자신 있게 말하지만 굴리기 전 결과는 모름 — 예측 가능하다면 카지노는 문 닫고 금융 예측·리스크 조직도 불필요. 함수처럼 **이산 집합(이산 확률 변수) 또는 연속 집합(연속 확률 변수)** 에서 결과 반환. **함수와의 가장 중요한 차이 = 결괏값의 무작위성 vs 확실성.**
> - **2.8.2 확률 분포**: 연속 확률 변수 → **확률 밀도 함수**, 불연속(이산) → **확률 질량 함수**. 이산/연속 여부는 문맥으로. 하나의 확률 변수 = 확률 분포에서 샘플링, 여러 개 = **결합 확률 분포**에서 샘플링. **실무에서 데이터의 전체 결합 확률 분포를 아는 경우는 드묾** — 알거나 학습할 수 있다면 매우 강력한 해법.
> - **2.8.3 주변 확률**: **주변 확률 분포** = 결합 확률 분포의 **가장자리**에 위치(위키피디아 표; 옮긴이 37: 결합 분포 표에서 한 변수를 고정하고 나머지 분포들의 합을 표 *주변*에 기록해서 '주변'). 결합 전체를 알면 **확률의 덧셈 규칙으로 쉽게**:
> $$p(x) = \sum_{y \,\in\, \text{all states of } y} p(x, y)$$
> ⚠️ 책 조판은 등호가 누락된 형태("$p(x)\sum\cdots$")로 보임 — 주변화(marginalization)의 올바른 진술은 위처럼 등호 포함(아래 🔑).

> [!abstract] 📘 책 핵심 — §2.8.4~2.8.8 분포 이름·조건부 계열·사전/사후·혼합
> - **2.8.4 균등·정규 분포**: 가장 많이 쓰는 연속 분포 둘. **정규 분포 ↔ 중심 극한 정리(central limit theorem)와 밀접** `(🔭 예고)`. 유용한 분포는 많지만 필요한 시점에 설명.
> - **2.8.5 조건부 확률과 베이즈 정리**: 여러 확률 변수를 다루는 순간(대부분의 경우 — 성별·키·체중·건강 지수처럼) **독립/조건부 독립**(한 변수의 확률을 안다고 다른 변수의 확률이 달라지지 않음)·**조건부 확률·베이즈 정리·조건부 확률의 곱(연쇄) 규칙**을 배우게 됨 → 전부 [[1.3 조건부 확률·베이즈 정리 — 방향·뒤집기·조건부 결합·합|MD 1.3]]에서 정복 완료.
> - **2.8.6 조건부 확률과 결합 분포**: 결합 확률 분포 그래프에서 **확률 변수 하나의 값을 고정해 분할하면 조건부 확률 분포**([그림 2-7] — ⚠️ 이번 발췌에 그림 미포함). [NOTE] **모든 확률 변수의 결합 확률 분포 전체를 알 수 있다면 베이즈 규칙은 불필요** — 베이즈 규칙은 **결합 분포를 모를 때** 원하는 조건부 확률을 계산하는 도구.
> - **2.8.7 사전 분포·사후 분포·가능도 함수**: 실무자들은 데이터/가중치(파라미터) 중 **무엇을 조건으로 하는지**에 따라 조건부 확률에 다른 이름을 붙임 — **사전 분포**(데이터 관찰 *전* 모델 가중치의 일반적 확률 분포) · **사후 분포**(관찰된 데이터가 주어졌을 때 가중치의 확률 분포) · **가능도(우도) 함수**(특정 가중치 분포가 주어졌을 때 데이터 포인트를 관찰할 확률을 인코딩하는 **함수**). 베이즈 규칙·결합 분포와 밀접. [NOTE] **가능도 분포가 아닌 가능도 함수인 이유**: 확률 분포는 합(연속이면 적분)이 항상 1이어야 하지만 **가능도 함수는 결과들의 합이 반드시 1이 되지 않기 때문**(연속에서 적분해도 1이 아닐 수 있음).
> - **2.8.8 혼합 분포**: 여러 확률 분포를 혼합 — **가우스 혼합 모델(gaussian mixture model)** 이 유명. 앞서 본 남녀 키 데이터가 좋은 예([[19 실제 vs 시뮬레이션 데이터 — 키-체중·조건부 층화·노이즈|19]] 그림 2-2 쌍봉의 이름 확정).

> [!abstract] 📘 책 핵심 — §2.8.9~2.8.15 합·곱, 그래프, 기대값·분산, 공분산, 마르코프, 정규화, 예시
> - **2.8.9 확률 변수의 합과 곱**: 단순 분포에서 샘플링한 확률 변수를 더하거나 곱해 더 복잡한 분포의 새 확률 변수 생성 — 자연스러운 질문: "**합·곱의 분포는 어떻게 되는가**" `(🔭 예고)`.
> - **2.8.10 그래프로 결합 분포 묘사**: 방향성/비방향성 그래프(다이어그램)로 결합 확률 분포의 구성 요소를 효율적으로 분해 → 훨씬 간편한 계산 `(🔭 예고 — 나중에 자세히)`.
> - **2.8.11 기대값·평균·분산·불확실성**: 핵심 4요소 = **기대값(expectation)·평균값(mean)·분산(variance)·표준 편차(standard deviation)**. 기대값·평균 = 확률 변수 평균의 정량화 / 분산·표준 편차 = 평균 중심 퍼짐의 정량화 → **확률 변수의 불확실성을 인코딩. 목표 = 분산을 제어해 불확실성 줄이기**(분산 클수록 평균 예측의 오류 큼 — 이 분야 수학 명제·부등식·정리 대부분이 기대값·분산 제어). 분포가 있으면 기대값(예상 결괏값)·분산(기대값으로부터 예상 거리의 **제곱**)·표준 편차(예상 거리) 계산; 관찰 데이터에선 표본 평균·분산·표준 편차. **미관찰 → '기대값'으로 추측 / 관찰 → 통계량 계산**. 관찰 통계량과 추측의 차이, 전체 인구 측정의 이상적 경우 → **큰 수의 법칙**: 표본 크기가 무한대(전체 인구 측정)면 **기대값 = 표본 평균**의 근거.
> - **2.8.12 공분산과 상관관계**: 확률 변수 2+ → **공분산·상관관계·공분산 행렬**. 선형대수(벡터·행렬·행렬 분해 — 고유값·특이값 분해)가 확률·통계와 결합. **각 변수의 분산 = 공분산 행렬의 대각선**, 쌍의 공분산 = 비대각. **공분산 행렬은 대칭** — 선형대수 기법으로 **대각화(diagonalize)** 하면 상관관계 제거 가능 `(🔭 예고 — 고유값)`. 독립 vs 공분산 0: 공분산·상관 = 두 변수의 **선형 관계** 파악 — 상관 = **정규화된** 변수 적용(단위가 수백만이든 0.001이든 척도 무관), 공분산 = 비정규화 변수. **독립성은 공분산 0보다 강한 개념**(세상엔 선형만 있지 않으므로 — 상관·공분산으로는 독립을 담보 못 함).
> - **2.8.13 마르코프 과정**: **강화 학습** 패러다임의 핵심 — 구성: 모든 가능한 상태(status) · 에이전트의 가능한 행동(action) 집합 · 상태 간 **전이 확률(transition probability)** 행렬 · 행동 후 상태 전이의 확률 분포 · 최대화할 **보상 함수(reward function)**. 예: 보드 게임 AI·스마트 온도 조절기 `(🔭 예고 — MA 11장 확률)`.
> - **2.8.14 정규화·스케일링·표준화**: normalizing·scaling·standardizing은 맥락 따라 **동의어** — 목표 동일: **특정 숫자를 빼고(이동 shift) 상수로 나눈다(스케일)**. 평균 빼고 표준 편차로 나누면 → 평균 0·표준 편차 1(**표준화**); 최솟값 빼고 (최대−최소)로 나누면 → 0~1; **벡터 정규화** = 길이로 나눠 길이 1. 목표 = 본질적 가변성을 유지하며 0 중심화 또는 퍼짐 제한.
> - **2.8.15 일반적인 예시**: 수학자들은 확률을 **사건 발생까지의 시간**으로 표현하길 좋아함 — 동전·주사위·항아리 공·카드·기차 도착·핫라인 전화·광고 클릭·질병과 증상·형사 재판 증거·기계 고장. 확률 이론 외에 **통계역학**(파티션 함수)·**정보 이론**(신호 vs 노이즈·엔트로피·교차 엔트로피 함수) 용어도 차용 `(🔭 예고 — 등장 시 설명)`.

### 🔑 MD 사건 어휘 ↔ MA 변수·분포 어휘 — 번역표
| MD 2장(사건 어휘, 원자) | MA §2.8(변수·분포 어휘) | 승격 내용 |

| 사건 $X$·$P(X)$ ([[1.1 확률 — 정의·가능도·오즈|1.1]]) | **확률 변수**·확률 분포 | 사건 하나 → 결과를 뱉는 변수 전체 |
| 주변·결합·덧셈 법칙 ([[1.2 확률 계산 — 주변·결합·합·덧셈 법칙|1.2]]) | 주변/결합 **확률 분포**·$p(x)=\sum_y p(x,y)$ | 수 하나 → 분포 전체의 주변화 |
| 조건부·베이즈·일반 곱셈 법칙 ([[1.3 조건부 확률·베이즈 정리 — 방향·뒤집기·조건부 결합·합|1.3]]) | 조건부 분포·**연쇄 규칙**·독립/조건부 독립 | "나중에"라던 연쇄 규칙 이름 등장 |
| pmf·CDF·가능도 방향 ([[1.4 이항 분포·베타 분포 — pmf·CDF·확률의 확률|1.4]]) | **pdf** 명명·사전/사후/**가능도 함수** | 베타의 "확률의 확률"이 사후 분포의 원형 |

- **새로 배운 건 사실상 넷**: 확률 변수(무작위성이 유일한 차이), 사전/사후(조건 대상이 가중치냐 데이터냐), 기대값·분산 계열(불확실성의 자), 공분산 행렬(선형대수와의 접합점). 나머지는 이미 정복한 개념의 새 이름 — 책이 말한 "어휘 먼저"의 효용을 그대로 체험.

### 🔑 두 책 삼각측량 완성 — 내 통찰 2건이 MA의 공식 서술로 확인됐다
- **가능도 함수(2.8.7)**: [[1.1 확률 — 정의·가능도·오즈|MD 1.1]]에서 책의 "과거 빈도" 서술을 입문 근사로 보고 🪂 심화에 "$L(\theta \mid D)$ — 파라미터의 함수, 합=1 규칙 없음"을 적어뒀다 — MA가 정확히 그 형태(**가중치를 조건으로 데이터 관찰 확률을 인코딩하는 함수, 합·적분이 1일 필요 없음**)로 공식화. 심화가 예고에서 사실로 승격.
- **베이즈 = 결합의 파생물(2.8.6 NOTE)**: [[1.3 조건부 확률·베이즈 정리 — 방향·뒤집기·조건부 결합·합|MD 1.3]]의 "베이즈는 새 공리가 아니라 결합 대칭성의 재배열" 통찰을 MA가 "**결합 분포 전체를 알면 베이즈 규칙 불필요**"로 뒷받침 — 베이즈의 존재 이유는 *결합 분포의 무지*라는 같은 결론.

### 🔑 "분포"·"주변" — 일상어의 공간적 어감이 낳은 오해 (2026-07-18 질문 회수)
- **분포(distribution([ˌdɪstrɪˈbjuːʃən], 디스트리뷰션, 디스트리뷰션)) ≠ 확률이 물리적으로 퍼져 있음.** 확률 변수 X가 가질 수 있는 각 값에 총 확률 1을 얼마씩 나눠줬는지의 **배분 규칙**일 뿐 — p(x)는 "x가 받는 몫"을 정하는 함수. 공정한 주사위면 6개 값에 1/6씩 배분하는 것이 분포이고, 이산이면 막대(pmf)로 연속이면 곡선(pdf)으로 그 배분을 그림으로 그린 것뿐이다.
- **주변(marginal) ≠ 확률이 어딘가의 가장자리에 위치.** 20세기 초 분할표(contingency table) 관행에서 나온 순수 역사적 명명이다 — 결합 분포를 행×열 표로 그리고 그 표의 **여백(margin)**에 행·열 합계를 적던 것이 그대로 이름이 됐다(칼 피어슨~1900년대; 상세·출처와 여백까지 채운 12칸 격자 예시는 [[1.2 확률 계산 — 주변·결합·합·덧셈 법칙|1.2]] 🔑).
- 두 오해 모두 뿌리가 같다 — "분포되다·주변에 있다"는 한국어 일상어의 **공간적 어감**에 낚인 것. 실제로는 각각 "값들에 대한 배분 방식"과 "표 계산 결과를 적어 넣던 위치"일 뿐 공간적 실체가 아니다.

### 🔑 ⚠️ 주변화 수식의 조판 — 등호 누락으로 보임
- 책 인쇄본의 주변 확률 수식은 "$p(x)\sum_{y}p(x,y)$"처럼 **등호 없이** 붙어 있다. 주변화의 올바른 진술은 $p(x) = \sum_{y \in \text{states of } y} p(x,y)$ — 결합에서 $y$를 전부 더해 $x$만의 분포를 얻는 것([[1.2 확률 계산 — 주변·결합·합·덧셈 법칙|1.2]] 덧셈 규칙의 분포판, 상호 배타 상태들의 합). 검산: 12칸 격자에서 $p(H) = \sum_{d=1}^{6} p(H, d) = 6 \times \frac{1}{12} = \frac{1}{2}$ ✓.

### 🔑 공분산 행렬 — 선형대수 원자 4개가 확률로 재취업하는 현장
- **대각 = 각 변수의 분산**: [[2.5 행렬 노름·대각합 — 프로베니우스·ℓ2·tr(AᵀA)|원자 2.5]]의 "대각 = 자기→자기 성분" 통찰 그대로(자기 자신과의 공분산 = 분산). **대칭**: [[2.3 행렬 전치 — 내적·외적 표기법·LIVE EVIL·대칭 행렬|원자 2.3]]의 대칭 행렬( $\text{cov}(x,y)=\text{cov}(y,x)$ — $A^{\mathsf{T}}A$ 곱셈 기법과 동족). **대각화로 상관 제거**: 고유값 분해 `(🔭)` — [[17 행렬 계수(rank) — 정보 차원·최대계수·이동·확장|17]] 이후 예고돼 온 고유값의 첫 실전 용도 예고. **상관 = 정규화된 변수**: [[1.9 상관관계와 코사인 유사도 — 피어슨 상관계수·평균중심화·분할정규화|원자 1.9]] 피어슨 = 평균중심화(빼기)+노름 나눔(나누기) — 2.8.14의 "빼고 나누기" 템플릿이 그것.
- "독립 > 공분산 0"의 이유도 1.9에서 이미 봄 — 피어슨은 **선형** 닮음만 잰다. 비선형 의존은 공분산 0으로도 숨는다([[20 수학 모델 — 선형·비선형·시뮬레이션과 AI 3중 관계|20]] 비선형 세상).

### 🔑 "분산 제어 = 불확실성 축소"가 이 분야의 숨은 목적 함수
- 2.8.11의 선언 — "이 분야 수학 명제·부등식·정리의 대부분은 기대값과 분산의 제어" — 는 앞으로 만날 도구들의 예고편: MD 1.4의 베타 분포가 좁아진 것(22.5→13.16%)도, 큰 수의 법칙도, 정규화(2.8.14)도 전부 **퍼짐을 줄이거나 표준화하는 조작**이다. 새 정리를 만나면 "이건 기대값·분산의 어느 쪽을 제어하나"를 물으면 좌표가 잡힌다.

## 🆕 용어
| 용어 | 한 줄 정의 |
|---|---|
| **분포** (distribution) | 확률 변수의 가능한 값들에 총 확률 1을 나눠주는 **배분** 규칙 — 일상어의 "퍼져 있다"가 아님; 이산은 pmf, 연속은 pdf로 구체화 |
| **확률 변수** (random variable, 이산/연속) | 결과가 무작위인 "함수의 사촌" — 보기 전까지 값을 모름; 확실성 대신 가능성 정량화; 이산/연속 집합에서 결과 반환 |
| **확률 밀도 함수** (pdf) | 연속 확률 변수의 분포 함수(이산은 pmf) — [[1.4 이항 분포·베타 분포 — pmf·CDF·확률의 확률|1.4]] 예고의 명명 확정(정식 수식은 🔭) |
| **결합 확률 분포** (joint probability distribution) | 여러 확률 변수를 함께 샘플링하는 분포 — 전체를 알면 매우 강력(그리고 베이즈 불필요); 실무에선 드묾 |
| **주변 확률 분포** (marginal probability distribution) | 결합의 가장자리 — $p(x)=\sum_y p(x,y)$ (덧셈 규칙·주변화; 표 '주변'에 기록해서 유래) |
| **사전/사후 분포** (prior/posterior distribution) | 데이터 관찰 *전* 가중치의 분포 / 관찰 데이터가 주어진 *후* 가중치의 분포 — 조건 대상에 따른 조건부 확률의 이름 |
| **가능도 함수** (likelihood function) | 특정 가중치가 주어졌을 때 데이터 관찰 확률을 인코딩하는 **함수** — 분포가 아님(합·적분이 1일 필요 없음) |
| **기대값·평균·분산·표준 편차** | 평균의 정량화(미관찰=기대값/관찰=표본 평균) · 퍼짐의 정량화(거리 제곱/거리) — 불확실성 인코딩, 분산 제어가 목표(정식 수식 🔭) |
| **큰 수의 법칙** (law of large numbers) | 표본 크기 무한대(전체 인구 측정의 이상적 경우)에서 기대값 = 표본 평균이 되는 근거 |
| **공분산·상관관계·공분산 행렬** | 두 변수의 **선형** 관계 — 상관=정규화(척도 무관)/공분산=비정규화; 행렬은 대칭·대각=분산·대각화로 상관 제거(🔭); 독립은 공분산 0보다 강함 |
| **마르코프 과정** (Markov process, 🔭 MA 11장) | 상태·행동·전이 확률 행렬·전이 분포·보상 함수 — 강화 학습의 뼈대(보드 게임 AI·온도 조절기) |
| **정규화·스케일링·표준화** | 동의어 계열 — **빼고(이동) 나누기(스케일)**: 평균/표준편차→표준화(0,1), 최소/범위→0~1, 노름→단위벡터([[02 벡터 크기와 단위벡터 (노름)|02]]) |
| **중심 극한 정리** (central limit theorem, 🔭) | 정규 분포와 밀접한 확률 이론의 기본 정리 — 이름만 예고 |

## 📌 암기 카드
> (→ [[_핵심 암기장]])

| 앞면 (cue) | 뒷면 (외울 내용) |
|---|---|
| 확률 이론의 목표(책)? 📌 | **랜덤한 숫자·확률적 사건을 결정론적 명제로** 만드는 것(인간은 불확실성을 싫어함) |
| 확률 변수 vs 함수의 결정적 차이? 📌 | **결괏값의 무작위성 vs 확실성** — x²(3)=9 확실 / 확률 변수는 보기 전까지 모름 → 가능성 정량화로 목표 교체 |
| 이산 vs 연속 확률 변수의 분포 함수? | 이산 → **pmf**(확률 질량) / 연속 → **pdf**(확률 밀도) — 문맥으로 구분 |
| 분포(distribution)란? 📌 | 확률 변수의 가능한 값에 총 확률 1을 나눠주는 **배분** — "퍼져 있다"는 뜻 아님 |
| 주변화 공식? 📌 | $p(x)=\sum_{y}p(x,y)$ — 결합에서 다른 변수를 전부 더함(덧셈 규칙; 표 가장자리라 '주변') |
| 주변(marginal)의 어원? 📌 | 결합 확률 **표의 여백**에 행·열 합계를 적던 분할표 관행(칼 피어슨~1900) — 공간적 은유 아님 |
| 결합 분포를 전부 알면 베이즈는? 📌 | **불필요** — 베이즈 규칙은 결합 분포를 *모를 때* 조건부를 구하는 도구(1.3 "결합 대칭성 재배열" 확인) |
| 사전/사후/가능도 — 조건 대상은? 📌 | 사전=관찰 **전** 가중치 분포 / 사후=**데이터가 주어졌을 때** 가중치 분포 / 가능도=**가중치가 주어졌을 때** 데이터 관찰 확률의 **함수** |
| 가능도가 '분포'가 아닌 이유? 📌 | 결과들의 **합(적분)이 1이 될 필요가 없음** — 확률 분포의 자격 미달(MD 1.1 심화의 공식 확정) |
| 불확실성의 4대 요소와 목표? 📌 | 기대값·평균(평균의 정량화) · 분산·표준 편차(퍼짐) — **분산 제어로 불확실성 축소**; 분야 수학 대부분이 기대값·분산 제어 |
| 기대값 vs 표본 평균의 다리? | **큰 수의 법칙** — 표본 무한대(전체 인구)면 기대값=표본 평균 |
| 공분산 행렬의 구조? 📌 | **대각 = 각 변수의 분산**, 비대각 = 쌍의 공분산, **대칭** — 대각화하면 상관 제거(🔭 고유값) |
| 독립 vs 공분산 0? 📌 | **독립이 더 강함** — 공분산·상관은 **선형** 관계만 포착(비선형 의존은 공분산 0으로 숨음) |
| 정규화·스케일링·표준화의 공통 형식? 📌 | **빼고(이동) 나누기(스케일)** — 평균·표준편차(표준화), 최소·범위(0~1), 노름(단위벡터) |
| 마르코프 과정의 5요소? | 상태 · 행동 집합 · 전이 확률 행렬 · 행동 후 전이 분포 · 보상 함수 — 강화 학습 핵심(🔭 MA 11장) |
