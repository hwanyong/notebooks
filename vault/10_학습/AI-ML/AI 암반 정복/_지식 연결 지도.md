# 🌉 지식 연결 지도 (사이드 지식 허브)

> **메인(교재 핵심)** 과 별개로, 공부 중 따라온 **사이드 지식**(배경 연계·파생·타 분야 비유)을 한곳에 모아 도메인별로 접근하는 지도. *(이 코스 내부의 타 분야 연결)*
> 노트 안 사이드 지식은 콜아웃 박스 + 태그로 구분됨 → 여기서 도메인으로 역추적.

🔙 [[_커리큘럼 MOC]] ｜ 🌐 영역을 넘는 연결은 → [[_지식 연결 허브]] (볼트 전역 cross-domain)

## 🔖 사이드 지식 4종류
| 종류 | 콜아웃 | 태그 | 뜻 |
|---|---|---|---|
| 🧭 비유 | `> [!example]` | `#사이드/비유` | 개념을 일상/타 분야에 빗댐 |
| 🔗 배경·연계 | `> [!info]` | `#사이드/배경` | 이해에 필요한 선수·주변 지식 |
| 🪂 파생·심화 | `> [!tip]` | `#사이드/심화` | 메인에서 한 발 더 나간 내용 |
| 🌉 타 분야 연결 | `> [!quote]` | `#연결/<도메인>` | 다른 전공 영역과의 구조적 대응 |

## 🌉 타 분야 연결 (도메인별)
> 같은 구조·패러다임을 공유하는 다른 분야와의 다리. 도메인 태그로도 검색 가능.

### 네트워크 `#연결/네트워크`
- **역전파 vs 라우팅 전파 — 같은 위상, 다른 알고리즘** — 둘은 *노드-에지 그래프 위 메시지 전달*이라는 골격만 공유(역전파=역방향 자동미분·메시지=그래디언트 / 거리벡터 라우팅=Bellman-Ford·메시지=거리). **같은 패러다임이 아니다**(과거 단정 교정). 단 *경사하강 수렴 동역학 ↔ 라우팅 수렴 동역학*은 비유로 유효(오버슈팅↔플래핑 등). → 정밀: [[두 전파 — 역전파 vs 라우팅 (그래프는 같고 메시지는 다르다)]] · [[T01 경사 하강법과 훈련 루프]] §🌉

### 분산 시스템 `#연결/분산시스템`
- **가중치 난수 초기화 ↔ 분산 데드락 회피** — 둘 다 *"중앙 통제 없이, 완벽 대칭이면 교착"* 라는 같은 문제. 해법도 같다: **의도적 무작위성 주입으로 대칭 깨기(Symmetry Breaking).** → [[06 가중치 — 상태·초기화·학습 실패]] §2-1

**메커니즘 (CSMA/CD · 지수 백오프):** 이더넷/와이파이에서 컴퓨터 A·B가 한 회선을 공유.
- 난수 없으면: 동시 전송 → 충돌 → "1초 뒤 재전송"이 양쪽 다 똑같아 **또 동시 전송 → 무한 충돌(데드락)**.
- 해법 **지수 백오프(Exponential Backoff):** 충돌 시 "1~N 사이 난수만큼 대기 후 재전송" → A=3ms, B=7ms로 갈려 A 먼저 성공. **난수가 대칭을 깸.**

**평행이론:**

| 분산 시스템(네트워크) | 딥러닝(PyTorch) |
|---|---|
| 노드가 같은 규칙(1초 대기)으로 시작 → 무한 충돌 | 뉴런이 같은 가중치(0)로 시작 → 역전파 명령 동일, 복제 |
| 랜덤 대기 시간으로 탈출, 각자 순서 확보 | 랜덤 가중치로 탈출, 각자 역할(가로선/세로선) 분화 |
| → 대칭성 깨기 | → 대칭성 깨기 |

> 한 줄: **"구조적 대칭이 시스템을 마비시키므로, 초기에 반드시 무작위성을 주입한다"** 는 거시 논리가 두 분야에 똑같이 적용. (관련: 네트워크 라우팅 연결도 참조)

### 물리 · 선형대수 `#연결/물리`
- **텐서 = 좌표계가 바뀌어도 안 변하는 불변량.** 물리의 힘(Force)이 축을 회전해도 실체가 안 변하듯, 텐서는 좌표 변환에 독립적인 선형 관계. → [[02 텐서 이해하기|텐서]]
- **임베딩 벡터의 의미 거리(코사인 유사도)도 같은 불변량** — 공간을 회전·이동해도 단어 사이 의미 거리는 유지 → RAG·시맨틱 검색이 좌표계와 무관하게 동작. (1단계에서 심화)
- 한 줄: **"보는 각도를 바꿔도 본질은 그대로"** 가 텐서·임베딩·물리량을 관통하는 공통 논리.
