## 🔬 코스 A — AI 리서처 (연구원 / 모델 연구자)
> 논문을 두려움 없이 읽고, 수식 모델 기반으로 새 구조·정렬 기법을 제안하는 학술 체력 완성.

### 🧠 개념적 원리 & 수리적 시각화
- [ ] 결정론→확률 이론적 사고 전환 + 최대 가능도 추정(MLE) — **[필수수학AI] 8.1~8.4**
- [ ] 명시적 vs 암시적 밀도 모델 비교 — **[필수수학AI] 8.5~8.7**
- [ ] VAE 변분법 근사·볼츠만 머신(마르코프 체인)·GAN 유도 — **[필수수학AI] 8.8~8.11**
- [ ] 기타 생성 모델(나이브 베이즈·가우스 혼합 모델)·생성 모델의 발전사(홉필드 네트워크·볼츠만 머신·RBM·오토인코더) — **[필수수학AI] 8.13~8.14** _(2026-07-15 전수조사 발견)_
- [ ] 그래프 모델(노드·엣지·피처)·케일리 그래프 그룹 구조 — **[필수수학AI] 9.1, 9.4**
- [ ] 그래프를 사용한 역행렬 계산·그래프의 실전 응용 사례·동적 그래프 모델·그래프 이론의 역사와 이론적 기초·그래프 알고리즘 — **[필수수학AI] 9.3, 9.6, 9.10, 9.13~9.15** _(2026-07-15 전수조사 발견)_
- [ ] 인과관계 그래프·펄의 Do 연산·인과 추론·베이즈 네트워크 — **[필수수학AI] 9.11~9.12 / 11장 전체**
- [ ] 수학적 논리(명제논리·1차논리·확률적논리·퍼지논리·시간논리) — **[필수수학AI] 12장 전체** _(2026-07-15 전수조사 발견 — 전체 커리큘럼에서 완전히 누락돼 있었음)_
- [ ] 편미분 방정식 기반 물리 신경망(PINN) — **[필수수학AI] 13장 전체**
- [ ] AI 윤리·정책·법률(편향·공정성·오용·정책 대응) — **[필수수학AI] 14장 전체** _(2026-07-15 전수조사 발견 — 전체 커리큘럼에서 완전히 누락돼 있었음)_

### 💻 밑바닥 코드 구현 & 시스템적 확장
- [ ] 생성 신경망 고에너지 물리 데이터 추정 + 확률적 언어 모델링 — **[필수수학AI] 8.12, 8.15**
- [ ] 페이지랭크 + 그래프 메시지 전달 구조 — **[필수수학AI] 9.2, 9.5**
- [ ] 랜덤 워크·노드 표현 학습 기반 GNN 응용 — **[필수수학AI] 9.7~9.9**
- [ ] 스팸 분류기 미세튜닝(분류 헤드 결합) 생코딩 — **[밑바닥LLM] 6장 전체**
- [ ] 지시 데이터셋 로더 + 지시 미세튜닝 + 최종 정렬 성능 평가 — **[밑바닥LLM] 7장 전체**

## 🛠️ 코스 B — AI 엔지니어 (AI 시스템 아키텍트)
> 한정된 하드웨어 자원에서 대규모 분산 AI 서비스를 안전하고 초고속으로 서빙하는 인프라 설계.

### 🧠 개념적 원리 & 수리적 시각화
- [ ] 공짜 점심은 없다(No Free Lunch) — **[필수수학AI] 10.1** _(2026-07-15 전수조사 발견)_
- [ ] 복잡도 분석·빅오 표기법 기반 인프라 최적화 — **[필수수학AI] 10.2**
- [ ] 네트워크 최적화·최단 경로·큐잉 이론(서빙 병목 진단) — **[필수수학AI] 10.3~10.5**
- [ ] N-퀸즈 문제·선형 최적화(심플렉스법)·게임 이론과 멀티 에이전트·큐잉·인벤토리·해밀턴-야코비-벨만 방정식·운영 연구를 위한 AI — **[필수수학AI] 10.6~10.13** _(2026-07-15 전수조사 발견 — 챕터 절반 이상, 특히 10.7 선형계획법·10.8 게임이론 비중 큼)_
- [ ] AI 엔지니어링 스택 정의, ML 엔지니어링/풀스택과의 제약 차이 — **[AI엔지니어링] 1장 전체**
- [ ] 파운데이션 모델 지도/선호도 파인튜닝 사후학습 제약 — **[AI엔지니어링] 2장 전체**
- [ ] 교차 엔트로피·퍼플렉시티·AI 평가자 운영 의사결정 — **[AI엔지니어링] 3장 전체**
- [ ] 모델 벤치마크 선택 전략(도메인·생성·비용·지연) — **[AI엔지니어링] 4장 전체**

### 💻 밑바닥 코드 구현 & 시스템적 확장
- [ ] 인컨텍스트 학습 최적화 + 탈옥·프롬프트 주입 방어 — **[AI엔지니어링] 5장 전체**
- [ ] 밀집 검색·리랭킹 RAG + 에이전트 메모리 시스템 — **[AI엔지니어링] 6장 전체**
- [ ] 역전파 메모리 병목·FP16/BF16·PEFT 이식 전략 — **[AI엔지니어링] 7장 전체**
- [ ] 합성 데이터 생성·중복 제거·클렌징 파이프라인 — **[AI엔지니어링] 8장 전체**
- [ ] AI 가속기 제어·추론 레이턴시/처리량 엔지니어링 — **[AI엔지니어링] 9장 전체**
- [ ] **프로덕션 7단계 아키텍처** (컨텍스트 보강→가드레일→라우터/게이트웨이→캐시→에이전트 패턴→모니터링→오케스트레이션) + 피드백 플라이휠 — **[AI엔지니어링] 10장 전체**
- [ ] 데이터 과학 강점 포지셔닝·직무 리스크 제어 — **[필수수학] 8장 전체**

## 🔭 확장 참조 (정렬·추론모델 부채 — 2026-07 갭 분석)
> [!info] 🔭 왜 이 표가 있나
> 필수수학AI(2023)는 VAE·GAN·볼츠만머신까지만 다룬다. RLHF/DPO·추론모델(GRPO)은 이 책 출간 이후 주류가 된 내용이라 별도 트리거로 관리한다. 코스 A/B 결정 전에는 열지 않는다.

| 구분 | 트리거 | 내용 | 자원 | 규모 |
|---|---|---|---|---|
| 코스A | 지시 미세튜닝([밑바닥LLM] 7장) 완료 직후 | DPO 구현(GPT-2 124M 기준) | aalokpatwa/dpo | 5~8h |
| 코스A | 리서치 신호 확인 이후 | GRPO·추론모델 미니 재현(캡스톤급) | TinyZero·microR1 | 15~25h |
| 코스A(선택) | 관심 생기면 | 디퓨전 모델 | — | 미정 |
| 공통(선택) | 관심 생기면 | 멀티모달(비전-언어) | — | 미정 |
| 코스B | 엔지니어 트랙 확정 시 | FSDP·텐서/파이프라인 병렬·Flash Attention | — | 미정 |

🔗 아키텍처 현대화(RoPE·RMSNorm·SwiGLU·GQA·토이 MoE) 확장 참조는 [[_3단계 - 트랜스포머와 LoRA]]에.
🔗 강화학습 일반 이론(MDP·Q러닝·정책 vs 보상 기반 학습) 배경은 [[강화학습 패러다임 — 정적 데이터 vs 실시간 상호작용]]에 — RLHF·GRPO는 이 패러다임의 LLM 적용 사례.
