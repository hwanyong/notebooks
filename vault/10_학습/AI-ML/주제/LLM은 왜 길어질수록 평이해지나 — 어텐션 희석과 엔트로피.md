# LLM([ɛl ɛl ɛm], 엘엘엠, 엘엘엠)은 왜 길어질수록 평이해지나 — 어텐션 희석과 엔트로피

> 파라미터·컨텍스트 창이 아무리 커도, **대화가 길어지면 출력이 평이·보편적인 답으로 수렴**한다. 기분 탓이 아니라 트랜스포머의 수학 구조 + 정보이론이 맞물려 생기는 **아키텍처적 한계**. "제약(어텐션 시그널)이 약해지면 → 정보 엔트로피가 최대화 → 인터넷 평균(Prior([ˈpraɪər], 프라이어, 프라이어))으로 회귀."

> 🐍 **AI 영역 노트(코스 외 주제)** — 책 진도와 무관한 LLM 메커니즘 심층.
> 🌐 영역 간 연결: [[_지식 연결 허브]] ｜ 📖 [[_AI-ML MOC]]
> 💡 메타 원리(범분야): [[엔트로피와 질서 — 최소 에너지 법칙과 엔지니어링|엔트로피와 질서]] — *이 노트는 그 통찰의 LLM 실증 사례*

> 🎯 **핵심 (TL;DR)**
> - **현상:** 컨텍스트가 길수록 어텐션 시그널이 흐려져 → 출력이 *가장 평균적이고 안전한 단어*로 수렴.
> - **4대 원인:** ① 어텐션 희석·Lost in the Middle([lɔːst ɪn ðə ˈmɪdəl], 로스트 인 더 미들, 로스트 인 더 미들) ② 자기회귀 오염 ③ 평균 회귀(Prior 지배) ④ 위치 임베딩 extrapolation([ɪkˌstræpəˈleɪʃən], 익스트래펄레이션, 엑스트라폴레이션) 한계.
> - **수학 뼈대:** 베이지안 붕괴(우도 무력화→Prior) + Softmax([ˈsɒftmæks], 소프트맥스, 소프트맥스) 평탄화(High Entropy([haɪ ˈɛntrəpi], 하이 엔트로피, 하이 엔트로피)).
> - **왜 자연스러운가:** 정보 엔트로피 = 열역학 엔트로피 = 같은 수식. 제약이 풀리면 가장 평평한 분포로.
> - **대응:** 외부 에너지 주입 — 컨텍스트 압축·요약·RAG([ræɡ], 래그, 래그).

## 1. 네 가지 아키텍처 원인

**① 어텐션 희석(Attention Dilution([əˈtɛnʃən daɪˈluːʃən], 어텐션 다일루션, 어텐션 딜루션)) + Lost in the Middle**
- Self-Attention([sɛlf əˈtɛnʃən], 셀프 어텐션, 셀프 어텐션)은 단어 간 연관성을 **softmax([ˈsɒftmæks], 소프트맥스, 소프트맥스) 확률**로 계산한다. 컨텍스트가 길수록 분모(토큰 총개수)가 커져, 특정 핵심 지시·맥락에 줄 수 있는 **어텐션 가중치의 절대 크기가 희석**된다.
- 연구상 LLM은 컨텍스트의 **맨 앞(프롬프트)·맨 뒤(최신)** 는 잘 기억하나 **중간 정보는 놓친다**(= *Lost in the Middle*). 날카로운 맥락이 대화의 바다 중간에 묻혀 답이 안전·평이해진다.

**② 자기회귀 오염(Autoregressive Contamination([ˌɔːtoʊrɪˈɡrɛsɪv kənˌtæmɪˈneɪʃən], 오토리그레시브 컨태미네이션, 오토리그레시브 컨태미네이션))**
- LLM은 방금 출력한 단어를 다시 입력에 넣어 다음을 예측(autoregressive). 대화가 길어지면 "종합하자면", "일반적으로" 같은 **관용·평이한 문장**을 뱉기 시작하고, 그 평범한 텍스트가 컨텍스트에 쌓이면 **다음 문장도 그 평범함의 지배**를 받는다 = *진부함이 진부함을 낳는 피드백 루프(엔트로피 증가)*.

**③ 평균 회귀(Regression to the Mean([rɪˈɡrɛʃən tə ðə miːn], 리그레션 투 더 민, 리그레션 투 더 민))**
- 프롬프트가 짧고 명확하면 모델은 그 강한 제약(Signal)에 묶여 독창적 답을 낸다. 길어져 어텐션이 흐려지면, **가장 확률적으로 안전한 선택** = 학습한 인터넷 데이터의 **평균적·대중적 분포(Prior)** 로 회귀 → 교과서적이고 지루한 답.

**④ 위치 임베딩 extrapolation 한계 (RoPE([roʊp], 로프, 로프))**
- 각 토큰의 순서를 알려주는 **위치 임베딩(RoPE 등)** 은 수백만 토큰까지 *억지로 늘려놓은* 것. 길어질수록 고차원 벡터 공간에서 토큰 간 **기하학적 거리·관계가 미묘하게 왜곡**되어, 멀리 있는 날카로운 연결고리를 못 낚고 뭉뚱그린다.

## 2. 수학적 뼈대 — 베이지안 붕괴 + Softmax 평탄화

**① 베이지안 붕괴(Bayesian Collapse([ˈbeɪziən kəˈlæps], 베이지안 컬랩스, 베이지안 콜랩스))**
다음 토큰 예측은 조건부 확률 추론이다:
$$P(\text{Token}\mid\text{Context}) \;\propto\; P(\text{Context}\mid\text{Token})\;\times\;P(\text{Token})$$
- `P(Context|Token)` = **우도(Likelihood([ˈlaɪklihʊd], 라이클리후드, 라이클리후드)):** 이 단어가 지금의 독특한 맥락에 얼마나 부합하나.
- `P(Token)` = **사전 확률(Prior):** 맥락 빼고, 인터넷 전체에서 이 단어가 평소 얼마나 흔한가.
- 대화가 길어져 어텐션이 흐려지면 → 우도 `P(Context|Token)`의 변별력이 떨어져 모든 단어에 **평평한(Uniform([ˈjuːnɪfɔːrm], 유니폼, 유니폼))** 값 → 결국 전체 확률이 **오직 Prior에 의해** 결정 → *인터넷 평균으로 회귀*.

**② Softmax 평탄화**
$$\text{Softmax}(x_i) = \frac{e^{x_i}}{\sum_j e^{x_j}}$$
- 컨텍스트가 커지면 분모의 항 수(`j`)가 수만~수십만으로 늘고, 특정 맥락에 강한 가중치(`x_i`)를 못 주면 확률 분포가 **바늘처럼 뾰족한 모양 → 부드러운 언덕(High Entropy)** 으로 평평해진다.
- 분포가 평평해지면 Language Modeling Head([ˈlæŋɡwɪdʒ ˈmɒdəlɪŋ hɛd], 랭귀지 모델링 헤드, 랭귀지 모델링 헤드)는 모험을 못 하고 **가장 확률 매스가 큰 안전한 단어**(가장 뻔하고 교과서적인 단어)만 고른다.
- 🔗 이 softmax는 이미 [[05 퍼셉트론·편향·은닉층|퍼셉트론]]에서 "로짓→확률"로 배운 그 함수. 같은 메커니즘의 장기 컨텍스트 부작용.

## 3. 왜 '자연스러운' 현상인가 — 정보 엔트로피 = 열역학 엔트로피
> 여기서 "에너지가 적게 든다"는 전력이 아니라 **정보이론의 엔트로피(무질서) 법칙**이다.

- **섀넌 정보 엔트로피:** $H(X) = -\sum_i P(x_i)\log_2 P(x_i)$ — 확률 분포가 불확실·평평할수록 H가 커진다. *"확률 분포가 평평해진다 = 정보 엔트로피가 증가한다"* 는 수학적으로 같은 말(비유가 아님).
- **LLM의 학습 목표가 엔트로피 낮추기:** 훈련 손실 함수 이름이 **Cross-Entropy Loss([krɔːs ˈɛntrəpi lɒs], 크로스 엔트로피 로스, 크로스 엔트로피 로스)** — 정답 단어에 확률을 몰아 분포를 뾰족하게(엔트로피 최소화) 깎는 게 학습. 컨텍스트가 길어져 제약이 풀리면, *억눌렸던 엔트로피가 자연 상태의 최대값으로 되돌아오는 것*.
- **최대 엔트로피 원리(통계·물리):** 제약(Information)이 사라지면 분포는 *가질 수 있는 가장 엔트로피 높은 상태(가장 평범)* 를 택한다.
- **세 학문이 한 수식:** 섀넌 `H=-ΣP log P` ↔ 볼츠만 `S=-k Σp ln p` ↔ 열역학 제2법칙(고립계 엔트로피 증가). LLM의 평이화는 이 **같은 법칙의 한 얼굴**.

## 4. 대응 — 외부 에너지로 질서 유지
자연 상태(엔트로피 최대)로 흐르는 걸 막으려면 **외부 에너지(제약)를 주입**해야 한다.
- **컨텍스트 압축·요약:** 긴 대화를 핵심만 남겨 어텐션 시그널을 다시 날카롭게.
- **RAG:** 필요한 날카로운 맥락을 그때그때 주입해 Prior 지배를 막음.
- **추론 모델(o1·R1):** Chain of Thought([tʃeɪn əv θɔːt], 체인 오브 쏘트, 체인 오브 소트)로 스스로 검증·리팩토링(System 2([ˈsɪstəm tuː], 시스템 투, 시스템 투) 강제) → 엔트로피 재하강.

## 🆕 새 용어 정리
| 용어 | 한 줄 정의 |
|---|---|
| **어텐션 희석** (Attention Dilution) | 컨텍스트가 길수록 softmax 분모가 커져 핵심 맥락의 가중치가 희석 |
| **Lost in the Middle** | 긴 컨텍스트의 앞·뒤는 기억하나 중간 정보를 놓치는 현상 |
| **자기회귀 오염** | 뱉은 평범한 텍스트가 쌓여 다음 출력도 평범해지는 피드백 루프 |
| **평균 회귀** (Regression to the Mean) | 제약이 약해지면 학습 데이터 평균(Prior)으로 수렴 |
| **베이지안 붕괴** | 우도가 무력화돼 사후확률이 Prior에만 지배되는 것 |
| **위치 임베딩** (RoPE) | 토큰 순서를 알려주는 기법; 긴 컨텍스트에서 거리 왜곡 |
| **섀넌 엔트로피** | `H=-ΣP log P`; 분포가 평평할수록 큼 |
| **Cross-Entropy Loss** | 정답에 확률을 몰아 엔트로피를 최소화하는 LLM 학습 손실 |
| **최대 엔트로피 원리** | 제약이 없으면 가장 평평한(엔트로피 최대) 분포를 택함 |
